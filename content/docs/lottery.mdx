# 樂透

定期開獎的樂透系統，提供大樂透與小樂透兩種玩法，獎池會隨著買氣累積。

## 玩法比較

| 玩法 | 開獎時間 | 每張票價 | 包牌 |
| --- | --- | --- | --- |
| 大樂透 6/49 | 每週日 21:00 | 50 逼幣 | 支援（最多 10 個基號） |
| 小樂透 3/20 | 每週三、週日 21:00 | 10 逼幣 | 不支援 |

開獎時間為台北時間。獎池累積到里程碑時會在公告頻道宣布。

## 指令

### 查看資訊

```
/樂透資訊
```

顯示目前的獎池金額、下次開獎時間與倒數。

### 買票

```
/樂透買 玩法 [張數] [號碼]
```

| 參數 | 說明 | 必填 | 範圍／預設 |
| --- | --- | --- | --- |
| `玩法` | 6/49 或 3/20 | 是 | 下拉選擇 |
| `張數` | 購買張數 | 否 | 1–100 |
| `號碼` | 自選號碼 | 否 | 不填則機選（快選） |

### 包牌

```
/樂透包牌 玩法 號碼
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `玩法` | 僅支援大樂透 6/49 | 是 |
| `號碼` | 選 7 個以上的基號 | 是 |

包牌會自動產生所選基號的所有組合（最多 10 個基號）。

### 訂閱

自動幫你在未來幾期持續買票。

```
/樂透訂閱 玩法 期數 [每期張數] [號碼]
```

| 參數 | 說明 | 必填 | 範圍 |
| --- | --- | --- | --- |
| `玩法` | 6/49 或 3/20 | 是 | 下拉選擇 |
| `期數` | 連續訂閱幾期 | 是 | 1–12 |
| `每期張數` | 每期買幾張 | 否 | 1–10 |
| `號碼` | 自選號碼 | 否 | 不填則每期機選 |

```
/樂透訂閱列表
```

查看與管理你目前的訂閱。

<Callout type="warn">
  訂閱在每期開獎前自動扣款買票。若連續 2 期因餘額不足扣款失敗，訂閱會自動取消。
</Callout>

### 歷史紀錄

```
/樂透歷史
```

查看近期的中獎紀錄。
