# 彩券

定期開獎的彩券系統（舊稱 `/樂透`），提供大樂透、威力彩與小樂透三種玩法，獎池會隨著買氣累積。

## 玩法比較

| 玩法 | 開獎時間 | 每張票價 | 包牌 |
| --- | --- | --- | --- |
| 大樂透 6/49 | 每週日 21:00 | 50 逼幣 | 支援（最多 10 個基號） |
| 威力彩 6/38 + 2 | 每週一、四 21:00 | 50 逼幣 | 不支援 |
| 小樂透 3/20 | 每週三、週日 21:00 | 10 逼幣 | 不支援 |

開獎時間為台北時間。獎池累積到里程碑時會在公告頻道宣布。

## 指令

彩券相關功能都整合在 `/彩券` 底下，以子指令呼叫：

| 子指令 | 用途 |
| --- | --- |
| `/彩券 購買` | 買票 |
| `/彩券 包牌` | 包牌（自動展開組合） |
| `/彩券 資訊` | 查看當期彩池、倒數 |
| `/彩券 歷史` | 查看自己全部的彩券紀錄 |
| `/彩券 訂閱 開啟` | 建立自動買票訂閱 |
| `/彩券 訂閱 列表` | 查看／取消個別訂閱 |
| `/彩券 訂閱 關閉` | 一次取消所有訂閱（需確認） |

### 查看資訊

```
/彩券 資訊
```

顯示目前的獎池金額、下次開獎時間與倒數。

### 買票

```
/彩券 購買 玩法 [張數] [號碼] [第二區]
```

| 參數 | 說明 | 必填 | 範圍／預設 |
| --- | --- | --- | --- |
| `玩法` | 大樂透 6/49、威力彩或小樂透 3/20 | 是 | 下拉選擇 |
| `張數` | 購買張數 | 否 | 1–100 |
| `號碼` | 自選號碼 | 否 | 不填則機選（快選） |
| `第二區` | 威力彩第二區號碼（1 選自 1–8） | 視玩法而定 | 自選號碼時必填 |

### 包牌

```
/彩券 包牌 玩法 號碼
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `玩法` | 僅支援大樂透 6/49 | 是 |
| `號碼` | 選 7 個以上的基號 | 是 |

包牌會自動產生所選基號的所有組合（最多 10 個基號）。

### 訂閱

自動幫你在未來幾期持續買票。

```
/彩券 訂閱 開啟 玩法 期數 [每期張數] [號碼]
```

| 參數 | 說明 | 必填 | 範圍 |
| --- | --- | --- | --- |
| `玩法` | 大樂透 6/49 或小樂透 3/20 | 是 | 下拉選擇 |
| `期數` | 連續訂閱幾期 | 是 | 1–12 |
| `每期張數` | 每期買幾張 | 否 | 1–10 |
| `號碼` | 自選號碼 | 否 | 不填則每期機選 |

```
/彩券 訂閱 列表
```

查看目前進行中的訂閱，每筆都有獨立的「取消」按鈕。

```
/彩券 訂閱 關閉
```

一次取消所有進行中的訂閱（會先跳出確認按鈕，避免誤觸）。

<Callout type="warn">
  訂閱在每期開獎前自動扣款買票。若連續 2 期因餘額不足扣款失敗，訂閱會自動取消。
</Callout>

### 歷史紀錄

```
/彩券 歷史
```

查看近期的中獎紀錄，可用下拉篩選玩法、來源、結果，超過一頁時提供分頁按鈕。

## 威力彩

```
/彩券 購買 玩法:威力彩
```

威力彩採**雙區**選號：

- **第一區**：從 1–38 選 6 個號碼。
- **第二區**：從 1–8 選 1 個號碼（`第二區` 參數，自選號碼時必填）。

每週一、四 21:00 開獎。共 9 個獎項，依對中第一區號碼數量與是否中第二區「特別號」決定：

| 獎項 | 中獎條件 | 獎金 |
| --- | --- | --- |
| 頭獎 | 第一區 6 + 特 | 彩池分配 |
| 貳獎 | 第一區 6 | 彩池分配 |
| 參獎 | 第一區 5 + 特 | 彩池分配 |
| 肆獎 | 第一區 5 | 固定獎金 |
| 伍獎 | 第一區 4 + 特 | 固定獎金 |
| 陸獎 | 第一區 4 | 固定獎金 |
| 柒獎 | 第一區 3 + 特 | 固定獎金 |
| 捌獎 | 第一區 2 + 特 | 固定獎金 |
| 玖獎 | 第一區 3，或第一區 1 + 特 | 固定獎金 |

<Callout type="info">
  頭獎、貳獎、參獎走彩池分配（隨買氣累積、中獎人均分），肆獎～玖獎為固定獎金。威力彩**暫不支援訂閱與包牌**。
</Callout>

## 大樂透加碼

大樂透 6/49 額外提供兩種加碼：

| 加碼 | 說明 | 加碼獎金 |
| --- | --- | --- |
| 加碼球 | 每期多開一顆加碼球，你的票若含該號碼 | +200 |
| 連號加碼 | 開出的中獎號碼含 3 個以上連續號碼 | +300 |

<Callout type="info">
  加碼為大樂透的額外彩頭，與一般中獎獎金可同時觸發累加。
</Callout>
