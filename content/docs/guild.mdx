# 公會 🏰

公會是一群玩家**組隊長期經營**的小組織。會長負責管理人員與金庫，成員一起**捐款累積等級**換取整個公會通用的 buff，並合力完成**每週任務**領取金庫獎勵。

## 指令總覽

| 子指令 | 用途 | 對象 |
| --- | --- | --- |
| `/公會 建立 [名稱]` | 建立新公會（花費 5,000 逼幣） | 未加入公會的玩家 |
| `/公會 資訊 [名稱]` | 查看公會詳情（不填＝查自己的公會） | 所有人 |
| `/公會 邀請 [使用者]` | 邀請玩家加入公會 | 會長 / 副會長 |
| `/公會 申請 [名稱] [理由]` | 主動申請加入指定公會 | 未加入公會的玩家 |
| `/公會 退會` | 退出目前所屬公會 | 一般成員、副會長 |
| `/公會 捐款 [金額]` | 捐逼幣進公會金庫 | 公會成員 |
| `/公會 任務` | 查看本週公會任務進度（達標後按鈕領取） | 公會成員 |
| `/公會 倉庫` | 查看公會倉庫存貨與今日領取額度 | 公會成員 |

  以下功能改由**按鈕**操作，沒有獨立 slash command：
  - **編輯簡介**：`/公會 資訊` → 會長 / 副會長看到「📝 編輯簡介」按鈕
  - **待審申請**：`/公會 資訊` → 有 pending 申請時會長 / 副會長看到「📨 待審申請 N」紅色按鈕
  - **領週任務獎勵**：`/公會 任務` → 達標時會長看到「🎁 一鍵領取」按鈕
  - **倉庫設定 / 紀錄**：`/公會 倉庫` → 會長 / 副會長看到對應按鈕
  - **公會管理**（踢人 / 轉讓會長 / 指派 / 撤銷副會長 / 解散）：`/公會 資訊` 底部的「⚙️ 公會管理」按鈕區，會長 / 副會長按權限顯示不同按鈕
</Callout>

## 建立公會

```
/公會 建立 [名稱]
```

繳交 **5,000 逼幣**即可建立一個新公會，並自動成為**會長**。

| 規則 | 說明 |
| --- | --- |
| 名稱長度 | 1–12 字 |
| 禁字 | `@`、`#`、`<`、`>`、`` ` `` |
| 名稱唯一 | 同伺服器不允許重複名稱 |
| 加入限制 | 每位玩家在同一伺服器**只能屬於一個公會** |

<Callout type="info">
  想另立門戶？先 `/公會 退會`，或在 `/公會 資訊` →「⚙️ 公會管理」→「💥 解散公會」結束目前公會，才能再 `/公會 建立`。
</Callout>

## 公會簡介 / 會規

每個公會都可以放一段**對外的自介文字**——可以寫公會風格、入會條件、活動時間、捐款建議等等，讓申請者進來前就先有概念。

編輯入口在 `/公會 資訊` 底部的「📝 編輯簡介 / 會規」按鈕，**會長與副會長**都能按；按下後彈出 Modal 輸入。

- 內容最多 **500 字**，支援多行。
- 留空送出＝清除目前的簡介。
- 在 `/公會 資訊` 會以「📜 公會簡介 / 會規」區塊顯示，**所有人**都看得到。

<Callout type="info">
  寫簡介時可以列幾條重點：例如「歡迎挖礦狂魔」「每週至少打一次 BOSS」「捐款隨意但越多越好」——對申請者來說比公會名稱更有參考價值。
</Callout>

## 副會長

公會可由會長指派**副會長**，協助處理日常營運。副會長的權限**幾乎等同會長**，但**不能解散公會、不能轉讓會長、也不能再指派／撤銷其他副會長**。

入口：`/公會 資訊` →「⚙️ 公會管理」→「🛡️ 指派副會長」/「🪶 撤銷副會長」，按下後從成員下拉選單挑人。

| 操作 | 會長 | 副會長 | 一般成員 |
| --- | --- | --- | --- |
| 邀請玩家加入 | ✅ | ✅ | ❌ |
| 處理申請（核准 / 拒絕） | ✅ | ✅ | ❌ |
| 踢出成員 | ✅ | ✅（不能踢會長 / 其他副會長） | ❌ |
| 編輯公會簡介 / 會規 | ✅ | ✅ | ❌ |
| 領取週任務獎勵 | ✅ | ✅ | ❌ |
| 指派 / 撤銷副會長 | ✅ | ❌ | ❌ |
| 轉讓會長 | ✅ | ❌ | ❌ |
| 解散公會 | ✅ | ❌ | ❌ |
| 捐款 / 看資訊 / 查任務 | ✅ | ✅ | ✅ |

- 副會長人數沒有硬性上限，會長可視公會規模指派。
- 在 `/公會 資訊` 的成員列表中，副會長會標 🛡️，會長標 👑，排序為 **會長 → 副會長 → 一般成員**。
- 會長身分被「⚙️ 公會管理 → 👑 轉讓會長」轉出去後，原會長自動降為一般成員；若新會長原本是副會長，會自動升任。

<Callout type="warn">
  副會長被踢出公會、退會或被會長從「⚙️ 公會管理 → 🪶 撤銷副會長」撤銷後，身分立刻失效。指派時請挑值得信任的長期成員。
</Callout>

## 加入公會：邀請 vs 申請

加入公會有兩條路：**會長主動邀請**，或**玩家主動申請**。兩者都要對方同意才會生效。

### 會長邀請

```
/公會 邀請 [使用者]
```

會長執行後，被邀請的玩家會收到一張帶有「接受 / 拒絕」按鈕的卡片。邀請有效期 **168 小時（7 天）**，超時自動失效。

- 不能邀請機器人或自己。
- 若對方已在其他公會、或公會人數已滿，會擋下並提示原因。
- 同一人同時只能有一張 pending 邀請。

### 玩家申請

```
/公會 申請 [名稱] [理由?]
```

玩家主動申請加入某公會。可附上**最多 100 字**的理由給會長參考。

- 申請有效期 **168 小時（7 天）**。
- 若被該公會拒絕，需等 **24 小時冷卻**才能再次申請同一個公會。
- 會長 / 副會長在 `/公會 資訊` 看到「📨 待審申請 N」紅色按鈕（pending > 0 才會顯示），按下後進入列表逐一核准 / 拒絕。

## 公會升等與 Buff

公會每收到一筆**捐款**就會累積到「總捐款額」，達到門檻時自動升等。等級越高，**人數上限**越高、提供給**所有成員**的 buff 也越強。

| 等級 | 升等門檻（累計捐款） | 人數上限 | 提供的 Buff |
| --- | --- | --- | --- |
| Lv.1 | 0 | 10 | — |
| Lv.2 | 10,000 | 12 | 挖礦數量 +1 |
| Lv.3 | 50,000 | 15 | 挖礦數量 +1、打工收入 +10% |
| Lv.4 | 150,000 | 18 | 挖礦數量 +1、打工收入 +10%、地下城體力上限 +1、**BOSS 戰攻擊力 +5%** |
| Lv.5 | 500,000 | 20 | 挖礦數量 +1、打工收入 +10%、地下城體力上限 +1、挖礦幸運 +5%、**BOSS 戰攻擊力 +10%**、**BOSS 戰每場攻擊次數 +1** |

<Callout type="info">
  公會 buff 會**自動套用**到所有公會成員身上，可在 `/狀態` 看到目前實際享有的數值。升等時會自動在公會頻道發布公告。
</Callout>

### 捐款

```
/公會 捐款 [金額]
```

把自己的逼幣轉進公會金庫。**個人累計捐款額**會記錄在公會資訊裡，公會總捐款額也會推升等級。

- 金額必須是正整數。
- 餘額不足會擋下並提示差距。
- 公會解散時，金庫會**平分**給所有成員。

## 每週公會任務

每週公會會有**共同任務**，全體成員的數值會合併計算進度，達標後由會長統一領獎。

| 任務 | 達標條件 | 獎勵（進金庫） |
| --- | --- | --- |
| ⛏️ 礦業大隊 | 公會成員本週合計挖礦 ≥ 100 次 | 5,000 逼幣 |
| 🗡️ 地下探索隊 | 公會成員本週合計通關地下城 ≥ 30 次 | 8,000 逼幣 |
| 🎰 賭場聯盟 | 公會成員本週合計賭場 ≥ 50 局 | 6,000 逼幣 |

- 重置時間：每週一 00:00（Asia/Taipei）。
- `/公會 任務` 任何成員都能查當週進度。
- 達標後，會長在 `/公會 任務` 畫面底部會看到「🎁 一鍵領取入金庫」按鈕；按下後所有已達標獎勵一次入庫並計入累計捐款（推升等級）。
- 達標後若沒人領、跨週重置就失效。

## 管理：踢人、轉讓、解散

這些**低頻會長操作**集中在 `/公會 資訊` 底部的「⚙️ 公會管理」按鈕區，會長 / 副會長按權限看到不同按鈕，避免 slash command 列表過長。

| 按鈕 | 說明 | 對象 |
| --- | --- | --- |
| 👢 踢人 | 從成員下拉選單挑一位踢出公會。不能踢自己；副會長另外不能踢會長 / 其他副會長。 | 會長 / 副會長 |
| 🛡️ 指派副會長 | 從成員下拉選單挑一位指派為副會長。 | 僅會長 |
| 🪶 撤銷副會長 | 從成員下拉選單挑一位副會長撤銷其身分（仍是公會成員）。 | 僅會長 |
| 👑 轉讓會長 | 選擇新會長後**多一層確認**才執行。轉讓完原會長變回一般成員，不可逆。 | 僅會長 |
| 💥 解散公會 | 紅色按鈕。按下後跳出確認 Container：列出分配人數、每人分得多少、被沒收的鎖定金。確認後金庫**平分**給有資格的成員，公會永久消失。 | 僅會長 |
| `/公會 退會` | 一般成員、副會長可直接退會；會長想退會請先用「👑 轉讓會長」或「💥 解散公會」。 | 一般成員、副會長 |

<Callout type="warn">
  解散後公會名稱可被新公會使用，原成員的個人捐款累積也會清空。會長還會進入**重建冷卻**，不能立刻再建新公會。確認過才按「確定解散」。
</Callout>

<Callout type="info">
  按鈕互動全程**只有按按鈕的人看得到**（ephemeral），不會洗版頻道。對選到非公會成員、或對方已是 / 不是副會長等狀況，會在同一則訊息回覆錯誤說明。
</Callout>

## 公會倉庫 📦

公會倉庫是會員間**互助分享原料**的池子：手上太多的礦／作物／魚可以存進來給公會用，缺料的成員付一點手續費就能領出對應數量。**Lv.2 解鎖**。

<Callout type="info">
  倉庫**不是**個人物品的暫存箱。**存入即放棄所有權，不可收回**——等同捐贈。整套機制設計是讓資源在公會內部流轉，不是給個人換倉。
</Callout>

### 怎麼用

```
/公會 倉庫                       ← 查看存貨 / 你今日已領紀錄
/公會 存入 [物品] [數量]         ← 把背包 / 魚袋的資源放進倉庫
/公會 領取 [物品] [數量]         ← 從倉庫領出（也可在倉庫畫面直接按按鈕）
```

倉庫畫面把礦石 / 作物 / 魚類分塊顯示，每項旁邊都有「領 1」「領 5」「領上限」快捷按鈕，數字就是實際扣的手續費——直接點即可，不需重打指令。

### 支援物品與單次上限

倉庫第一版只收**原料型成品**（已進入背包 / 魚袋），田裡未收割、料理成品、種子、合成材料、Buff 道具**不可存**。

| 分類 | 物品 | 單次取出上限 |
| --- | --- | --- |
| ⛏️ 礦石 | 石頭、煤炭 | 15 |
| | 鐵礦 | 10 |
| | 黃金 | 5 |
| | 鑽石 | 1 |
| 🌾 作物 | 紅蘿蔔 | 15 |
| | 玉米 | 10 |
| | 草莓 | 5 |
| | 黑玫瑰 | 1 |
| 🎣 魚類 | 小雜魚、鯽魚 | 15 |
| | 鯊魚 | 10 |
| | 章魚 | 5 |
| | 熔岩魚 | 1 |

<Callout type="info">
  **建材 🧱 / 鋼錠 ⛓️** 是公會專屬加工品，只能由[熔爐](#熔爐把礦石煉成建材--鋼錠)製造、由[公會建築](#公會建築礦坑--訓練場--倉庫擴建)消耗，**不能** `/公會 存入` / `/公會 領取` / `/公會 寄售`。
</Callout>

倉庫**容量上限**隨公會等級放大（Lv.2 = 1.0 倍 → Lv.10 = 3.0 倍），並可由[倉庫擴建建築](#-倉庫擴建提升公會倉庫容量)再追加：

| 等級 | 容量乘數 | 例：鐵礦容量 |
| --- | --- | --- |
| Lv.2 | 1.0 | 300 |
| Lv.3 | 1.3 | 390 |
| Lv.4 | 1.6 | 480 |
| Lv.5 | 1.9 | 570 |
| Lv.10 | 3.0 | 900 |

### 領取的規則（避免資源被吸乾）

| 規則 | 預設 | 會長可調 |
| --- | --- | --- |
| 公會等級 ≥ 2 | 系統解鎖門檻 | ❌ |
| 已加入公會 ≥ N 小時 | 24 小時 | ✅ 6h ~ 168h |
| 個人累積貢獻 ≥ V | 200 | ✅ 0 ~ 5000 |
| 每日領取次數 | 2 次 | ✅ 1 ~ 3 |
| 同一物品一天只能取一次 | 固定 | ❌ |
| 同存入者 24 小時內不能領回自己存的物品 | 固定 | ❌ |
| 新存物品保護期（全員都不能領） | 1 小時 | ❌ |

> 「累積貢獻」= 金庫捐款 + 倉庫存入市價 × 10%（換算貢獻值）+ BOSS 戰貢獻。

### 手續費（按市價比例，不是固定值）

```
fee = max(20, ⌈ 單價 × 數量 × 費率 ⌉)
```

預設費率 **10%**，會長可調 5% ~ 20%。**手續費直接進公會金庫，不消失**——付出去的錢還是留在公會內部循環。

| 範例 | 計算 | 手續費 |
| --- | --- | --- |
| 取 15 顆石頭（單價 8） | max(20, ⌈ 8×15×0.1 ⌉) | 20 |
| 取 5 顆鐵礦（單價 60） | max(20, ⌈ 60×5×0.1 ⌉) | 30 |
| 取 1 顆鑽石（單價 800） | max(20, ⌈ 800×1×0.1 ⌉) | 80 |
| 取 1 條熔岩魚（單價 600） | max(20, ⌈ 600×1×0.1 ⌉) | 60 |

<Callout type="warn">
  原本提案的「固定 20 元手續費」對鑽石、黑玫瑰、熔岩魚這種高單價物資等於白送，會被當作免費取貨。改成市價比例後，越貴的物品手續費越高，貢獻門檻才扛得住。
</Callout>

### 倉庫設定（會長 / 副會長）

`/公會 倉庫` 畫面右下「⚙️ 倉庫設定」按鈕，打開 Modal 後可調這四項：

| 設定 | 預設 | 範圍 |
| --- | --- | --- |
| 每日次數（每位成員） | 2 | 1 ~ 3 |
| 手續費率 | 10% | 5% ~ 20% |
| 入會時間門檻 | 24 小時 | 6h ~ 168h |
| 最低貢獻門檻 | 200 | 0 ~ 5000 |

- 留**空白**送出 = 恢復系統預設。
- 數字超出範圍會被擋下，不會破壞平衡。

### 倉庫紀錄（會長 / 副會長）

`/公會 倉庫` 畫面右下「📜 紀錄」按鈕，列出近 30 天的存 / 取流水，每筆顯示時間、操作者、物品、數量、手續費。頂部固定一行**會長監督**：

> 近 30 天會長淨領取價值：N 幣（含手續費回流 M 幣）

讓所有會員一眼確認會長有沒有自肥。

### 大額存入廣播

單次存入**市價 ≥ 10,000 逼幣**（例如黑玫瑰 ×8、熔岩魚 ×17）時，會自動在公會頻道發布「🎁 大額捐贈！」公告，讓會員看到誰在挺公會。

### 公會解散時的倉庫怎麼處理

解散公會時，倉庫所有物品依當下單價**整批折現入金庫**，再依現行解散規則平分給有資格的成員（在會時間達門檻者）。沒有人會「私吞倉庫」這種事，因為倉庫從來就不是個人資產。

## 公會擴張：熔爐 / 精煉站 / 公會建築 🏗️

## 公會 × BOSS 連動

公會不只挖礦／打工有用，也會直接影響每週 BOSS 戰的表現與獎勵分潤。

### 戰場上：公會 buff 滲透 BOSS 戰

當公會升到 **Lv.4 / Lv.5**，所有成員在打 BOSS 時自動吃到專屬加成：

- **Lv.4**：BOSS 戰攻擊力 +5%
- **Lv.5**：BOSS 戰攻擊力 +10%、每場攻擊次數上限 +1（5 → 6 次）

加成會自動套用在 `/魔王 攻擊` 的傷害計算與次數判定上，可在 `/狀態` 看到目前實際生效的數值。

### 結算時：個人 BOSS 貢獻

每場 BOSS 結算時，玩家造成的傷害會依比例（**傷害 × 10%**）換算成**個人公會貢獻點**。

- 累計分為「總貢獻」與「本週貢獻」兩個維度。
- `/公會 資訊` 會顯示「**🐉 本週 BOSS 貢獻 Top 5**」榜單。
- 跨週自動歸零、換公會自動歸戶到新公會。

### 結算時：BOSS 分潤入公會公庫

每場 BOSS 結算還會自動把獎勵分一部份進**公會公庫**，分兩種管道：

| 來源 | 入庫比例 | 入帳方式 |
| --- | --- | --- |
| 全體成員造成的傷害 | 傷害合計 × 5% | 直接進**可分配餘額** |
| 擊殺者尾刀獎勵 | 尾刀金 × 10% | 進**鎖定金**，168 小時後解鎖 |
| 本場 MVP 個人分潤 | MVP 分得獎金 × 10% | 進**鎖定金**，168 小時後解鎖 |

<Callout type="info">
  公庫累計也計入**升級進度**，所以打 BOSS 不只賺自己的逼幣，也直接幫公會升等解鎖更多 buff。鎖定金的機制與週任務獎勵相同：解散公會時尚未解鎖的部分會被沒收，防止刷 BOSS 然後立刻解散分錢。
</Callout>

## 常見問題

- **公會 buff 跟其他來源的 buff 怎麼疊？** 公會 buff 會跟個人 buff（食物、裝備等）一起累加，在 `/狀態` 可以看到所有來源的明細。
- **退會 / 被踢後個人累計捐款會保留嗎？** 不會。離開公會就清空，重新加入需從 0 開始累積。
- **公會升等需要做什麼？** 不用手動操作。只要總捐款達到門檻就會自動升等並通知，新 buff 立刻對所有成員生效。
- **副會長可以指派其他副會長嗎？** 不行，**只有會長**能在「⚙️ 公會管理」按鈕區指派或撤銷副會長。
- **轉讓會長給副會長後，副會長身分會怎樣？** 直接升任會長；原會長變回一般成員，原本的副會長頭銜消失。
- **找不到 `/公會 解散`、`/公會 踢人` 這些指令？** 已合併到 `/公會 資訊` 底部的「⚙️ 公會管理」按鈕區。低頻、會長專屬的操作改成按鈕，避免 slash command 列表爆量。
- **存進倉庫的東西可以拿回來嗎？** 不能。存入即放棄所有權，這是設計上的硬規則，避免有人「存進去刷貢獻 → 自己領回」白嫖。
- **倉庫滿了怎麼辦？** 升公會等級會自動擴張容量（Lv.2 → Lv.10 三倍），或請會員多用掉一些。
- **為什麼我剛存進去就不能領？** 新存物品有 **1 小時保護期**，全員都不能領，避免協同掃倉。即使保護期過了，「同存入者 24 小時內」也不能領回自己存的那項。
- **手續費怎麼算？為什麼有時候 20 有時候 80？** 公式 `max(20, 市價 × 10%)`。低單價物資（石頭、小雜魚）一律 20 起跳，高單價物資（鑽石、熔岩魚）會按 10% 算，越貴越貴。
- **手續費跑去哪了？** 直接進**公會金庫**，沒有消失。會員之間付的手續費還是留在公會內部。
- **熔爐 / 建築升級費誰出？** 一律從**公會金庫**扣，不是會長自己的錢。會長 / 副會長是「代表公會執行」，所以拿公會的錢理所當然——金庫流水也會有對應紀錄，可在 `/公會 倉庫 → 📜 紀錄` 查到。
- **熔爐製造建材 / 鋼錠是個人的還是公會的？** 公會的。原料從公會倉庫扣，產出物入公會倉庫，全程不會碰到個人背包。
- **建材 / 鋼錠可以拿出來賣嗎？** 不行。它們是公會專屬中間物資，只用於建築升級。要賣的話請改賣原料（石頭、煤炭、鐵礦）。
- **倉庫擴建是擴個人背包還是公會倉庫？** **公會倉庫**。倉庫擴建建築會增加公會倉庫**每個物品**的容量上限，但個人背包不受影響。
