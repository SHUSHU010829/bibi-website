# 公會 🏰

公會是一群玩家**組隊長期經營**的小組織。會長負責管理人員與金庫，成員一起**捐款累積等級**換取整個公會通用的 buff，並合力完成**每週任務**領取金庫獎勵。

<Callout type="info">
  `/公會` 只能在特定類別下的頻道使用。若回覆「此頻道不能使用公會指令」，請移動到伺服器指定的公會類別頻道再執行。
</Callout>

## 建立公會

```
/公會 建立 [名稱]
```

繳交 **5,000 逼幣**即可建立一個新公會，並自動成為**會長**。

| 規則 | 說明 |
| --- | --- |
| 名稱長度 | 1–12 字 |
| 禁字 | `@`、`#`、`<`、`>`、`` ` `` |
| 名稱唯一 | 同伺服器不允許重複名稱 |
| 加入限制 | 每位玩家在同一伺服器**只能屬於一個公會** |

<Callout type="info">
  想另立門戶？先 `/公會 退會` 或 `/公會 解散`，才能再 `/公會 建立`。
</Callout>

## 加入公會：邀請 vs 申請

加入公會有兩條路：**會長主動邀請**，或**玩家主動申請**。兩者都要對方同意才會生效。

### 會長邀請

```
/公會 邀請 [使用者]
```

會長執行後，被邀請的玩家會收到一張帶有「接受 / 拒絕」按鈕的卡片。邀請有效期 **168 小時（7 天）**，超時自動失效。

- 不能邀請機器人或自己。
- 若對方已在其他公會、或公會人數已滿，會擋下並提示原因。
- 同一人同時只能有一張 pending 邀請。

### 玩家申請

```
/公會 申請 [名稱] [理由?]
```

玩家主動申請加入某公會。可附上**最多 100 字**的理由給會長參考。

- 申請有效期 **168 小時（7 天）**。
- 若被該公會拒絕，需等 **24 小時冷卻**才能再次申請同一個公會。
- 會長使用 `/公會 申請列表` 查看並逐一處理（核准 / 拒絕）。

## 公會升等與 Buff

公會每收到一筆**捐款**就會累積到「總捐款額」，達到門檻時自動升等。等級越高，**人數上限**越高、提供給**所有成員**的 buff 也越強。

| 等級 | 升等門檻（累計捐款） | 人數上限 | 提供的 Buff |
| --- | --- | --- | --- |
| Lv.1 | 0 | 10 | — |
| Lv.2 | 10,000 | 12 | 挖礦數量 +1 |
| Lv.3 | 50,000 | 15 | 挖礦數量 +1、打工收入 +10% |
| Lv.4 | 150,000 | 18 | 挖礦數量 +1、打工收入 +10%、地下城體力上限 +1 |
| Lv.5 | 500,000 | 20 | 挖礦數量 +1、打工收入 +10%、地下城體力上限 +1、挖礦幸運 +5% |

<Callout type="info">
  公會 buff 會**自動套用**到所有公會成員身上，可在 `/加成` 看到目前實際享有的數值。升等時會自動在公會頻道發布公告。
</Callout>

### 捐款

```
/公會 捐款 [金額]
```

把自己的逼幣轉進公會金庫。**個人累計捐款額**會記錄在公會資訊裡，公會總捐款額也會推升等級。

- 金額必須是正整數。
- 餘額不足會擋下並提示差距。
- 公會解散時，金庫會**平分**給所有成員。

## 每週公會任務

每週公會會有**共同任務**，全體成員的數值會合併計算進度，達標後由會長統一領獎。

| 任務 | 達標條件 | 獎勵（進金庫） |
| --- | --- | --- |
| ⛏️ 礦業大隊 | 公會成員本週合計挖礦 ≥ 100 次 | 5,000 逼幣 |
| 🗡️ 地下探索隊 | 公會成員本週合計通關地下城 ≥ 30 次 | 8,000 逼幣 |
| 🎰 賭場聯盟 | 公會成員本週合計賭場 ≥ 50 局 | 6,000 逼幣 |

- 重置時間：每週一 00:00（Asia/Taipei）。
- `/公會 任務` 任何成員都能查當週進度。
- `/公會 領獎` 由**會長**一鍵領取所有已達標的獎勵；獎金直接進**公會金庫**並計入累計捐款（推升等級）。
- 達標後若沒人領、跨週重置就失效。

## 管理：踢人、轉讓、解散

| 指令 | 說明 |
| --- | --- |
| `/公會 踢人 [使用者]` | 會長把成員踢出公會。不能踢自己。 |
| `/公會 轉讓 [使用者]` | 把會長身分轉讓給目前公會的其他成員，自己變回一般成員。 |
| `/公會 退會` | 一般成員直接退會；會長若想退會，**請先轉讓**或解散。 |
| `/公會 解散` | 會長解散公會，金庫**平分**給所有成員，公會永久消失。 |

<Callout type="warn">
  解散後公會名稱可被新公會使用，原成員的個人捐款累積也會清空。確認過才按確認鍵。
</Callout>

## 公會排行

```
/公會 排行
```

查看本伺服器的公會排行榜（前 10 名），依**等級 → 總捐款額**排序。你目前所屬公會會被標示出來，方便比較進度。

## 常見問題

- **公會 buff 跟其他來源的 buff 怎麼疊？** 公會 buff 會跟個人 buff（食物、裝備等）一起累加，在 `/加成` 可以看到所有來源的明細。
- **退會 / 被踢後個人累計捐款會保留嗎？** 不會。離開公會就清空，重新加入需從 0 開始累積。
- **公會升等需要做什麼？** 不用手動操作。只要總捐款達到門檻就會自動升等並通知，新 buff 立刻對所有成員生效。
