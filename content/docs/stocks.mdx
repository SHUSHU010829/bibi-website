# 股票市場

用逼幣投資虛擬股市，低買高賣賺價差，還能領配息。

<Callout type="info">
  股票指令都在 **逼逼金證卷 │ 🪙** 頻道使用，詳見[指令頻道規範](/docs/getting-started#指令頻道規範)。
</Callout>

## 指令

股市相關功能整合在 `/股市` 底下，以子指令呼叫：

| 子指令 | 用途 |
| --- | --- |
| `/股市 報價` | **互動面板**：所有股票即時報價，下拉選股 + 按鈕直接買賣與看走勢（推薦） |
| `/股市 持股` | 查看自己當前持股、損益與總市值，每筆附「🔴 賣出」按鈕 |
| `/股市 買` | 直接以市價買入指定股票 |
| `/股市 賣` | 直接以市價賣出指定股票 |
| `/股市 走勢` | 查看股票價格走勢圖 |
| `/股市 配息` | 查詢自己的配息紀錄 |

<Callout type="info">
  股票代號 **不需要手動輸入**——所有需要選股的子指令都會跳出下拉選單讓你直接點選；最方便的方式是 `/股市 報價`，連選股、買賣、看圖都在同一個面板裡完成。
</Callout>

### 互動報價面板（推薦）

```
/股市 報價
```

打開後會出現一張面板，顯示所有上市股票的即時報價。互動方式：

1. 從**下拉選單**選擇想操作的股票。
2. 選完後下方的按鈕會解鎖，可以：
   - 🟢 **買入** — 跳出輸入框，填股數後直接成交。
   - 🔴 **賣出** — 跳出輸入框，填股數（或 `all`）後直接成交。
   - 📜 **1d / 1w / 1m** — 直接看該股票對應期間的走勢圖。
3. 成交完面板會更新最新報價，可繼續換股操作。

面板 5 分鐘無動作後自動失效。

### 買股

```
/股市 買 股票代號 數量
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `股票代號` | 下拉選擇上市股票（不需手動輸入） | 是 |
| `數量` | 買入股數（正整數） | 是 |

### 賣股

```
/股市 賣 股票代號 數量
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `股票代號` | 下拉選擇上市股票（不需手動輸入） | 是 |
| `數量` | 股數（填 `all` 表示全部賣出） | 是 |

### 持股

```
/股市 持股
```

列出你目前所有持股、均價、現價、各檔損益與總市值；每筆持股右側會掛一顆「🔴 賣出」按鈕，可直接彈出 Modal 填股數（或 `all`）成交。

### 走勢

```
/股市 走勢 股票代號 [期間]
```

查看某股票的價格走勢圖；`期間` 可選 1d / 1w / 1m，預設 1w。

### 配息紀錄

```
/股市 配息 [期間]
```

查看你領取的配息紀錄；`期間` 可選 1m / 3m，預設 1m。

## 市場規則

| 項目 | 數值 |
| --- | --- |
| 開盤時間 | 每日 09:00–21:00（台北時間） |
| 價格跳動 | 每 5 分鐘更新一次 |
| 交易手續費 | 1%（最低 5 逼幣） |
| 每人單檔持股上限 | 500 股 |
| 配息時間 | 每週一 09:00 |

<Callout type="info">
  每日開盤時市場會輪換「多頭／空頭／盤整」情緒，影響整體漲跌方向；個股仍會依自身波動度起伏。
</Callout>

<Callout type="warn">
  股票有漲有跌，投資前請評估風險。手續費買賣都要收，頻繁交易會侵蝕獲利。
</Callout>
