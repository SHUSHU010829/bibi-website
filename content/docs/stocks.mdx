# 股票市場

用逼幣投資虛擬股市，低買高賣賺價差，還能領配息。

## 指令

股市相關功能整合在 `/股市` 底下，以子指令呼叫：

| 子指令 | 用途 |
| --- | --- |
| `/股市 報價` | **互動面板**：所有股票即時報價，下拉選股 + 按鈕直接買賣與看走勢 |
| `/股市 持股` | 查看自己當前持股、損益與總市值，每筆附「🔴 賣出」按鈕 |
| `/股市 買` | 直接以市價買入指定股票 |
| `/股市 賣` | 直接以市價賣出指定股票 |
| `/股市 走勢` | 查看股票價格走勢圖 |
| `/股市 配息` | 查詢自己的配息紀錄 |
| `/股市 紀錄` | 查詢自己過去的買賣成交與已實現損益 |
| `/股市 停損停利` | 設定自動觸發價，命中時開盤掃描會全倉自動賣出 + 私訊通知 |
| `/股市 融券` | 借股做空：看跌先高賣，跌了低買回賺價差 |
| `/股市 回補` | 買回股票平掉融券部位、結算損益 |
| `/股市 排行` | 已實現損益操盤手排行榜 |

<Callout type="info">
  股票代號 **不需要手動輸入**——所有需要選股的子指令都會跳出下拉選單讓你直接點選；最方便的方式是 `/股市 報價`，連選股、買賣、看圖都在同一個面板裡完成。
</Callout>

### 互動報價面板

```
/股市 報價
```

打開後會出現一張面板，顯示所有上市股票的即時報價。互動方式：

<Steps>
  <Step>
    從**下拉選單**選擇想操作的股票。
  </Step>
  <Step>
    選完後下方的按鈕會解鎖，可以：
    - 🟢 **買入** — 跳出輸入框，填股數後直接成交。
    - 🔴 **賣出** — 跳出輸入框，填股數（或 `all`）後直接成交。
    - 📜 **1d / 1w / 1m** — 直接看該股票對應期間的走勢圖。
  </Step>
  <Step>
    成交完面板會更新最新報價，可繼續換股操作。
  </Step>
</Steps>

面板 5 分鐘無動作後自動失效。

### 買股

```
/股市 買 股票代號 數量
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `股票代號` | 下拉選擇上市股票（不需手動輸入） | 是 |
| `數量` | 買入股數（正整數） | 是 |

### 賣股

```
/股市 賣 股票代號 數量
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `股票代號` | 下拉選擇上市股票（不需手動輸入） | 是 |
| `數量` | 股數（填 `all` 表示全部賣出） | 是 |

### 持股

```
/股市 持股
```

列出你目前所有持股、均價、現價、各檔損益與總市值；每一支持股的右側會掛一顆 **🔴 賣 `<代號>`** 按鈕：點下去跳出輸入框填股數（可填 `all`）後直接成交，省去再跑 `/股市 賣`。

### 走勢

```
/股市 走勢 股票代號 [期間]
```

查看某股票的價格走勢圖；`期間` 可選 1d / 1w / 1m，預設 1w。

### 配息紀錄

```
/股市 配息 [期間]
```

查看你領取的配息紀錄；`期間` 可選 1m / 3m，預設 1m。

### 買賣紀錄

```
/股市 紀錄 [期間] [股票代號]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `期間` | 查詢區間，可選 1w / 1m / 3m，預設 1m | 否 |
| `股票代號` | 只看單一股票，預設顯示全部 | 否 |

列出你過去的買賣成交紀錄，包含買入/賣出筆數與股數、總支出與淨入帳、手續費合計，以及賣出累積的**已實現損益**，並附上最近 15 筆成交明細。交易紀錄保留 90 天。

### 停損 / 停利

```
/股市 停損停利 股票代號 [停損價] [停利價]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `股票代號` | 要設定的股票 | 是 |
| `停損價` | 跌到此價自動全倉賣出（須**低於**現價 1% 以上）；填 `0` 可清除已設停損 | 否 |
| `停利價` | 漲到此價自動全倉賣出（須**高於**現價 1% 以上）；填 `0` 可清除已設停利 | 否 |

開盤時段（09:00–20:59）每分鐘掃描一次持倉，命中即呼叫 `/股市 賣` 全倉賣出，並私訊告知成交價、淨入帳與盈虧。觸發後該股的停損 / 停利會**自動清除**，要再啟動須重設。

<Callout type="info">
  防護機制：同一玩家同一支股票 **24 小時內最多自動成交 1 次**，避免極端波動連續刷洗。仍會收 1% 手續費（最低 5 逼幣），不是免費避險工具。
</Callout>

<Callout type="warn">
  停損 / 停利價必須距現價 ≥ 1%，否則指令會擋下；這是避免設定當下就抖動觸發。兩個參數都不填會清除該股的所有自動觸發。
</Callout>

### 融券做空

```
/股市 融券 股票代號 數量
```

看壞某支股票時，可以**借股放空**：系統凍結等額保證金後幫你以現價高賣，等股價下跌再 `/股市 回補` 低價買回，賺中間的價差。跌越多賺越多，看錯反被套。

```
/股市 回補 股票代號 數量
```

買回股票平掉融券部位並結算損益（`數量` 可填 `all`）。你的**最大虧損以保證金為限**——就算股價暴衝，也不會倒賠。

<Callout type="warn">
  斷頭風險：當浮動虧損達保證金的 **80%**，開盤掃描會**強制回補**你的融券部位（斷頭）並私訊通知，避免虧損擴大到吃光保證金。單檔融券上限 300 股。
</Callout>

### 操盤手排行

```
/股市 排行 [期間]
```

依**已實現損益**（賣出 + 融券回補）排出操盤手榜；`期間` 可選 本週 / 近 30 天 / 近 90 天，預設本週。每週一結算上週冠軍，頒發 📊 **最強操盤手** 稱號（每週更替）。

## 市場規則

| 項目 | 數值 |
| --- | --- |
| 開盤時間 | 每日 09:00–21:00（台北時間） |
| 價格跳動 | 每 5 分鐘更新一次 |
| 交易手續費 | 1%（最低 5 逼幣） |
| 每人單檔持股上限 | 500 股 |
| 每日漲跌停 | 以開盤價為基準 ±50% |
| 交易衝擊 | 買單推升、賣單壓低股價（隨股數增加，單筆最多 ±6%） |
| 融券單檔上限 | 300 股（凍結等額保證金） |
| 融券斷頭 | 浮虧達保證金 80% 強制回補 |
| 配息時間 | 每週一 09:00 |

<Callout type="info">
  每日開盤時市場會輪換「多頭／空頭／盤整」情緒，影響整體漲跌方向；個股仍會依自身波動度起伏。
</Callout>

<Callout type="info">
  **交易會影響股價**：大量買進會把價格頂上去、大量賣出會砸下來，成交後的結果卡會顯示你這筆單推動了多少。想大買大賣前記得留意衝擊成本。
</Callout>

<Callout type="warn">
  **漲跌停**：單日漲跌以開盤價 ±50% 為限。鎖漲停時買方掛單排隊、暫時買不進（仍可賣）；鎖跌停時賣方排隊、暫時賣不掉（仍可買）。
</Callout>

<Callout type="warn">
  股票有漲有跌，投資前請評估風險。手續費買賣都要收，頻繁交易會侵蝕獲利。
</Callout>
