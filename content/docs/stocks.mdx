# 股票市場

用逼幣投資虛擬股市，低買高賣賺價差，還能領配息。

## 上市股票

| 代號 | 名稱 | 類型 | 起始價 |
| --- | --- | --- | --- |
| TSPP | 嗶積電 | 科技 | 500 |
| UPPI | 統嗶超商 | 績優 | 300 |
| EGPP | 嗶嗶海運 | 迷因 | 120 |
| CTPP | 嗶嗶金控 | 績優 | 800 |
| MTKP | 嗶發科 | 科技 | 200 |

不同股票的波動度（風險）與配息率不同；迷因股波動最大，績優股配息較高。

## 指令

### 買賣

```
/買股 股票代號 數量
/賣股 股票代號 數量
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `股票代號` | 例如 TSPP / UPPI / EGPP / CTPP / MTKP | 是 |
| `數量` | 股數（賣出可填 `all` 全賣） | 是 |

### 查詢

```
/持股
/股歷 股票代號 [期間]
/配息紀錄 [期間]
```

| 指令 | 說明 |
| --- | --- |
| `/持股` | 查看持股、損益與總市值 |
| `/股歷 股票代號 [期間]` | 查看某股票的價格走勢圖 |
| `/配息紀錄 [期間]` | 查看你領取的配息紀錄 |

## 市場規則

| 項目 | 數值 |
| --- | --- |
| 開盤時間 | 每日 09:00–21:00（台北時間） |
| 價格跳動 | 每 5 分鐘更新一次 |
| 交易手續費 | 1%（最低 5 逼幣） |
| 每人單檔持股上限 | 500 股 |
| 配息時間 | 每週一 09:00 |

<Callout type="info">
  每日開盤時市場會輪換「多頭／空頭／盤整」情緒，影響整體漲跌方向；個股仍會依自身波動度起伏。
</Callout>

<Callout type="warn">
  股票有漲有跌，投資前請評估風險。手續費買賣都要收，頻繁交易會侵蝕獲利。
</Callout>
