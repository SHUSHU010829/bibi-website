# 蔬菜的用途

收成的蔬菜會放進你的**菜籃**（`/背包` → 🌾 農場 分類），有兩種去化方式：

## 1. 烹飪成 buff 食物

蔬菜可以走兩條路線——**純農作物配方**（收成後直接煮）或**蔬菜＋魚混合配方**（雙倍效益）：

<Callout type="info">
  **每次烹飪都會自動產出 1 份 🍂 廚餘堆肥**放進背包——所以「煮飯」本身就在累積肥料。常烹飪的人，農場根本不缺堆肥。
</Callout>

## 2. 賣給系統

```
/賣出 物品:<蔬菜> [數量]
```

蔬菜售價會像礦石／魚一樣**每日浮動**（在系統基礎價的 **0.7×～1.3×** 之間隨日期變動）：

| 蔬菜 | 系統基礎價 |
| --- | --- |
| 🥕 紅蘿蔔 | 30 逼幣 |
| 🌽 玉米 | 90 逼幣 |
| 🍓 草莓 | 250 逼幣 |
| 🌹 黑玫瑰 | 900 逼幣 |

每天 00:00（台北時間）凍結當日各蔬菜的市價，整日不變；`/背包` 的「🌾 農場」菜籃估值與 `/賣出 蔬菜` 都以當日市價計算。

`/背包` 的「🌾 農場」分類也有「賣全部」一鍵按鈕。

<Callout type="info">
  烹飪通常**比直接賣值更多**，尤其是草莓和黑玫瑰——所以蔬菜不夠時優先煮飯，多到溢出再賣。
</Callout>

## 跨系統循環

農場是把所有系統串起來的中樞：

```
挖礦 → 煤炭 ──┐
釣魚 → 小雜魚 / 章魚 / 月光露水 ──┐
                                                              ├──► 農場（肥料 / 加速）→ 蔬菜
地下城 → 怪物黏液 / 黑玫瑰種子 ──┘                                                     │
                                                                                                                            ▼
                            烹飪（魚 + 蔬菜）─────────────────► 廚餘堆肥（回流農場）
                                                  │
                                                  ▼
                                      farm_yield buff（提升下次收成）
```

## 每週／每日任務

農場帶四個固定任務，順手就能多領一點逼幣：

| 任務 | 條件 | 獎勵 |
| --- | --- | --- |
| 晨間耕作（日） | 種植 ≥ 1 次 | 100 逼幣 |
| 今日豐收（日） | 收成 ≥ 2 次 | 200 逼幣 |
| 週末市集（週） | 累積收成 ≥ 10 次 | 1,500 逼幣 |
| 黑玫瑰栽培家（週） | 黑玫瑰 ≥ 1 次 | 2,500 逼幣 |

任務進度可在 `/逼幣任務` 查看。
