# 怪物入侵（Raider）

田鼠、烏鴉、野豬之類的小傢伙會**隨機**跑來偷吃你的莊稼——種下 **30 分鐘後**起，每次打開 `/農場` 都有 **8% 機率**觸發入侵，**不再綁成熟時間**，成長中與已成熟的地塊都可能被偷襲。

入侵中的地塊狀態變成 ⚔️ **被入侵**，無法直接收成。你必須點「防禦」按鈕戰鬥：

- 消耗 **1 點地下城體力** ＋ **武器耐久 1**（需先到 `/合成` 打一把劍）
- 勝率 = 你的攻擊力 / 怪物 HP（夾在 20%–90%）
- **勝利** → 怪物退散，作物**恢復到原本狀態**（還在長就繼續長，已成熟就可立即收成），獎勵 **200–600 逼幣** ＋ **怪物黏液 1–2** 顆
- **失敗** → 作物被啃光，地塊清空

<Callout type="warn">
  入侵是隨機事件，沒辦法靠提早收成完全避免。把「打怪換黏液」當成順便的副業，需要肥料時就放著等怪物來。
</Callout>

## 高級陷阱（被動防護）

戰鬥防禦勝利時 **20% 機率**掉「損壞的陷阱碎片」（除了原本的金幣與黏液之外）。集 5 個碎片到工坊合成「🪤 高級陷阱」會**自動使用**：

- 每張高級陷阱 = **+4 次被動抵擋**
- 同時最多保護 **12 次**（多餘的合成會棄掉）
- 打開 `/農場` 時，raid 觸發**先扣高級陷阱**，扣到 0 才會真正進入被入侵狀態
- 抵擋掉的次數會在農場 Container 頂端顯示（「🪤 高級陷阱抵擋了 X 次本次來犯」）

期望取得難度：20% 掉率 × 5 碎片 ≈ **25 場戰鬥勝利**換 1 張 = 4 次保護，不至於讓你完全脫離農田 raid 系統。
