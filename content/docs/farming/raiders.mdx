# 怪物入侵（Raider）

田鼠、烏鴉、野豬之類的小傢伙會**隨機**跑來偷吃你的莊稼。入侵**在種植當下就一次決定命運**：每批作物種下時有 **35% 機率**排定一個入侵時刻（落在種下 **30 分鐘後** ～ 凋萎前之間），沒抽中就這批完全平安。**不再綁成熟時間**，成長中與已成熟的地塊都可能被偷襲。

<Callout type="info">
  **看幾次都不影響命運**：入侵不是每次開 `/農場` 才擲骰，而是種植時就抽好、藏在背後。之後不論你用 `/農場 查看`、`/農場 收成` 或「一鍵收成」，只要時間一到就會兌現成入侵——所以再也不能靠「不看農場」或「直接收成」繞過。用 `/收成` 時若剛好撞上入侵，會先擋下這次收成、把地塊翻成被入侵狀態。
</Callout>

入侵中的地塊狀態變成 ⚔️ **被入侵**，無法直接收成。你必須點「防禦」按鈕戰鬥：

- 消耗 **1 點地下城體力** ＋ **武器耐久 1**（需先到 `/合成` 打一把劍）
- 勝率 = 你的攻擊力 / 怪物 HP（夾在 20%–90%）
- **勝利** → 怪物退散，作物**恢復到原本狀態**（還在長就繼續長，已成熟就可立即收成），獎勵 **400–1,000 逼幣** ＋ **怪物黏液 2–4** 顆
- **失敗** → 作物被啃光，地塊清空，但還是能**撿回殘局** **80–200 逼幣**（外加 **0–1** 顆怪物黏液），不至於全損

<Callout type="warn">
  入侵是隨機事件，沒辦法靠提早收成完全避免。把「打怪換黏液」當成順便的副業，需要肥料時就放著等怪物來。
</Callout>

<Callout type="info">
  **沒體力也能碰運氣**：體力用完時，防禦鈕會改成「擺陷阱」——不耗體力與武器，但只有 **30%** 成功率。成功擊退怪物撿 **120–300 逼幣**，失敗作物被啃光仍能撿回 **30–80 逼幣**。想穩就乖乖等體力回復再用「防禦」。
</Callout>

## 高級陷阱（被動防護）

戰鬥防禦勝利時 **30% 機率**掉「損壞的陷阱碎片」（除了原本的金幣與黏液之外）。集 5 個碎片到工坊合成「🪤 高級陷阱」會**自動使用**：

- 每張高級陷阱 = **+4 次被動抵擋**
- 同時最多保護 **12 次**（多餘的合成會棄掉）
- 入侵兌現時（不論是查看或收成路徑觸發）**先扣高級陷阱**，扣到 0 才會真正進入被入侵狀態
- 抵擋掉的次數會在農場 Container 頂端顯示（「🪤 高級陷阱抵擋了 X 次本次來犯」）

期望取得難度：30% 掉率 × 5 碎片 ≈ **17 場戰鬥勝利**換 1 張 = 4 次保護，不至於讓你完全脫離農田 raid 系統。
