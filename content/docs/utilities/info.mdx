# 生活資訊

## 天氣

```
/天氣 縣市名稱
/全台天氣
```

| 指令 | 參數 | 說明 |
| --- | --- | --- |
| `/天氣` | `縣市名稱`（必填） | 查詢單一縣市的今日天氣 |
| `/全台天氣` | — | 查看全台各地天氣總覽 |

## 加密貨幣

```
/加密貨幣 貨幣代碼
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `貨幣代碼` | 幣種代碼，例如 BTC、ETH、SOL | 是 |

查詢加密貨幣的即時美金報價（資料來源：CryptoCompare）。

## 即時匯率

```
/即時匯率 [欲兌貨幣]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `欲兌貨幣` | ISO 貨幣代碼，例如 JPY、USD、EUR | 否 | 不填則顯示美元對台幣 |

查詢各貨幣兌換新台幣的匯率（資料來源：RTER.info）。
