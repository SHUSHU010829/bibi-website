# 地點推薦

在指定頻道分享的店家／地點會被自動收錄並分類，之後可以隨時查詢。

## 查詢

```
/推薦 查詢 [關鍵字] [類別] [地區]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `關鍵字` | 搜尋關鍵字 | 否 |
| `類別` | 推薦類別（餐廳／酒吧／飲料／娛樂等） | 否 |
| `地區` | 篩選地區 | 否 |

不帶條件可瀏覽全部推薦，帶條件則進行篩選。

## 編輯

```
/推薦 編輯 訊息連結 [類別] [料理]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `訊息連結` | 要編輯的推薦訊息連結 | 是 |
| `類別` | 重新指定類別 | 否 |
| `料理` | 標註料理類型 | 否 |

<Callout type="info">
  推薦會自動依內容分類；若分類不正確，可用 `/推薦 編輯` 手動修正。刪除或重新分析推薦需要管理權限。
</Callout>
