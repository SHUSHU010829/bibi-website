# 趣味指令

## 抽籤

```
/抽籤 [諮詢方向]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `諮詢方向` | 想抽籤的主題 | 否 |

抽一支傳統籤詩風格的籤，回傳籤號、生肖、日期與籤詩解讀。

## 隨機選擇

```
/二選一 選擇一 選擇二 [選擇三] [選擇四] [選擇五]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `選擇一` / `選擇二` | 前兩個選項 | 是 |
| `選擇三` ~ `選擇五` | 額外選項 | 否 |

讓機器人幫你從 2–5 個選項中隨機挑一個。

## 是非題

```
/我想問 問題
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `問題` | 想問的是非題，例如「今天該加班嗎？」 | 是 |

機器人會回答 Yes / No / Maybe。

## 情勒文

```
/生成情勒文
/新增情勒文 內文
```

| 指令 | 參數 | 說明 |
| --- | --- | --- |
| `/生成情勒文` | — | 從情勒文庫隨機抽一篇 |
| `/新增情勒文` | `內文`（必填） | 把你的情勒大作加入庫存 |

## 連結自動修正

當你在頻道貼上 Twitter（X）或 Threads 連結時，機器人會自動改寫成預覽更完整的鏡像連結（例如 fxtwitter / fixthreads），讓圖片與影片能正常顯示。這是自動功能，不需任何指令。
