# 實用工具

除了遊戲與經濟系統，逼逼機器人還有一票好用又好玩的工具：

- [吃什麼 / 食物清單](/docs/utilities/food) — 選擇障礙救星，幫你決定吃什麼、喝什麼。
- [地點推薦](/docs/utilities/recommendations) — 查詢與管理餐廳、酒吧、飲料、娛樂推薦。
- [生活資訊](/docs/utilities/info) — 天氣、加密貨幣報價、即時匯率。
- [趣味指令](/docs/utilities/fun) — 抽籤、隨機選擇、是非題、情勒文與連結修正。
