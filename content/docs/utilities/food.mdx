# 吃什麼

有選擇障礙嗎？讓逼逼機器人幫你決定吃什麼、喝什麼。

## `/吃什麼`

```
/吃什麼 [類別] [飲料店]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `類別` | 選擇食物類別 | 否 | 不選則從所有食物與飲料中隨機 |
| `飲料店` | 指定飲料店（會自動視為飲料類別） | 否 | — |

## 查清單與排行

### 食物清單

```
/food list [類別]
```

查看目前的食物清單（`類別` 選填）。

### 食物排行榜

```
/food ranking [類別] [數量]
```

查看最受歡迎的食物排行榜。`類別` 選填；`數量` 選填，預設 10、最多 20。

### 飲料店清單

```
/food stores
```

查看所有可用的飲料店清單。

<Callout type="info">
  想新增食物或匯入整間飲料店的菜單，需要管理權限，詳見[管理員指令](/docs/admin-commands)。
</Callout>
