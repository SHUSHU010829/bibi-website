# 賭場總覽

用逼幣在賭場小試手氣！賭場提供多種遊戲，每種都有自己的下注上下限與賠率規則。

<Callout type="warn">
  賭博有風險，所有遊戲長期期望值對玩家不利（莊家優勢）。請量力而為，把它當娛樂而非賺錢手段。
</Callout>

<Callout type="info">
  賭場與[樂透](/docs/lottery)指令都在 **拉斯維加斯 │ 🃏** 或 **濱海灣金沙 │ 🃏** 頻道使用，詳見[指令頻道規範](/docs/getting-started#指令頻道規範)。
</Callout>

## 排行榜與個人紀錄

```
/排行榜 類別:賭場贏家
/排行榜 類別:賭場輸家
```

進入後可用上方下拉切換類別、用下方按鈕切換「今日 / 本週 / 本月」。

```
/檔案
```

到「🎰 賭場紀錄」分頁，顯示你的下注次數、總派彩、投報率（RTP）以及各遊戲的盈虧明細。

## 再玩一次

許多賭場遊戲結算後會提供「再玩一次」按鈕，可用相同的下注快速再開一局，省去重打指令。
