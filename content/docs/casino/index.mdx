## 排行榜與個人紀錄

```
/排行榜 類別:賭場贏家
/排行榜 類別:賭場輸家
```

進入後可用上方下拉切換類別、用下方按鈕切換「今日 / 本週 / 本月」。

```
/檔案
```

到「🎰 賭場紀錄」分頁，顯示你的下注次數、總派彩、投報率（RTP）以及各遊戲的盈虧明細。

## 再玩一次

許多賭場遊戲結算後會提供「再玩一次」按鈕，可用相同的下注快速再開一局，省去重打指令。
