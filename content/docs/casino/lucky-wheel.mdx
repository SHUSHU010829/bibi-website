# 幸運轉盤

轉動命運之輪，停在哪格就乘上該格倍率，附旋轉動畫。

```
/賭場 幸運轉盤 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

下注限制：最低 10、最高 5000 逼幣。

## 倍率與規則

轉盤分為多格，各格倍率不同且為**加權**（高倍率格出現機率較低）。

| 格子 | 倍率 |
| --- | --- |
| 槓龜 | ×0 |
| 小獎 | ×1.2 |
| 小獎 | ×1.5 |
| 中獎 | ×2 |
| 中獎 | ×3 |
| 大獎 | ×5 |
| 大獎 | ×10 |
| 頭獎 | ×50 |

派彩為「下注金額 × 停下的格子倍率」。整體投報率（RTP）約 93%。

  按下指令後會播放轉盤旋轉的 GIF 動畫，停下後依停在的格子結算。倍率越高的格子越罕見。
</Callout>
