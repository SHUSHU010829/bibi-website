# 幸運轉盤

轉動命運之輪，停在哪格就乘上該格倍率，附旋轉動畫。

```
/賭場 幸運轉盤 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

下注限制：最低 10、最高 5000 逼幣。

## 倍率與規則

轉盤共 17 格、各格倍率不同且為**加權**（高倍率格出現機率較低）。**沒有槓龜**——最低也會拿回一部分本金（< 1 的「少賠」格），只是賠多賠少。

| 類型 | 倍率 |
| --- | --- |
| 少賠（拿回部分本金） | ×0.2 / ×0.3 / ×0.5 / ×0.6 / ×0.7 / ×0.8 |
| 小賺 | ×1.2 / ×1.5 |
| 中獎 | ×2 / ×2.5 / ×3 / ×4 |
| 大獎 | ×6 / ×10 |
| 頭獎 | ×25 / ×50 / 大獎 ×100 |

派彩為「下注金額 × 停下的格子倍率」。整體投報率（RTP）約 90%。

<Callout type="info">
  按下指令後會播放轉盤旋轉的 GIF 動畫，停下後依停在的格子結算。沒有全輸的槓龜格，但多數情況會「少賠」，靠少數高倍率格翻身。倍率越高的格子越罕見。
</Callout>
