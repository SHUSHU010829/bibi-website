# 踩地雷（Mines）

在 20 格的盤面上逐格翻開，避開地雷。翻到安全格倍率就往上加，隨時可以收手；踩到地雷本局全失。

```
/賭場 踩地雷 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

## 規則

| 項目 | 數值 |
| --- | --- |
| 最低 / 最高下注 | 10 / 5000 逼幣 |
| 莊家優勢 | 2% |
| 盤面 | 5 × 4 = 20 格 |
| 地雷數 | 5 顆 |
| 每局時限 | 5 分鐘 |

- 盤面有 20 格、藏 5 顆地雷，逐格翻開：翻到安全格 → 倍率累積、可繼續翻或收手；翻到地雷 → 本局全失。
- 至少翻過 1 格才能收手；翻完所有安全格則封頂大獎。
- 倍率以「連續避雷的機率倒數」計算，翻越多格、倍率成長越誇張（翻 1 格約 ×1.3、翻 5 格約 ×5，越後面越驚人）。

<Callout type="info">
  踩地雷的莊家優勢只在最終倍率上收一次，因此**不論在哪個時間點收手，長期投報率都維持在約 98%**。
</Callout>
