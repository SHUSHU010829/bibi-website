# 射龍門

先看兩張「門柱」牌，再決定要不要押注第三張牌會落在兩者之間。

```
/賭場 射龍門
```

此指令不需參數，開局後透過按鈕操作。

## 玩法

1. 支付**入場費 50 逼幣**，發出兩張門柱牌。
2. 看完兩張柱牌後，決定要押多少注「射龍門」。
3. 翻出第三張牌：
   - 點數落在兩張柱牌**之間** → 獲勝派彩。
   - 落在門柱**之外** → 輸掉押注。
   - 剛好**撞柱**（等於門柱點數） → 加倍賠付（懲罰）。

## 限制

| 項目 | 數值 |
| --- | --- |
| 入場費 | 50 逼幣 |
| 最低 / 最高押注 | 50 / 1000 逼幣 |
| 莊家優勢 | 5% |
| 每局時限 | 5 分鐘 |

<Callout type="info">
  兩張柱牌差距越大，中間範圍越廣，獲勝機率越高、賠率越低；差距小則高風險高賠率。
</Callout>
