# 刮刮樂（Scratch）

台灣彩券最經典的「對中獎號碼」玩法。卡面上方有 3 個幸運號碼，下方 3×3 是你的號碼，每格各帶一個倍率。刮開你的號碼，只要對中任一個幸運號碼，就贏該格的倍率。

```
/賭場 刮刮樂 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 買卡金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

## 規則

| 項目 | 數值 |
| --- | --- |
| 最低 / 最高下注 | 10 / 2000 逼幣 |
| 投報率（RTP） | 約 84% |
| 中獎率 | 約 32% |
| 每張時限 | 5 分鐘 |

- 上方 3 個**幸運號碼**直接明示，不用刮。
- 下方 9 格是**你的號碼**，逐格刮開或按「全部刮開」一次揭曉。
- 你的號碼若對中任一幸運號碼 → 中該格倍率；對中的格子會以金框綠底標出。
- 中獎與否在買卡當下就抽定（如同真實的預印刮刮樂），刮開只是揭曉過程。

<Callout type="warn">
  刮刮樂屬於高莊家優勢的即開型遊戲（投報率約 84%），中獎率高但多為小獎，偶有大獎。當作即開即爽的小賭怡情即可。
</Callout>
