# HI-LO

猜下一張牌比目前的牌「大（HI）」、「小（LO）」還是「一樣（SAME）」，猜中可連續累積。

```
/賭場 猜大小 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

## 規則

| 項目 | 數值 |
| --- | --- |
| 最低 / 最高下注 | 10 / 1000 逼幣 |
| 莊家優勢 | 5% |
| 最多連勝 | 10 連 |
| 每局時限 | 5 分鐘 |

- 開局翻出一張底牌，你預測下一張的大小。
- 賠率會依牌堆中剩餘牌的組成動態計算（剩越少符合條件的牌，賠率越高）。
- 連續猜中時派彩會**累乘**，最多累積 10 輪；任何時候都可以選擇「收手」鎖定目前的派彩。
- 一旦猜錯，本局押注全失。

<Callout type="warn">
  連勝越長，累積派彩越誘人，但只要一次猜錯就全部歸零。記得適時收手。
</Callout>
