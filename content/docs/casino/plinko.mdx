# 彈珠台（Plinko）

彈珠從釘板頂端落下，每碰到一排釘子就隨機往左或往右，最後落進底部某一格，乘上該格倍率即為派彩。一翻定生死，沒有收手。

```
/賭場 彈珠台 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

## 規則

| 項目 | 數值 |
| --- | --- |
| 最低 / 最高下注 | 10 / 5000 逼幣 |
| 風險 / 排數 | 中風險・8 排（固定） |
| 投報率（RTP） | 約 99% |
| 落點格數 | 9 格 |

- 下注後彈珠落下，每排往左或往右各 50%。
- 落點越靠**邊緣**倍率越高、越靠**中央**越容易賠（中央格倍率小於 1）。
- 8 排的倍率分布：邊緣 ×13、往內 ×3、×1.3、×0.7，最中央 ×0.4。

<Callout type="info">
  彈珠台是即時結算的遊戲，每次落球都是獨立的一翻，沒有收手或連續機制。結算後可用「再玩一次」快速再來。
</Callout>
