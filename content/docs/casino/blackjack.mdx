# 二十一點

跟莊家比點數，越接近 21 點但不爆牌者勝。

```
/二十一點 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

下注限制：最低 10、最高 1000 逼幣。每局限時 5 分鐘內操作完成。

## 操作

開局後用按鈕操作：

| 按鈕 | 說明 |
| --- | --- |
| 要牌（Hit） | 再拿一張牌 |
| 停牌（Stand） | 停止拿牌，換莊家行動 |
| 加倍（Double） | 加倍下注並只再拿一張牌 |
| 分牌（Split） | 兩張同點數時可拆成兩手分別進行 |

## 規則與賠率

| 規則 | 說明 |
| --- | --- |
| 莊家停牌 | 點數 ≥ 17 必停（含軟 17，stand on soft 17） |
| Blackjack | 起手兩張即 21 點，拿回 **2.5 倍**（賠率 3:2） |
| 一般獲勝 | 拿回 **2 倍**（賠率 1:1） |
| 過五關（Five-Card Charlie） | 持有 5 張牌未爆即自動獲勝，拿回 2 倍 |
| 莊家過五關 | 莊家拿到 5 張未爆牌則莊家勝（玩家 BJ／過五關優先結算，不受影響） |
| 和局（Push） | 點數相同退回本金 |

<Callout type="info">
  「拿回」金額包含本金。例如下注 100 取得 Blackjack，拿回 250（淨賺 150）。
</Callout>
