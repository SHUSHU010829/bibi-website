# 火箭（Crash）

火箭升空，倍率隨時間不斷上升，越晚收手賺越多——但火箭隨時可能爆炸，慢一步就血本無歸。

```
/火箭 下注 [自動收手]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（可輸入數字或「梭哈」） | 是 | — |
| `自動收手` | 設定自動收手的倍率，到此倍率自動鎖定派彩 | 否 | 不自動收手 |

## 玩法

- 下注後火箭起飛，倍率從 1.00× 開始往上爬。
- 按「收手」即可用當下倍率鎖定派彩（下注 × 當前倍率）。
- 若在收手前火箭爆炸，本局下注全失。
- 可預設 `自動收手` 倍率，到點自動結算，避免手速不夠。

## 限制與開放時段

| 項目 | 數值 |
| --- | --- |
| 最低 / 最高下注 | 10 / 5000 逼幣 |
| 莊家優勢 | 2% |
| 兩局間冷卻 | 5 分鐘 |
| 開放時段 | **每週六 21:00–24:00（台北時間）** |

<Callout type="warn">
  火箭是**限時開放**的遊戲，只有每週六晚上 21:00 到 24:00 才能玩。其他時間使用指令會提示尚未開放。
</Callout>
