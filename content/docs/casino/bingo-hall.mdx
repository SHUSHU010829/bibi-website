# 賓果大廳

排程式的群組賓果大廳，**免人開房**，固定間隔自動開球。買卡參與，依連線數分彩池，還有累積頭彩。

```
/賓果大廳 買卡
/賓果大廳 資訊
```

| 子指令 | 用途 |
| --- | --- |
| `/賓果大廳 買卡` | 買一張本場賓果卡 |
| `/賓果大廳 資訊` | 查看本場彩池、開球進度與倒數 |

## 規則與限制

| 項目 | 數值 |
| --- | --- |
| 每張卡 | 50 逼幣 |
| 開球間隔 | 預設每 6 小時自動開球 |
| 每場開球數 | 30 顆 |
| 投報率（RTP） | 約 88% |

- 大廳**固定間隔自動開球**，不需要任何人手動開房或開球。
- 一張卡 50 逼幣，買越多張中獎機會越高。
- **連線獎**：依你的卡完成的連線數，按比例分配彩池。
- **累積頭彩**：由本場連線最多者獨得；若整場無人連線，則滾存到下一場累積。
- 中獎者會收到私訊「Bingo 兌獎圖」。

<Callout type="info">
  累積頭彩會隨無人中獎的場次不斷累積，越滾越大。買卡後耐心等下次開球，結算後私訊通知中獎。
</Callout>
