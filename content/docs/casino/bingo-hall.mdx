# 賓果大廳

排程式的群組賓果大廳，**免人開房、免指令**，固定間隔自動開球。直接在公告頻道按按鈕買卡，依連線數分彩池，還有累積頭彩。

## 怎麼玩

每一場開賣時，機器人會在賓果大廳公告頻道貼出一則訊息，下方有買卡按鈕：

- 點 **買 1 張 / 5 張 / 10 張 / 20 張** 即可直接購入，會私訊回覆你買了幾張、本場目前持有幾張。
- 訊息上會顯示本場累積頭彩、單張價格與開球倒數。
- 時間到自動開球，舊訊息按鈕關閉、發布開球結果，並接著貼出下一場的買卡訊息——完全不需要輸入任何指令。

## 規則與限制

| 項目 | 數值 |
| --- | --- |
| 每張卡 | 50 逼幣 |
| 開球間隔 | 預設每 6 小時自動開球 |
| 每場開球數 | 30 顆 |
| 投報率（RTP） | 約 88% |

- 大廳**固定間隔自動開球**，不需要任何人手動開房或開球。
- 一張卡 50 逼幣，買越多張中獎機會越高。
- **連線獎**：依你的卡完成的連線數，按比例分配彩池。
- **累積頭彩**：由本場連線最多者獨得；若整場無人連線，則滾存到下一場累積。
- 中獎者會收到私訊「Bingo 兌獎圖」。

<Callout type="info">
  累積頭彩會隨無人中獎的場次不斷累積，越滾越大。買卡後耐心等下次開球，結算後私訊通知中獎。
</Callout>
