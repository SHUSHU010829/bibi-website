# 魚蝦蟹

傳統的魚蝦蟹（骰寶變體），擲三顆符號骰，押你看好的符號出現幾顆。

```
/賭場 魚蝦蟹 符號 金額 [梭哈] ...
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `符號` | 押注的符號（🐟 魚、🦐 蝦、🦀 蟹、🍐 葫蘆、🪙 金錢、🐓 雞） | 是 |
| `金額` | 該注的下注金額（最低 10） | 是 |
| `梭哈` | 把可用餘額全押 | 否 |

每顆骰子有六個符號面（魚🐟、蝦🦐、蟹🦀、葫蘆、金錢、雞）。可一次押多種符號，最多同時 3 注。下注限制：最低 10、最高 1000 逼幣。

## 賠率表

押中的符號出現幾顆，就賠幾倍（下表為**純獎金倍率**，不含本金）。

| 中的顆數 | 賠率 |
| --- | --- |
| 1 顆 | ×1 |
| 2 顆 | ×2 |
| 3 顆 | ×3 |

押的符號完全沒出現則該注輸掉。

<Callout type="info">
  結算時會附上三顆骰子的圖卡，顯示開出的符號。出現越多顆你押的符號，派彩越高。
</Callout>
