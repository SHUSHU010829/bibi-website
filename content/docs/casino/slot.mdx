# 拉霸

經典的拉霸機（吃角子老虎），轉動轉軸湊出相同符號就中獎。

```
/拉霸 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 5） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

## 下注限制

| 項目 | 數值 |
| --- | --- |
| 最低下注 | 5 逼幣 |
| 最高下注 | 500 逼幣 |
| 每日虧損保護 | 2000 逼幣 |

<Callout type="info">
  **每日虧損保護**：拉霸單日的淨虧損超過一定額度後會受到保護，避免一天輸太多。
</Callout>

## Jackpot 累積獎池

拉霸設有全伺服器共享的 **Jackpot 累積獎池**：

- 每次下注會提撥 **3%** 注入獎池。
- 獎池初始種子金額為 5000 逼幣。
- 獎池累積達里程碑（10000、25000、50000、100000）時，機器人會在公告頻道宣布。
- 轉出 Jackpot 組合即可獨得整個獎池！
