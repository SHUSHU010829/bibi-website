# 紅包

在頻道發一包逼幣紅包，大家限時搶。屬於群組玩法，無莊家優勢（純分配，發多少搶多少）。

```
/賭場 紅包 金額 包數 [手氣]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `金額` | 紅包總金額（最少 100） | 是 | — |
| `包數` | 拆成幾包（最多 20 包） | 是 | — |
| `手氣` | 分配方式：拚手氣（隨機）或均分 | 否 | 拚手氣 |

## 規則

| 項目 | 數值 |
| --- | --- |
| 最少金額 | 100 逼幣 |
| 最多包數 | 20 包 |
| 限時搶 | 預設 10 分鐘 |

- **拚手氣**：每包金額隨機分配，手氣好的搶到大包。
- **均分**：每包金額相同，先搶先得。
- 在頻道按按鈕搶包，每人限搶一包，搶完或時間到即結束。
- 時限內**未搶完的包數會自動退回**給發包人。

<Callout type="info">
  紅包是純分配玩法，沒有莊家抽水，發出去的總額會完整分給搶到的人，未搶完的部分原額退回。
</Callout>
