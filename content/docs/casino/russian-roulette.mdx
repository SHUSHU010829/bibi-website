# 俄羅斯輪盤

多人左輪對決（俄羅斯輪盤），大家押相同賭注進池，限時自動開轉，隨機一人「中彈」輸光，其餘倖存者平分池底。屬於群組玩法。

```
/賭場 左輪 賭注
```

| 參數 | 說明 | 必填 | 範圍 |
| --- | --- | --- | --- |
| `賭注` | 每人投入的相同賭注 | 是 | 50–5000 |

開局後，其他人可在限時內按按鈕加入，所有人押**相同**賭注進入池底。

## 規則與限制

| 項目 | 數值 |
| --- | --- |
| 賭注範圍 | 50–5000 逼幣 |
| 參與人數 | 最多 6 人 |
| 房水（抽成） | 5% |

- 招募時間結束後**自動開轉**，隨機抽出一人「中彈」，其賭注輸光。
- 其餘**倖存者平分池底**（扣 5% 房水後分配）。
- 人越多、賭注越大，倖存者分得越多。

<Callout type="info">
  中彈者由系統隨機決定，每位參與者中彈機率相同。池底扣 5% 房水後由倖存者均分。
</Callout>
