# 賓果（Bingo）

單人即時版賓果，買一張 5×5 賓果卡，系統開球後依完成的連線數結算。

```
/賭場 賓果 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 買卡金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

下注限制：最低 10、最高 2000 逼幣。

## 規則

- 賓果卡為 **5×5**，分 **B / I / N / G / O** 五欄，中央格為 **FREE**（免費已對中）。
- 系統一次開 **30 顆球**，對中卡上號碼。
- 依完成的**連線數**（橫、直、斜，共 12 條線）結算派彩。
- 即時結算，開完球當下就揭曉中了幾條線與派彩。

## 賠率

完成的連線數越多，賠率越高（0～12 條線）。連線數低時可能僅退回部分或不賠，連滿（12 條）為最大獎。
