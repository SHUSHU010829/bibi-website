# 百家樂

經典的百家樂（Baccarat），押莊家或閒家哪邊點數較接近 9，可同時下多種押注。

```
/賭場 百家樂 押法 金額 [梭哈] ...
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `押法` | 押注類型（莊、閒、和、莊對、閒對） | 是 |
| `金額` | 該注的下注金額（最低 10） | 是 |
| `梭哈` | 把可用餘額全押 | 否 |

可一次下多組押注，最多同時 3 注。下注限制：最低 10、最高 50000 逼幣。

## 賠率表

下表的賠率為**含本金的派彩倍率**。

| 押法 | 說明 | 賠率 |
| --- | --- | --- |
| 莊 | 莊家贏（抽 5% 水） | 1.95:1 |
| 閒 | 閒家贏 | 2:1 |
| 和 | 莊閒點數相同 | 9:1 |
| 莊對 | 莊家前兩張同點 | 12:1 |
| 閒對 | 閒家前兩張同點 | 12:1 |

<Callout type="info">
  押「莊」獲勝抽 5% 水，故淨獎金為 0.95:1；押「閒」獲勝則為 1:1。出現「和」時，押莊／閒退回本金。
</Callout>

## 補牌規則

採標準百家樂補牌規則：依閒家、莊家前兩張的點數決定是否補第三張牌，玩家不需手動操作，系統自動結算。

- 點數計算：A 計 1 點，10／J／Q／K 計 0 點，其餘依面值，總和取個位數。
- 任一方前兩張為 8 或 9 點（天牌）則直接定勝負，不再補牌。

結算時會附上牌桌圖卡，呈現莊閒雙方的牌與點數。
