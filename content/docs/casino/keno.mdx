# 尋寶（Keno）

從 20 格中挑選 5 格藏寶位置，看你猜中幾個寶藏。

```
/賭場 尋寶 [下注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `下注` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

開局後在 20 格的盤面上**選 5 格**（可手動或隨機），開獎後比對命中數量。

## 賠付表

下表倍率為**派彩倍率**（押注金額 × 倍率為拿回總額）。

| 命中數 | 倍率 |
| --- | --- |
| 0–1 | ×0（沒中） |
| 2 | ×2 |
| 3 | ×4 |
| 4 | ×20 |
| 5 | ×400 |

## 限制

| 項目 | 數值 |
| --- | --- |
| 最低 / 最高下注 | 10 / 50000 逼幣 |
| 每局時限 | 5 分鐘 |

<Callout type="info">
  全中 5 格可獲得 ×400 的高額派彩，但機率極低。命中 2 格以上才有獎金。
</Callout>
