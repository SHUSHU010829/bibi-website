# 賽馬

開一場賽馬，售票期間大家進來押注自己看好的馬，時間到自動開賽（含文字動畫）。

```
/賽馬
```

開出一場賽馬後，所有人都可以在售票時間內透過按鈕進來押注。

## 馬匹與賠率

共有 6 匹馬，奪冠機率與賠率（含本金的派彩倍率）如下：

| 馬 | 奪冠機率 | 賠率 |
| --- | --- | --- |
| 🐎 閃電 | 30% | ×3.0 |
| 🐴 黑風 | 22% | ×4.0 |
| 🦄 金箭 | 17% | ×5.5 |
| 🦌 銀月 | 13% | ×7.0 |
| 🐂 紅炎 | 10% | ×9.0 |
| 🦓 夜影 | 8% | ×11.0 |

押中冠軍可得「下注金額 × 賠率」的派彩（整體約 10% 莊家優勢）。

## 限制與時間

| 項目 | 數值 |
| --- | --- |
| 最低 / 最高下注 | 10 / 1000 逼幣 |
| 售票（下注）時間 | 10 分鐘 |
| 賽事有效時間 | 30 分鐘 |

<Callout type="info">
  賠率越高的馬奪冠機率越低。售票時間結束後自動開賽，賽果以文字動畫呈現並公告派彩。
</Callout>
