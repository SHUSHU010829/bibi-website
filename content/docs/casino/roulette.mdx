# 輪盤

經典的輪盤遊戲，在轉盤停下前下注，猜中號碼或區域即可派彩。

```
/輪盤 金額 [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `金額` | 單注金額（每注最低 30） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

下注後透過按鈕／選單選擇要押的區域，可同時下多個區域。

## 下注種類

支援常見的輪盤押注：紅／黑、單／雙、大／小、打（dozens）、列（columns）等，不同押注有不同賠率。

## 限制

| 項目 | 數值 |
| --- | --- |
| 每注最低 | 30 逼幣 |
| 單局總下注上限 | 2000 逼幣 |
| 下注時間 | 90 秒 |
| 每局時限 | 5 分鐘 |

<Callout type="info">
  下注時間結束後輪盤即開轉。記得在 90 秒內完成所有押注。
</Callout>
