# 倍率轉盤

押注並選一個倍率，轉盤上有 ×2 / ×5 / ×10 的扇區，中間穿插一堆「0」。**轉到你押的那個倍率**才中獎，獎金 = 下注 × 倍率；轉到 0 或別的倍率都算沒中。

```
/賭場 輪盤 押注 金額 [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `押注` | 押哪個倍率：×2 / ×5 / ×10 | 是 | — |
| `金額` | 下注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

## 玩法

- 轉盤由多個等分扇區組成：較多的 **×2**、較少的 **×5**、最少的 **×10**，其餘為 **0（沒中）**。
- 你押的倍率，轉盤上對應的扇區會被標亮。
- 倍率越高、扇區越少越難中，但中了賠得越多。
- 即時單轉，結算有旋轉動畫。

## 賠率

| 押注 | 中獎賠付 | 大致中獎機率 |
| --- | --- | --- |
| ×2 | 下注 × 2 | 較高 |
| ×5 | 下注 × 5 | 中等 |
| ×10 | 下注 × 10 | 較低 |

三種押注的長期回報率相近（莊家優勢約一成）；數值由設定檔調整。

<Callout type="info">
  押 ×2 最常中但賠得少，押 ×10 最難中但翻得多。中間那一堆 0 就是莊家優勢的來源，量力而為。
</Callout>
