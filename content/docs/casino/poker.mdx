# 德州撲克

單人版德州撲克（Casino Hold'em），你直接對莊家對戰，依牌型賠率表派彩。

```
/賭場 德州撲克 [底注] [梭哈]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `底注` | 底注金額（最低 10） | 否 | 預設最低下注 |
| `梭哈` | 把可用餘額全押 | 否 | False |

下注限制：底注最低 10、最高 5000 逼幣。

## 玩法流程

1. 下底注後，發給你 **2 張底牌**，莊家 **2 張暗牌**，並翻開 **3 張公共牌（flop）**。
2. 看牌後選擇：
   - **跟注**：追加 **2× 底注**，補滿轉牌（turn）與河牌（river），與莊家攤牌比牌型。
   - **蓋牌**：放棄本局，輸掉底注。
3. 莊家需「**一對 4**」以上才**合格**。

## 賠率與結算

莊家合格且你贏，**底注依牌型賠率表**派彩，跟注以 1:1 賠付：

| 牌型 | 底注賠率 |
| --- | --- |
| 皇家同花順 | 100:1 |
| 同花順 | 20:1 |
| 四條 | 10:1 |
| 葫蘆 | 3:1 |
| 同花 | 2:1 |
| 順子（含以下） | 1:1 |

- **莊家合格且你贏**：底注依上表賠付，跟注 1:1。
- **莊家不合格**：跟注**退回**（不賠不輸），底注依上表賠付。
- **莊家合格且你輸**：底注與跟注皆輸。
- **和局**：退回本金。

整體投報率（RTP）約 97.6%，結算時會附上牌桌圖卡。

<Callout type="info">
  跟注金額固定為底注的 2 倍。莊家需有「一對 4」以上才合格；不合格時你的跟注會原額退回，底注仍依牌型賠率結算。
</Callout>
