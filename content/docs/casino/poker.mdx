# 德州撲克

多人連線的德州撲克（Texas Hold'em），開桌後會建立專屬討論串讓玩家入座對戰。

```
/賭場 德州撲克 開桌 max_players blind
```

| 參數 | 說明 | 必填 | 範圍 |
| --- | --- | --- | --- |
| `max_players` | 最多入座人數 | 是 | 2–8 |
| `blind` | 盲注大小 | 是 | 10–500 |

開桌後，機器人會建立一個討論串（thread）作為牌桌，其他玩家可在限定時間內加入，人數足夠即開始。

## 規則與限制

| 項目 | 數值 |
| --- | --- |
| 盲注範圍 | 10–500 逼幣 |
| 入座人數 | 2–8 人 |
| 買入金額 | 盲注的 50 倍 |
| 每人每日買入上限 | 50000 逼幣 |
| 等待入座時限 | 5 分鐘 |
| 每次行動時限 | 60 秒 |
| 牌桌有效時間 | 15 分鐘 |

- 玩法為標準德州撲克：翻牌前（Pre-flop）、翻牌（Flop）、轉牌（Turn）、河牌（River），再比牌型大小。
- 行動逾時未操作會自動蓋牌（Fold）。

<Callout type="info">
  買入金額為盲注的 50 倍，例如盲注 10 則買入 500 逼幣。每人每日買入總額上限為 50000 逼幣。
</Callout>
