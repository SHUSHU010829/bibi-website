# 互動活動

成員可以自掏腰包當獎金，舉辦活動、預測或問答，讓大家一起同樂。

## 活動

舉辦一個自訂活動，由你提供獎金池，機器人協助報名與發獎。

```
/活動 名稱 獎金 名次數 [描述] [最少人數] [最多人數]
```

| 參數 | 說明 | 必填 | 範圍／預設 |
| --- | --- | --- | --- |
| `名稱` | 活動名稱 | 是 | 最多 80 字 |
| `獎金` | 獎金池總額（鎖定用，確認後才扣款） | 是 | — |
| `名次數` | 有幾個得獎名次 | 是 | 1–5 |
| `描述` | 活動說明 | 否 | 最多 500 字 |
| `最少人數` | 達到此人數才能結算 | 否 | 預設 1 |
| `最多人數` | 達上限後拒絕新報名 | 否 | 不填則無上限 |

<Callout type="info">
  **扣款前先確認**：送出 `/活動` 後不會馬上扣錢，而是先跳出一張只有你看得到的確認卡，列出「將鎖定的金額、名次／人數、目前餘額 → 建立後餘額」以及發獎／退款的防洗錢抽成規則，按「確認建立」才真正扣款鎖定獎金池，按「取消」則不扣款。確認卡有 5 分鐘時效。
</Callout>

<Callout type="warn">
  獎金池於**確認後從錢包扣除並鎖定**。若活動取消退款，剩餘款會收 **30% 手續費**再退回主辦人。
</Callout>

### 發獎防洗錢規則

為防止「開活動 → 全額指定發給小帳」淪為繞過轉帳管制的免費搬幣管道，發獎套用與轉帳同一套護欄：

- **獎金抽成**：每筆發出的獎金課 **10% 防洗錢抽成**，得獎者實拿扣抽成後的淨額；但活動**參賽達 5 人**時整場**免抽成**——真實社群活動不受影響。
- **得獎者年資檢查**：結算前檢查每位得獎者，**加入伺服器未滿 7 天**者無法領獎，需重新指定得獎者。
- **金流納入偵測**：活動獎金金流（主辦 → 得獎者）會併入既有的雙向對敲／圈狀轉帳偵測，異常會被標記審查並扣信用分。

## 預測

發起一個預測題，大家先下注選邊，主辦者稍後公布正解，猜中者瓜分獎金。

```
/預測 題目 獎金 選項a 選項b [選項c] [選項d] [分鐘]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `題目` | 預測的問題 | 是 |
| `獎金` | 獎金池總額（立即扣除鎖定） | 是 |
| `選項a` / `選項b` | 至少兩個選項 | 是 |
| `選項c` / `選項d` | 額外選項 | 否 |
| `分鐘` | 開放下注的時間 | 否 |

## 問答

發布一題有獎問答，可選擇平分或搶答模式。

```
/問答 題目 獎金 正確答案 選項a 選項b [選項c] [選項d] [分鐘] [模式]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `題目` | 問題內容 | 是 |
| `獎金` | 獎金池總額（立即扣除鎖定） | 是 |
| `正確答案` | 哪一個選項是正解 | 是 |
| `選項a` / `選項b` | 至少兩個選項 | 是 |
| `選項c` / `選項d` | 額外選項 | 否 |
| `分鐘` | 作答時間 | 否 |
| `模式` | 獎金分配方式 | 否（預設平分） |

| 模式 | 說明 |
| --- | --- |
| 平分獎金 | 所有答對者平分獎金池 |
| 搶答獨佔 | 第一個答對者獨得全部獎金 |

## 紅包

在頻道發一包逼幣紅包，大家限時搶。屬於群組同樂玩法，無莊家抽水——發多少就完整分給搶到的人。

```
/紅包 金額 包數 [手氣]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `金額` | 紅包總金額（最少 100） | 是 | — |
| `包數` | 拆成幾包（最多 20 包） | 是 | — |
| `手氣` | 分配方式：拚手氣（隨機）或均分 | 否 | 拚手氣 |

- **拚手氣**：每包金額隨機分配，手氣好的搶到大包。
- **均分**：每包金額相同，先搶先得。
- 在頻道按按鈕搶包，每人限搶一包，搶完或時間到（預設 10 分鐘）即結束。
- 時限內**未搶完的包數會自動退回**給發包人。
- 可在**公會大廳**（含其聊天討論串）、**賭場遊戲頻道**與**各自的遊戲房**發紅包。

<Callout type="info">
  紅包沒有莊家抽水，發出去的總額會完整分給搶到的人，未搶完的部分原額退回。
</Callout>
