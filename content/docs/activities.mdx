# 互動活動

成員可以自掏腰包當獎金，舉辦活動、預測或問答，讓大家一起同樂。

## 活動

舉辦一個自訂活動，由你提供獎金池，機器人協助報名與發獎。

```
/活動 名稱 獎金 名次數 [描述] [最少人數] [最多人數]
```

| 參數 | 說明 | 必填 | 範圍／預設 |
| --- | --- | --- | --- |
| `名稱` | 活動名稱 | 是 | 最多 80 字 |
| `獎金` | 獎金池總額（立即從你錢包扣除並鎖定） | 是 | — |
| `名次數` | 有幾個得獎名次 | 是 | 1–5 |
| `描述` | 活動說明 | 否 | 最多 500 字 |
| `最少人數` | 達到此人數才能結算 | 否 | 預設 1 |
| `最多人數` | 達上限後拒絕新報名 | 否 | 不填則無上限 |

<Callout type="warn">
  舉辦活動時獎金會**立即從你的錢包扣除並鎖定**。若活動取消退款，可能收取手續費。
</Callout>

## 預測

發起一個預測題，大家先下注選邊，主辦者稍後公布正解，猜中者瓜分獎金。

```
/預測 題目 獎金 選項a 選項b [選項c] [選項d] [分鐘]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `題目` | 預測的問題 | 是 |
| `獎金` | 獎金池總額（立即扣除鎖定） | 是 |
| `選項a` / `選項b` | 至少兩個選項 | 是 |
| `選項c` / `選項d` | 額外選項 | 否 |
| `分鐘` | 開放下注的時間 | 否 |

## 問答

發布一題有獎問答，可選擇平分或搶答模式。

```
/問答 題目 獎金 正確答案 選項a 選項b [選項c] [選項d] [分鐘] [模式]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `題目` | 問題內容 | 是 |
| `獎金` | 獎金池總額（立即扣除鎖定） | 是 |
| `正確答案` | 哪一個選項是正解 | 是 |
| `選項a` / `選項b` | 至少兩個選項 | 是 |
| `選項c` / `選項d` | 額外選項 | 否 |
| `分鐘` | 作答時間 | 否 |
| `模式` | 獎金分配方式 | 否（預設平分） |

| 模式 | 說明 |
| --- | --- |
| 平分獎金 | 所有答對者平分獎金池 |
| 搶答獨佔 | 第一個答對者獨得全部獎金 |
