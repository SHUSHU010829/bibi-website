# 世界事件 🌍

世界事件是**全伺服器**的共同目標。事件不是排程跑出來的，而是玩家在挖礦 / 釣魚 / 收成 / 通關地下城時**有機率觸發**——某個玩家「中獎」就開了一個事件，全服都看得到、都能一起捐物資。

達成目標後，全服啟動一個 **24 或 48 小時的 buff**，所有人都能享受。沒達標的話事件就靜靜過期，buff 不會啟動。

## 怎麼觸發

每次以下行為都會 roll 一個獨立機率，命中就**自動開啟對應事件**：

| 觸發行為 | 對應觸發物資 |
| --- | --- |
| 挖到指定礦石 | 黃金 / 鑽石 / 鐵礦 |
| 釣到指定魚 | 章魚 / 鯊魚 |
| 收成指定作物 | 草莓 / 黑玫瑰 |
| 通關一次地下城 | 任意 |

<Callout type="info">
  事件**自動開啟**，玩家不需要做任何事；命中後伺服器公告頻道會發一則 Container 告知大家「事件開始」。
</Callout>

### 並發與冷卻

- **同時最多 1 個募集中事件**：避免一次開太多搶物資。如果已經有事件在募集中，新觸發的事件會被擋下。
- **同類事件 168 小時（7 天）冷卻**：避免短時間反覆觸發同一個。
- **buff 中的事件不占募集名額**：buff 期間可以再開新事件。

## 指令

```
/世界事件
```

列出目前**進行中**的所有世界事件，包含募集中與 buff 中。

- **個人面板**（ephemeral，只有你看得到）。
- 每個事件下方有「🎁 捐獻物資」按鈕，按了會跳出**物資選單** → **數量按鈕**。
- 系統會**優先扣個人背包**，不足時才扣公會倉庫（你必須在公會中）。

## 捐獻流程

<Steps>
  <Step>
    `/世界事件` 看到正在募集的事件，按「🎁 捐獻物資」。
  </Step>
  <Step>
    從 Select Menu 選一個還缺的物資。
  </Step>
  <Step>
    看到目前**個人背包**與**公會倉庫**的可捐量，按「捐 1 / 10 / 50 / 全捐」。
  </Step>
  <Step>
    系統優先扣個人背包，不夠才扣公會倉庫。
  </Step>
</Steps>

<Callout type="info">
  捐獻**不會退款**。物資一旦捐出去就消耗掉，請確認再按。
</Callout>

### 個人 vs 公會倉庫的扣帳邏輯

- **預設**：個人背包優先 → 不足扣公會倉庫。
- **沒加入公會**：只能扣個人背包。如果個人背包不夠，會擋下，不會自動跳到公會倉庫。
- **建材 / 鋼錠 等公會專屬物資**：只能從公會倉庫扣（個人背包不會有）。

## 達標與 Buff

當事件的所有需求物資都被捐滿，事件**自動切換成 buff 階段**：

1. 公告頻道發「✅ 達標！全服 buff 啟動」Container，列出 buff 內容與結束時間。
2. 全服所有玩家立刻吃到 buff——不用再做任何操作，下次挖礦 / 打工 / 釣魚 / 戰鬥就會自動套用。
3. 在 `/狀態` 可以看到「🌍 世界事件」區塊，列出目前所有生效中的 buff 與結束時間。
4. buff 結束時公告頻道會發一則「🏁 buff 結束」訊息。

## 募集逾時

如果在 24 小時內沒湊齊物資，事件會**自動結束**並廣播「⏰ 募集逾時」公告，buff 不啟動。已經捐進去的物資**不會退回**——它是合作型的全服任務，付出的物資視為消耗品。

<Callout type="warn">
  別獨自挑戰需求量大的事件（例如礦災修復的鐵礦 ×80 + 煤炭 ×100）；快速貼到公告或聊天頻道，找隊友一起補完才不會做白工。
</Callout>

## Buff 與其他來源的疊加

世界事件的 buff **獨立於**個人 buff（食物、消耗品）、公會 buff（等級 / 建築）、Twitch 訂閱、伺服器加成之外，直接**加總**進去：

- **百分比加成**：與既有 buff 累加（例 25% + 10% = 35%）。
- **倍率加成**（金幣 / XP）：與既有倍率累乘。
- **整數加成**（挖礦額外個數、體力上限）：直接相加。

完整明細可在 `/狀態` 看到所有來源。

## 常見問題

<Accordions>
  <Accordion title="事件多久會開一次？">
    沒有固定排程。要看伺服器整體活躍度——挖礦 / 釣魚 / 種田 / 地下城越熱，事件越常開。
  </Accordion>

  <Accordion title="我要怎麼提高觸發機率？">
    多去挖稀有礦（黃金、鑽石）、釣稀有魚（鯊魚、章魚）、收成黑玫瑰，這些機率比較高。普通挖礦不會觸發。
  </Accordion>

  <Accordion title="可以同時開兩個事件嗎？">
    募集中事件最多 1 個；但「buff 進行中」的事件不算數，所以可以**最多 2 個事件並行**：一個還在募集、一個已達標進入 buff 期。
  </Accordion>

  <Accordion title="捐錯物資能退嗎？">
    不能。物資視為**單向消耗**，跟公會倉庫的「存入即放棄所有權」一樣。
  </Accordion>

  <Accordion title="沒加入公會也能參加嗎？">
    可以，但只能用個人背包的物資捐。建材 / 鋼錠 這類公會專屬物資需要在公會中才能用。
  </Accordion>

  <Accordion title="buff 對所有人有效嗎？">
    是。**全伺服器**所有玩家都吃到，不分公會。
  </Accordion>

  <Accordion title="逾時的事件還會再開嗎？">
    同一個事件 7 天冷卻後可能再次觸發。冷卻不會影響其他事件。
  </Accordion>

  <Accordion title="可以強制開啟事件嗎？">
    一般玩家不行；管理員測試指令未對外開放。
  </Accordion>
</Accordions>
