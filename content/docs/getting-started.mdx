# 開始使用

## 斜線指令怎麼用

逼逼機器人的所有功能都透過 Discord 的**斜線指令（Slash Command）**操作：

1. 在任一訊息框輸入 `/`，Discord 會跳出可用指令清單。
2. 輸入指令名稱（例如 `/錢包`、`/拉霸`），選單會提示這個指令需要哪些參數。
3. 依提示填入參數後送出即可。

許多指令的回覆只有你自己看得到（臨時訊息），不會洗版。

## 參數標記說明

本文件描述指令時，採用以下格式：

```
/指令 必填參數 [選填參數]
```

- 沒有方括號的參數為**必填**。
- 方括號 `[ ]` 內的參數為**選填**，不填會使用預設值。
- 部分參數為**下拉選項**（只能從固定清單中挑選），會在參數表的「選項」欄標示。

例如 `/轉帳 對象 金額 [備註]`：`對象` 與 `金額` 必填，`備註` 可不填。

## 指令頻道規範

部分指令會限制只能在特定頻道使用，避免洗版：

| 類別 | 說明 |
| --- | --- |
| 一般指令 | 大多數功能（等級、錢包、工具等）使用的指令頻道。 |
| 賭場指令 | 拉霸、二十一點、輪盤等賭場遊戲限定頻道。 |
| 股票指令 | 買股、賣股、持股等股市相關指令頻道。 |

<Callout type="info">
  若在錯誤頻道使用指令，機器人會提示你到正確的頻道操作。實際頻道請以伺服器內公告為準。
</Callout>

## 查指令說明

不確定有哪些指令、或某個指令怎麼用時，可使用：

```
/help [指令]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `指令` | 想查看詳細說明的指令名稱 | 否 | 不填則列出所有指令分類 |

- 不帶參數：列出機器人所有指令的分類總覽。
- 帶上指令名稱：顯示該指令的用途與參數說明。
