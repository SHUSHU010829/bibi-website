# 開始使用

## 斜線指令怎麼用

逼逼機器人的所有功能都透過 Discord 的**斜線指令（Slash Command）**操作：

1. 在任一訊息框輸入 `/`，Discord 會跳出可用指令清單。
2. 輸入指令名稱（例如 `/檔案`、`/拉霸`），選單會提示這個指令需要哪些參數。
3. 依提示填入參數後送出即可。

許多指令的回覆只有你自己看得到（臨時訊息），不會洗版。

## 參數標記說明

本文件描述指令時，採用以下格式：

```
/指令 必填參數 [選填參數]
```

- 沒有方括號的參數為**必填**。
- 方括號 `[ ]` 內的參數為**選填**，不填會使用預設值。
- 部分參數為**下拉選項**（只能從固定清單中挑選），會在參數表的「選項」欄標示。

例如 `/轉帳 對象 金額 [備註]`：`對象` 與 `金額` 必填，`備註` 可不填。

部分指令會限制只能在特定頻道使用，避免洗版：

| 類別 | 頻道 | 包含指令 |
| --- | --- | --- |
| 一般指令 | 市民服務處 │ 🏢 | 等級、簽到、錢包、轉帳、定存、任務、食物、工具等大多數功能 |
| 賭場指令 | 拉斯維加斯 │ 🃏、濱海灣金沙 │ 🃏 | 拉霸、骰寶、二十一點、HI-LO、輪盤、尋寶、火箭、射龍門、賽馬、德州撲克，以及樂透 |
| 股票指令 | 逼逼金證卷 │ 🪙 | `/股市` 的所有子指令（買、賣、走勢、配息、報價、持股、紀錄） |
| 挖礦指令 | 財富牢改城 │ 💼 | 挖礦、賣出、打工、合成、裝備、地下城、決鬥、贈送、拍賣 |
| 釣魚指令 | 深淵垂釣所 | 釣魚（`/烹飪` 在這裡或神農百草園都能用） |
| 農場指令 | 神農百草園 │ 🥕 | `/農場`、`/種植`、`/收成`、`/施肥`、`/農場擴建`，以及 `/烹飪`、`/賣出`（蔬菜可在這裡賣） |
| 市集指令 | 暗巷易物所 | `/商店`，以及 `/市集` 的所有子指令（賣礦、賣魚、換礦、徵求、競標、逛攤、我的攤位） |

<Callout type="info">
  若在錯誤頻道使用指令，機器人會提示你到正確的頻道操作。管理員／開發者專用指令，以及只有自己看得到的私人指令不受頻道限制。實際頻道仍以伺服器內公告為準。
</Callout>

## 查指令說明

不確定有哪些指令、或某個指令怎麼用時，可使用：

```
/help [指令]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `指令` | 想查看詳細說明的指令名稱 | 否 | 不填則列出所有指令分類 |

- 不帶參數：列出機器人所有指令的分類總覽。
- 帶上指令名稱：顯示該指令的用途與參數說明。
