# 逼幣任務

完成每日與每週任務可獲得大量逼幣。**任務不再自動入帳**，完成後需要自己到 `/逼幣任務` 按 **領取**，或下方「**一鍵領取**」一次收完。

```
/逼幣任務
```

查看你本期被指派到的任務、進度、可領取狀態與重抽額度。介面只有自己看得到（ephemeral）。

## 個人化任務池

每位玩家**每天**從 14 個每日任務裡**隨機指派 5 個**、**每週**從 11 個每週任務裡**隨機指派 5 個**。看到的任務每個人不一樣，每天 / 每週 00:00（台北時間）會重抽一次新的清單。

<Callout type="info">
  介面右上會顯示 **下次刷新 `<相對時間>`**，提醒你目前這份指派還剩多久。
</Callout>

### 重抽 / 不做了

如果指派到不喜歡的任務，可以付費調整：

| 動作 | 效果 | 每日成本 | 每週成本 |
| --- | --- | --- | --- |
| 重抽 🔄 | 換成同 tier 的另一個隨機任務（**不保證更輕鬆**） | 100 逼幣 | 200 逼幣 |
| 不做了 ⏭️ | 直接消除該任務、本期不再出現（**不會發任何獎勵**） | 30 逼幣 | 100 逼幣 |

每位玩家**本期合計調整額度**：

| 期間 | 合計可用次數 |
| --- | --- |
| 每日 | 4 次（重抽 + 不做了任意分配） |
| 每週 | 2 次（重抽 + 不做了任意分配） |

<Callout type="warn">
  **「不做了」按下去逼幣是直接花掉、任務也直接消失，不會發任何獎勵**。它的作用是把整理畫面或讓你眼不見為淨，不是「付費完成」。想要獎勵就乖乖去做，或用「重抽」換一個。
</Callout>

### 已 ready 的任務不能重抽 / 不做了

任務只要還沒**達標**都可以重抽或丟掉。一旦狀態變成「待領取（ready）」就只剩下「**領取**」按鈕——這時候已經是你的獎勵了，按下去入帳就好。

## 任務領取

完成 ≠ 入帳。任務達標後系統會：

<Steps>
  <Step>**私訊提醒你**：「**X 任務完成！可領 Y 逼幣**」（可用 `/通知設定` 或 `/逼幣任務` 下方按鈕關閉）</Step>
  <Step>該任務在 `/逼幣任務` 顯示為「✅ 待領取」+ **領取（+Y）** 按鈕</Step>
  <Step>你按下「領取」入帳，或按下方「**一鍵領取**」一口氣領完所有 ready 的任務</Step>
</Steps>

```
/逼幣任務 → 按該任務的「💰 領取 (+Y)」
```

或

```
/逼幣任務 → 下方「🪙 一鍵領取」
```

<Callout type="info">
  **忘了領也沒關係**：每天 **23:50** 系統會自動結算當日所有未領的任務，金幣會在重置前直接入帳。但**強烈建議自己領**，因為自動結算只是最後保險，沒辦法保證每筆都成功。
</Callout>

## 限時活動任務

伺服器活動期間可能會推出額外的活動任務，會獨立顯示在 `/逼幣任務` 的「🎉 限時活動任務」區塊。

<Callout type="warn">
  **活動任務不可重抽 / 不做了**，且所有玩家都看得到（非個人化指派）。
</Callout>

## 不會被指派但也不會印錢的隱藏邏輯

- **未被指派的任務不會累積進度**：即使你達成了文字活躍的條件，只要它不在你今天的 5 個裡，就完全不會記錄。
- **重抽出來的新任務從 0 進度開始**：不會繼承被換掉那一個的進度。
- **「不做了」之後的任務不會再被重抽選到**：本期內它就是徹底消失。

<Callout type="info">
  任務獎勵為固定金額，**不受**經驗 / 逼幣倍率影響。
</Callout>
