# 逼幣任務

完成每日與每週任務可獲得大量逼幣。**任務一完成，獎勵就會自動入帳**，不需要手動領取。

```
/逼幣任務
```

查看目前的每日／每週任務進度。

```
/領錢
```

補領未入帳的任務獎勵。這是**備用指令**——正常情況下獎勵會在任務完成當下自動發放，只有在極少數自動入帳失敗時，才需要用它來補領。

## 每日任務

每天重置（台北時間午夜）。

| 任務 | 條件 | 獎勵 |
| --- | --- | --- |
| 早安打卡 | 07:00–10:00 在指定頻道發言 | 150 逼幣 |
| 文字活躍 | 當日傳送 ≥ 10 則訊息 | 200 逼幣 |
| 語音初段 | 當日語音待滿 30 分鐘 | 150 逼幣 |
| 語音進階 | 當日語音待滿 60 分鐘（累加，與初段合計 250） | 100 逼幣 |
| 賭桌新手 | 完成任意賭博遊戲一局（不論輸贏） | 300 逼幣 |
| 股市初探 | 完成任意一筆股票交易（買或賣） | 250 逼幣 |

## 每週任務

每週重置。

| 任務 | 條件 | 獎勵 |
| --- | --- | --- |
| 週週出席 | 本週簽到 ≥ 5 天 | 1200 逼幣 |
| 活躍市民 | 本週發送訊息 ≥ 50 則 | 1500 逼幣 |
| 人氣王 | 本週收到 ≥ 20 個表情符號反應 | 2000 逼幣 |

<Callout type="info">
  任務獎勵為固定金額，**不受**經驗 / 逼幣倍率影響，且完成後會**自動入帳**。`/領錢` 只是自動入帳失敗時的補領退路。
</Callout>
