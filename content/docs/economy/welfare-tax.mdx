# 福利、稅務與邀請

## 乞討（救濟金）

破產時可領取救濟金，幫你東山再起。

```
/乞討
```

符合資格會直接發放，否則顯示目前狀態。

| 資格條件 | 數值 |
| --- | --- |
| 餘額需低於 | 100 逼幣 |
| 帳號需加入伺服器滿 | 30 天 |

連續乞討的金額依連續天數分級：

| 連續天數 | 救濟金 |
| --- | --- |
| 第 1 天 | 500 逼幣 |
| 第 2–3 天 | 600 逼幣 |
| 第 4–7 天 | 700 逼幣 |
| 第 8 天以上 | 800 逼幣 |

## 財產稅

為了控制通膨，逼幣餘額較高者每週會被課徵**累進財產稅**。

- **課徵時間**：每週一台北時間 04:00。
- **累進稅率**（依餘額分級）：

| 餘額門檻 | 稅率 |
| --- | --- |
| 50,000 以上 | 2% |
| 100,000 以上 | 5% |
| 300,000 以上 | 10% |
| 1,000,000 以上 | 20% |
| 5,000,000 以上 | 35% |
| 20,000,000 以上 | 50% |
| 100,000,000 以上 | 70% |

被課稅時會收到私訊通知。查看自己的歷史紀錄：

```
/我的稅務紀錄 [period]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `period` | 查詢期間 | 否 |

## 邀請獎勵

邀請新朋友加入伺服器可獲得獎勵。

```
/邀請
```

查看你的邀請統計與獎勵狀況。

| 規則 | 數值 |
| --- | --- |
| 邀請人基礎獎勵 | 3000 逼幣起（每累積一定邀請數遞增，上限 10000） |
| 被邀請人歡迎獎勵 | 500 逼幣 |
| 每人每日有效邀請上限 | 3 位 |
| 被邀請人帳號最低年齡 | 7 天 |
| 追回機制 | 被邀請人在 14 天內離開，獎勵會被追回 |

<Callout type="warn">
  若被邀請者於 14 天內退出伺服器，先前發放的邀請獎勵會被追回，避免刷邀請。
</Callout>
