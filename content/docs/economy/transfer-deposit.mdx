# 銀行系統

除了轉帳以外，所有銀行功能都收在 **`/銀行`** 一個指令底下，用**分頁 + 按鈕 + 彈窗**操作：打開後用下方選單切換頁面，各頁的動作（買進、存入、借款…）用按鈕點開視窗輸入數字即可。

| 指令 | 功能 |
| --- | --- |
| `/銀行 總覽` | 錢包、信用分、定存、黃金、貸款一次看 |
| `/銀行 信用` | 信用評等與各項銀行權益 |
| `/銀行 定存` | 定期存款（開戶 / 查詢 / 提款） |
| `/銀行 黃金` | 黃金存摺（買進 / 賣出 / 精煉 / 定存 / 領回） |
| `/銀行 貸款` | 貸款（借款 / 還款） |
| `/轉帳` | 轉帳給其他玩家（獨立指令，需對方收下才入帳） |
| `/待收` | 查看並處理待收：收下 / 拒收別人送的，或取消自己送出的轉帳、贈送 |

<Callout type="info">
  `/銀行` 各頁只有自己看得到（ephemeral）。**`/轉帳` 與 `/贈送` 都改為「待收」制**：送出後金額 / 物品先預扣託管，逼逼會**私訊（DM）通知收款人**，對方按「收下」才真正入帳，用 `/待收` 追蹤或取消（見「📮 待收」分頁）。
</Callout>

  <Tab value="💸 轉帳">
    把逼幣轉給其他人。可用 `/轉帳` 指令，或從 `/銀行 總覽 → 轉帳給玩家`（選收款人後輸入金額）。**轉出後不會立刻入帳，需對方按「收下」才生效**，詳見「📮 待收」分頁。

    ```
    /轉帳 對象 金額 [備註]
    ```

    | 規則 | 數值 |
    | --- | --- |
    | 單筆最低 | 100 逼幣 |
    | 單筆最高 / 每日上限 / 每日次數 | **依信用分等級**（見「信用分」分頁） |
    | 手續費（金額 ≤ 1000） | 1% |
    | 手續費（金額 > 1000） | 3% |

    <Callout type="info">
      **改為「每日次數」而非冷卻**：不再有 30 分鐘冷卻，改成每天可轉帳 N 次，次數、單筆上限、每日額度都隨信用分提升。次數與額度每日 00:00 重置。
    </Callout>

    <Callout type="warn">
      收款人加入伺服器需滿 **7 天**才能收款（防小帳洗幣）。異常的**雙向對敲、圈狀轉帳（A→B→C→A）與市集不合理高價成交**都會被系統標記審查並**扣信用分**，請正常使用、勿刻意搬幣。
    </Callout>
  </Tab>

  <Tab value="📮 待收">
    **`/轉帳` 送出的逼幣、`/贈送` 送出的礦石都不會立刻入帳**，而是先「預扣託管」在系統，等收款人按下**收下**才真正進對方的帳戶／背包。`/待收` 一頁看完你**要收的**與**送出中的**待收。

    ```
    /待收
    ```

    ### 流程

    1. 送出方：`/轉帳` 或 `/贈送` 成功後，金額（含手續費）／物品**立即從你身上預扣**。
    2. 逼逼**私訊（DM）通知收款人**，附「收下 / 拒收」按鈕；若對方關閉私訊，改在原頻道 tag 通知。
    3. 收款人按**收下** → 入帳 / 入袋（禮物背包放不下的部分自動折成逼幣入帳）；按**拒收** → 全額退回送出方。
    4. **24 小時**內未回覆 → 系統**自動退回**送出方（逼幣連手續費全退，且**不計入當日轉帳額度**）。逾期前每 **6 小時**私訊提醒一次。

    ### `/待收` 看得到什麼

    | 區塊 | 內容 | 可做的動作 |
    | --- | --- | --- |
    | 📥 我要收的 | 別人送給你、還沒處理的 | **收下** / **拒收** |
    | 📤 我送出的 | 你送出、對方還沒收的 | **取消並取回** |

    | 規則 | 數值 |
    | --- | --- |
    | 待收保留時間 | 24 小時（逾時自動退回送出方） |
    | 提醒間隔 | 每 6 小時私訊提醒收款人一次 |
    | 同時送出上限 | 轉帳、贈送各 **10** 筆待收 |

    <Callout type="info">
      `/待收` 只有自己看得到（ephemeral）。送出後想反悔，只要對方還沒收下，用 `/待收 → 取消並取回`（或原通知訊息上的「寄件方取消」按鈕）就能拿回預扣的逼幣 / 物品。
    </Callout>
  </Tab>

  <Tab value="📊 信用分">
    信用分反映你的銀行信譽，**同時決定轉帳上限、每日次數、定存加開數量與貸款額度**。

    ### 分數怎麼算

    綜合信用分由兩部分相加，最後夾在 **0 ~ 1000** 之間：

    ```
    綜合信用分 = 累積分 + 年資加成（上下限 0 ~ 1000）
    ```

    - **累積分**：起始 **100 分**，隨你的行為即時增減（見下方加分／扣分表）。
    - **年資加成**：在伺服器**每待一天 +0.4 分，最多 +120**（約 300 天封頂）。這部分是即時換算的，不會寫死。

    <Callout type="info">
      **每日加分上限 20 分**：所有「加分行為」每天合計最多 +20，避免狂刷小額交易灌分；**扣分沒有上限**。分數是即時換算的——一還清貸款、一被判洗幣，等級立刻跟著變。
    </Callout>

    範例：新進成員第一天 = 100 + 0 = **100 分（🔴 D）**；待滿約 125 天什麼都沒做 = 100 + 50 = **150 分（🟠 C）**；老玩家 + 常互動則會往 A／S 爬。

    ### 等級與權益

    | 等級 | 門檻 | 轉帳單筆 | 每日額度 | 每日次數 | 定存加開 | 貸款額度 |
    | --- | --- | --- | --- | --- | --- | --- |
    | 🔴 信用 D | 0 | 50,000 | 100,000 | 3 | — | 不開放 |
    | 🟠 信用 C | 150 | 100,000 | 300,000 | 6 | +2 | 50,000 |
    | 🟡 信用 B | 350 | 250,000 | 800,000 | 10 | +4 | 200,000 |
    | 🟢 信用 A | 600 | 500,000 | 2,000,000 | 15 | +6 | 500,000 |
    | 💎 信用 S | 850 | 1,000,000 | 5,000,000 | 25 | +10 | 1,500,000 |

    **加分**（每日加分上限 20）：

    | 行為 | 分數 |
    | --- | --- |
    | 完成一次轉帳 | +2 |
    | 定存到期領回 | +5 |
    | 黃金買進 / 賣出 | +1 |
    | 貸款準時（或提前）還清 | +3 ~ +15（依本金級距，需持有滿 2 天） |

    **扣分**：

    | 行為 | 分數 |
    | --- | --- |
    | 被判定洗幣（每日最多一次） | −80 |
    | 定存提前解約 | −3 |
    | 貸款每逾期一天 | −8 |
    | 貸款違約 | −150 |
  </Tab>

  <Tab value="📜 定存">
    把逼幣鎖進定期存款，到期領回本金加利息。存款期間的錢**不會被偷、也不計入財富稅**。

    | 天期 | 到期利率 |
    | --- | --- |
    | 7 天 | 2% |
    | 14 天 | 5% |
    | 30 天 | 12% |
    | 90 天 | 40% |

    | 規則 | 數值 |
    | --- | --- |
    | 單筆最低 / 最高 | 100 / 500,000 逼幣 |
    | 同時持有存單上限 | 10 張（另可由信用分與 Twitch 訂閱加開） |
    | 提前解約罰金 | 扣本金 10% |

    到期的存單會在頁面上出現「領回」按鈕，一鍵即可領回；也可用 `提款` 按鈕輸入存單 ID 領回或提前解約。
  </Tab>

  <Tab value="🪙 黃金存摺">
    純金是高單價貴金屬，價格獨立於便宜的黃金礦，**每日浮動**、全服一致。

    | 規則 | 數值 |
    | --- | --- |
    | 基準價 | 3,000 逼幣 / 克 |
    | 每日浮動 | ±25% |
    | 銀行買賣價差 | 各 4%（買進較貴、賣出較便宜，價差即手續） |

    **取得純金的兩種方式**：

    1. **買進**：用逼幣按當日買進價買入。
    2. **精煉**：把挖礦得到的**黃金礦 + 煤炭（燃料）**煉成純金 —— 每 **5 黃金礦 + 5 煤炭 → 1 克純金**。黃金頁會顯示你背包原料「可精煉約 N 克」。

    **黃金定存**：把金庫的純金鎖倉，到期領回更多克數。

    | 天期 | 到期加成 |
    | --- | --- |
    | 7 天 | +1% |
    | 30 天 | +5% |
    | 90 天 | +18% |

    黃金定存最低 1 克、同時上限 5 筆，提前領回扣 15%。

    <Callout type="info">
      低買高賣賺價差，也能當作對抗財富稅與通膨的保值工具。
    </Callout>
  </Tab>

  <Tab value="💳 貸款">
    憑信用分向銀行借款，到期一次還本息。

    | 規則 | 數值 |
    | --- | --- |
    | 借款額度 | 依信用分等級（信用 C 以上開放） |
    | 單筆最低 | 1,000 逼幣 |
    | 同時貸款上限 | 1 筆 |
    | 利息 | 10% |
    | 借款天期 | 7 天 |

    **逾期與違約**：

    - 逾期後**每天**：應還金額 +本金 3%、信用分 −8。
    - 逾期期間**黃金定存領回會被凍結**，直到還清為止。
    - 逾期滿 **14 天** → **違約**：信用分再 −150，且在還清前無法再借新貸款。
    - 準時（或提前）還清可累積信用分，金額**依借款本金級距**：本金 &lt; 50,000 為 +3、50,000+ 為 +6、200,000+ 為 +10、500,000+ 為 +15。
    - 為避免「即借即還」刷分，**持有未滿 2 天就還清不給信用分**（借款本身仍可正常還清）。

    <Callout type="warn">
      借了不還不會被自動扣款，但信用分會被打到谷底、連帶拉低你所有的轉帳與銀行權益，並凍結提領。請量力而借。
    </Callout>
  </Tab>
</Tabs>
