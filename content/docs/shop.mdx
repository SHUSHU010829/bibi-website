# 商店與背包

把賺來的逼幣花在商店裡，買顏色身份組、加成藥水、錢包卡面或自訂稱號。

<Callout type="info">
  `/商店` 移至 **暗巷易物所** 頻道使用（與 `/市集` 同頻道）；`/背包` 是只有自己看得到的私人指令，哪裡都能開。詳見[指令頻道規範](/docs/getting-started#指令頻道規範)。
</Callout>

## 商店

```
/商店
```

開啟商店互動面板，所有操作都在面板上用按鈕與選單完成：

- **切換分類**：用下拉選單選擇商品分類（顏色身份組、加成藥水、卡面風格等）。
- **翻頁**：用上一頁／下一頁按鈕瀏覽該分類的商品。
- **一鍵購買**：直接點商品旁的購買按鈕即可結帳，購買結果只有你看得到。

買完顏色身份組、卡面或自訂稱號後，面板會出現按鈕讓你**立即裝備／套用／設定稱號文字**；之後也能隨時在 `/背包` 裡調整。

<Callout type="info">
  商店面板是**僅自己可見**的私人訊息，翻頁、切換分類、購買與餘額提示都只有你看得到，不會洗版也不會洩漏。
</Callout>

## 背包

```
/背包
```

查看你購買的道具、生效中的加成（buff）與到期時間。**裝備道具（例如切換卡面、套用顏色身份組）與設定自訂稱號，都在 `/背包` 介面用按鈕／選單操作。** 同一張卡片也整合了**挖礦背包**（礦石、鎬子耐久、挖礦道具與下次可挖時間），詳見 [挖礦與打工](/docs/mining)。背包只有自己看得到。
