# 玩家交易

把礦石、魚、蔬菜跟其他玩家換金幣或互換的兩個系統都集中在這裡——你在 [挖礦](/docs/mining)、[釣魚](/docs/fishing)、[農場](/docs/farming) 累積的成果，都靠這裡流通。

<Cards>
  <Card title="市集與贈送" href="/docs/trading/marketplace" description="一口價賣礦／賣魚、徵求收購、競標，以及直接 /贈送" />
  <Card title="交易所" href="/docs/trading/exchange" description="礦石／作物／魚的純物物交換告示板" />
</Cards>

兩者的差別整理如下：

| | `/市集` | `/交易所` |
| --- | --- | --- |
| 換什麼 | 物品 ↔ 金幣 | 物品 ↔ 物品 |
| 可上架品項 | 礦石、魚 | 礦石、作物、魚（共 14 種） |
| 形式 | 一口價、徵求、競標 | 純物物交換 |
| 手續費 | 5%（買方付） | 1%（接受方付） |
| 掛單時效 | 24 小時 | 7 天 |
| 同時上限 | 8 件 | 3 筆 |

想換的東西如果有**金幣**這一邊（賣礦換錢、付錢收魚）就掛市集；純物換物（例如黃金換黑玫瑰種子、章魚換月光露水）就用交易所。

<Callout type="info">
  直接賣給系統不算「玩家交易」，請用 `/賣出`（礦石／魚／蔬菜都通用），詳見 [挖礦與打工 — 賣出](/docs/mining#賣出)。`/贈送` 目前只支援礦石。
</Callout>
