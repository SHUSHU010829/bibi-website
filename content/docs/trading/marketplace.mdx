# 市集與贈送

把礦石、魚拿去 `/市集` 跟其他玩家交易換逼幣（一口價、徵求收購、競標都有），或直接 `/贈送` 給朋友。物物交換（礦石／作物／魚互換）請見 [交易所](/docs/trading/exchange)。

## 市集

`/市集` 是玩家之間的交易市集 🏪：可以**一口價賣礦**、**一口價賣魚**、貼出**徵求單**收購別人的礦，或掛**競標品**讓玩家出價搶標。所有掛單都會**託管**你交出的礦石、魚或逼幣，**24 小時**後到期；無人成交或自行下架時，託管的東西會**自動退回**。每人同時最多掛 **8 件**（所有類型合計）。`/市集` 在 **暗巷易物所** 頻道使用。

<Callout type="info">
  **手續費全部由買方／得標者支付**：賣方拿到的金額就是掛牌價／成交價的全額，買方則要多付 **5%** 手續費。徵求單（付金幣）建單時會直接託管 `coin + 5% 手續費`，賣方接單後一樣拿全額。純物物交換（徵求付礦石）沒有金幣手續費邏輯。
</Callout>

<Callout type="info">
  `/市集 賣礦` 是**掛牌賣給其他玩家**（收逼幣），和直接賣給系統的 `/賣出` 不一樣；想賺多一點就掛市集，想立刻清背包就用 `/賣出`。想跟人**以物易物**（礦石／作物／魚互換）請改用 [`/交易所`](/docs/trading/exchange)。
</Callout>

### 逛攤

```
/市集 逛攤
```

瀏覽市集所有掛單，**僅自己可見**，可用**交易類型**與**物品類別**（礦石／魚類）下拉篩選想看的掛單。每筆掛單右側都有對應的按鈕可直接成交——賣礦／賣魚按 **購買**、換礦按 **接受交換**、徵求按 **賣給他**、競標按 **出價**。成交前都會跳出**確認面板**，避免誤觸。

### 我的攤位

```
/市集 我的攤位
```

查看自己目前的掛單，每筆右側附 **下架** 按鈕。下架後託管的礦石／逼幣會退回你的帳戶。

### 賣礦（一口價）

```
/市集 賣礦 礦石 數量 總價
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `礦石` | 要賣的礦石（下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `總價` | 一口價**總額**，至少為系統收購價的 **80%** | 是 |

買家一鍵買走需多付 **5%** 手續費（買方付），賣方拿全額。

### 賣魚（一口價）

```
/市集 賣魚 魚 數量 總價
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `魚` | 要賣的魚種（下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `總價` | 一口價**總額** | 是 |

和賣礦一樣掛牌賣給其他玩家，買家成交時加收 **5%** 手續費（買方付），賣方拿全額；無人購買會自動退回魚袋。釣魚詳見 [釣魚](/docs/fishing)，烹飪詳見 [烹飪](/docs/cooking)。

<Callout type="info">
  以前的 `/市集 換礦` 子指令已**移除**，物物交換改用涵蓋更廣（礦石／作物／魚都可）的 [`/交易所`](/docs/trading/exchange)；既有未結束的換礦掛單仍可在 `/市集 逛攤` 接受，過期會由排程自動退還託管礦石。
</Callout>

### 徵求（收購單）

```
/市集 徵求 想要礦石 想要數量 付款方式 [金幣總額] [付的礦石] [付的數量]
```

貼出「我要收某礦」的收購單，付款方式可選**金幣**或**礦石**：

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `想要礦石` | 你想收購的礦石 | 是 |
| `想要數量` | 想收購的數量 | 是 |
| `付款方式` | `金幣` 或 `礦石` | 是 |
| `金幣總額` | 付款方式為金幣時填 | 否 |
| `付的礦石` / `付的數量` | 付款方式為礦石時填 | 否 |

掛單時會先**託管**你要付的金幣或礦石。**手續費由買方付（即徵求方）**：付款方式為金幣時，建單會直接託管 `金幣總額 + 5% 手續費`，賣方接單後拿全額；取消或過期會把含手續費的全額退還給徵求者。付款方式為礦石時純物物交換，不收手續費。

### 競標

```
/市集 競標 礦石 數量 起標價 [一口價]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `礦石` | 要競標的礦石 | 是 |
| `數量` | 數量 | 是 |
| `起標價` | 起標**總價**，至少為系統收購價的 **80%** | 是 |
| `一口價` | 選填，出價達到此價立即成交（須 ≥ 起標價） | 否 |

出價時會託管「出價額 + 5% 手續費」（**買方／得標者付**），被超越時**自動退款全額**（含手續費）；得標時賣方拿全額。每筆競標 **24 小時**後結標，無人出價會自動退回礦石。

## 贈送

```
/贈送 對象 礦石 數量
```

把背包裡的礦石免手續費送給其他玩家 🎁。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `對象` | 收禮的玩家（不能是 bot 或自己） | 是 |
| `礦石` | 要送的礦石種類（下拉選項） | 是 |
| `數量` | 要送的數量 | 是 |

每日最多贈送 **3 次**，且對方背包要有足夠空間才送得出去。
