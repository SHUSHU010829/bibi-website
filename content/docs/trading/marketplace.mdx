# 市集與贈送

`/市集` 是玩家之間的交易市集 🏪，採**訂單簿**制：掛一張**買單**或**賣單**，其他玩家可以**分批**跟你成交，直到收滿／賣完為止。也可以掛**物物交換**（以物易物）或**競標**，或直接 `/贈送` 礦石給朋友。

<Callout type="info">
  **改版說明**：舊的「一口價賣礦／賣魚／賣菜」「徵求」「換礦」子指令已移除，改用涵蓋更廣、可分批成交的訂單簿。既有未結束的舊掛單仍可在 `/市集 逛攤` 成交，過期會由排程自動退還託管。
</Callout>

<Callout type="info">
  `/市集` 是**掛牌賣給其他玩家**（收逼幣或換物）；想立刻清背包、直接賣給系統請用 `/賣出`。想跟人以物易物用 `/市集 物物交換`。
</Callout>

## 共通規則

- **可交易品項**：礦石、魚、作物三大類共用。
- **掛單時長**：大量收購／大量賣出／物物交換上架時可選 **1／3／7 天**（`時長` 參數，預設 1 天）。天數越長**上架費**越高：1 天免費、3 天 200 幣、7 天 500 幣，上架時一次付清。
- **到期續租**：掛單到期前，逼逼會**私訊你**附「續租」按鈕；付費（每天 100 幣）可一鍵延長，**6 小時**內沒回應才會結算並退回託管。
- **反洗幣保護**：涉及金幣的掛單上架時會先跳 **ephemeral 預覽**，比對近期成交行情，價格明顯偏高會**警示**；物物交換則比對兩邊物品價值。成交時系統也會偵測異常交易並回報管理員。
- **託管**：所有掛單都會先託管你交出的物品或金幣；未成交、下架或到期時**自動退回**（背包滿的礦石會進 `/信箱`）。

### 逛攤

```
/市集 逛攤
```

瀏覽市集所有掛單，**僅自己可見**，可用**交易類型**（大量收購／大量賣出／物物交換／競標／公會寄售）與**物品類別**（礦石／魚類／作物）下拉篩選。每筆掛單右側都有對應按鈕可直接成交——大量收購按 **賣給他**、大量賣出按 **購買**、物物交換按 **賣給他**、競標按 **出價**。成交前都會跳出**確認面板**避免誤觸。

### 我的攤位

```
/市集 我的攤位
```

查看自己目前的掛單，每筆右側附 **下架** 按鈕。下架後託管的物品／逼幣會退回你的帳戶。

## 大量收購（掛買單）

```
/市集 大量收購 物品 數量 單價 [標題] [時長]
```

你是**買家**，一次收購大量素材：鎖定金幣，讓多位玩家**分批**賣給你。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `物品` | 你要收購的物品（礦石／魚／作物下拉選項） | 是 |
| `數量` | 總共要收購多少個 | 是 |
| `單價` | 每個願意付多少金幣，**不得低於系統售價** | 是 |
| `標題` | 選填：自訂顯示在 `#編號` 前的標題（≤30 字） | 否 |
| `時長` | 選填：掛單天數 1／3／7（預設 1 天） | 否 |

上架時**一次鎖定** `單價 × 數量 + 5% 手續費`（發起方預付）。賣家每賣入一批就**即時撥款**（每件單價，賣家拿全額）；收滿或到期後，**未用完的金額自動退回**。每人同時最多 **10 張**收購單。

## 大量賣出（掛賣單）

```
/市集 掛賣單 物品 數量 單價 [標題] [時長]
```

你是**賣家**，一次掛售大量物品，讓多位玩家**分批**向你買。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `物品` | 你要掛售的物品（礦石／魚／作物下拉選項） | 是 |
| `數量` | 總共要賣多少個 | 是 |
| `單價` | 每個賣多少金幣，**不得低於系統售價** | 是 |
| `標題` | 選填：自訂顯示在 `#編號` 前的標題（≤30 字） | 否 |
| `時長` | 選填：掛單天數 1／3／7（預設 1 天） | 否 |

上架時**託管你的物品**（不預付手續費）。每有買家買走一批，你就收到 `單價 × 數量`，並從中扣 **5% 手續費**（賣方發起時不付、成交時才扣）；賣完或到期後，**未賣出的物品自動退回**你的袋子。每人同時最多 **10 張**賣單。

## 物物交換

```
/市集 物物交換 付出物品 付出數量 想要物品 想要數量 [標題] [時長]
```

純以物易物：你**付出 X 收 Y**，多位玩家可分批跟你換、依比例撥付你付出的物品。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `付出物品` | 你要付出（託管）的物品 | 是 |
| `付出數量` | 總共付出多少個 | 是 |
| `想要物品` | 你想換到的物品 | 是 |
| `想要數量` | 總共想換多少個 | 是 |
| `標題` | 選填：自訂顯示在 `#編號` 前的標題（≤30 字） | 否 |
| `時長` | 選填：掛單天數 1／3／7（預設 1 天） | 否 |

付出與想要不能是同一個物品。上架會先跳**預覽**：以近期成交中位數（樣本不足時退回系統基礎價）估兩邊價值，價值嚴重不對等會**警示**（可能被判定變相轉帳）。確認後託管你付出的物品；接單方交出你想要的物品、依比例領走你付出的物品，並支付 **1% 手續費**（接單方付）。換滿或到期後未換出的自動退回。每人同時最多 **8 張**交換單。

## 競標

```
/市集 競標 物品 數量 起標價 [一口價] [標題]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `物品` | 要競標的物品（礦石／魚／作物下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `起標價` | 起標**總價**，至少為系統收購價的 **80%** | 是 |
| `一口價` | 選填：出價達到此價立即成交（須 ≥ 起標價） | 否 |
| `標題` | 選填：自訂顯示在 `#編號` 前的標題（≤30 字） | 否 |

出價時託管「出價額 + 5% 手續費」（**得標者付**），被超越時**自動退款全額**；得標時賣方拿全額。每筆競標 **24 小時**後結標，無人出價會自動退回原物品。

<Callout type="info">
  `標題` 填了會顯示成 `📌 你的標題` 一行貼在編號上方，純文字 + emoji（mention / @everyone 會被過濾），方便辨識「保留給某人」「佛心給鑽」之類的脈絡。
</Callout>

## 贈送

```
/贈送 對象 礦石 數量
```

把背包裡的礦石免手續費送給其他玩家 🎁。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `對象` | 收禮的玩家（不能是 bot 或自己） | 是 |
| `礦石` | 要送的礦石種類（下拉選項） | 是 |
| `數量` | 要送的數量 | 是 |

每日最多贈送 **3 次**，且對方背包要有足夠空間才送得出去。
