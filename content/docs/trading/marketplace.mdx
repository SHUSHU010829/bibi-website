# 市集與贈送

把礦石、魚、農產品拿去 `/市集` 跟其他玩家交易換逼幣（一口價、徵求收購、競標都有），或直接 `/贈送` 給朋友。物物交換（礦石／作物／魚互換）請見 [交易所](/docs/trading/exchange)。

## 市集

`/市集` 是玩家之間的交易市集 🏪：可以**一口價賣礦**、**一口價賣魚**、**一口價賣菜**，貼出**徵求單**收購別人的礦石／魚／作物，或掛**競標品**讓玩家出價搶標（三大類都支援）。所有掛單都會**託管**你交出的礦石、魚、作物或逼幣，**24 小時**後到期；無人成交或自行下架時，託管的東西會**自動退回**。每人同時最多掛 **8 件**（所有類型合計）。

<Callout type="info">
  **手續費全部由買方／得標者支付**：賣方拿到的金額就是掛牌價／成交價的全額，買方則要多付 **5%** 手續費。徵求單（付金幣）建單時會直接託管 `coin + 5% 手續費`，賣方接單後一樣拿全額。純物物交換（徵求付礦石）沒有金幣手續費邏輯。
</Callout>

<Callout type="info">
  `/市集 賣礦` 是**掛牌賣給其他玩家**（收逼幣），和直接賣給系統的 `/賣出` 不一樣；想賺多一點就掛市集，想立刻清背包就用 `/賣出`。想跟人**以物易物**（礦石／作物／魚互換）請改用 [`/交易所`](/docs/trading/exchange)。
</Callout>

### 逛攤

```
/市集 逛攤
```

瀏覽市集所有掛單，**僅自己可見**，可用**交易類型**與**物品類別**（礦石／魚類／作物）下拉篩選想看的掛單。每筆掛單右側都有對應的按鈕可直接成交——賣礦／賣魚／賣菜按 **購買**、換礦按 **接受交換**、徵求按 **賣給他**、競標按 **出價**。成交前都會跳出**確認面板**，避免誤觸。

容器最底部還會有一顆 **🔁 逛交易所** 快捷按鈕，按下會 ephemeral 顯示 [`/交易所 列表`](/docs/trading/exchange) 的內容，方便在金幣交易與物物交換之間切換瀏覽。

### 我的攤位

```
/市集 我的攤位
```

查看自己目前的掛單，每筆右側附 **下架** 按鈕。下架後託管的礦石／逼幣會退回你的帳戶。

### 一口價掛牌

    | `總價` | 一口價**總額**，至少為系統收購價的 **80%** | 是 |
    | `標題` | 選填：自訂顯示在 `#編號` 前的標題（≤30 字，例：佛心給鑽） | 否 |

    買家一鍵買走需多付 **5%** 手續費（買方付），賣方拿全額。
  </Tab>

    | --- | --- | --- |
    | `魚` | 要賣的魚種（下拉選項） | 是 |
    | `數量` | 數量 | 是 |
    | `總價` | 一口價**總額** | 是 |

    和賣礦一樣掛牌賣給其他玩家，買家成交時加收 **5%** 手續費（買方付），賣方拿全額；無人購買會自動退回魚袋。釣魚詳見 [釣魚](/docs/fishing)，烹飪詳見 [烹飪](/docs/cooking)。
  </Tab>

    | --- | --- | --- |
    | `農產品` | 要賣的農產品（紅蘿蔔／玉米／草莓／黑玫瑰） | 是 |
    | `數量` | 數量 | 是 |
    | `總價` | 一口價**總額**，至少為當日系統收購價的 **80%** | 是 |

  </Tab>
</Tabs>

<Callout type="info">
  以前的 `/市集 換礦` 子指令已**移除**，物物交換改用涵蓋更廣（礦石／作物／魚都可）的 [`/交易所`](/docs/trading/exchange)；既有未結束的換礦掛單仍可在 `/市集 逛攤` 接受，過期會由排程自動退還託管礦石。
</Callout>

### 徵求（收購單）

```
/市集 徵求 想要物品 想要數量 付款方式 [金幣總額] [付的物品] [付的數量] [標題]
```

貼出「我要收某物」的收購單，**想要的物品與付的物品都可選礦石／魚／作物三大類任一**，付款方式則可選**金幣**或**物品**：

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `想要物品` | 你想收購的物品（礦石／魚／作物下拉選項） | 是 |
| `想要數量` | 想收購的數量 | 是 |
| `付款方式` | `金幣` 或 `物品` | 是 |
| `金幣總額` | 付款方式為金幣時填 | 否 |
| `付的物品` / `付的數量` | 付款方式為物品時填（同樣可選礦石／魚／作物） | 否 |
| `標題` | 選填：自訂顯示在 `#編號` 前的標題（≤30 字） | 否 |

掛單時會先**託管**你要付的金幣或物品。**手續費由買方付（即徵求方）**：付款方式為金幣時，建單會直接託管 `金幣總額 + 5% 手續費`，賣方接單後拿全額；取消或過期會把含手續費的全額退還給徵求者。付款方式為物品時純物物交換，不收手續費。賣方接受後，徵求方得到的物品會自動進對應袋子（礦石進背包、魚進魚袋、作物進菜籃）。

### 競標

```
/市集 競標 物品 數量 起標價 [一口價] [標題]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `物品` | 要競標的物品（礦石／魚／作物下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `起標價` | 起標**總價**，至少為系統收購價的 **80%** | 是 |
| `一口價` | 選填，出價達到此價立即成交（須 ≥ 起標價） | 否 |
| `標題` | 選填：自訂顯示在 `#編號` 前的標題（≤30 字） | 否 |

出價時會託管「出價額 + 5% 手續費」（**買方／得標者付**），被超越時**自動退款全額**（含手續費）；得標時賣方拿全額。每筆競標 **24 小時**後結標，無人出價會自動退回原物品（礦石回背包、魚回魚袋、作物回菜籃）。

<Callout type="info">
  `標題` 填了會顯示成 `📌 你的標題` 一行貼在編號上方，純文字 + emoji（mention / @everyone 會被過濾），方便在逛攤時辨識「保留給某人」、「佛心給鑽」之類的脈絡。沒填就維持原本 `#編號` 顯示。
</Callout>

## 贈送

```
/贈送 對象 礦石 數量
```

把背包裡的礦石免手續費送給其他玩家 🎁。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `對象` | 收禮的玩家（不能是 bot 或自己） | 是 |
| `礦石` | 要送的礦石種類（下拉選項） | 是 |
| `數量` | 要送的數量 | 是 |

每日最多贈送 **3 次**，且對方背包要有足夠空間才送得出去。
