# 交易所

`/交易所` 是玩家之間的**物物交換告示板** 🔁，跟 `/市集` 同樣在 **暗巷易物所** 頻道使用，但專做「**X 換 Y**」的純以物易物——可用的物品**橫跨三個系統共 14 種**：

- **礦石**：石頭、煤炭、鐵礦、黃金、鑽石
- **作物**：紅蘿蔔、玉米、草莓、黑玫瑰
- **魚類**：小雜魚、鯽魚、鯊魚、章魚、熔岩魚

上架時會**託管**你給出的物品，**7 天**後過期；過期或取消會自動退回。每人同時最多 **3 筆**活躍掛單。**接受方需付 1% 手續費**（依交易雙邊物品總價值 = 給出價值 + 想要價值；以系統收購價計），上架者不收費。

<Callout type="info">
  跟 `/市集` 的差別：[`/市集`](/docs/trading/marketplace) 是「礦石／魚換**金幣**」（一口價、徵求、競標），手續費 5% 由買方付；`/交易所` 是「**任意物品**互換」（含蔬菜），手續費 1% 由接受方付。想換的東西如果有金幣這一邊，就掛市集；純物換物（例如黃金換黑玫瑰種子、章魚換月光露水）就用交易所。
</Callout>

## 上架

```
/交易所 上架 給出物品 給出數量 想要物品 想要數量
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `給出物品` | 你要給出什麼（礦石／作物／魚下拉選項） | 是 |
| `給出數量` | 給出的數量 | 是 |
| `想要物品` | 你想換到什麼（礦石／作物／魚下拉選項） | 是 |
| `想要數量` | 想要的數量 | 是 |

給出與想要不能是同一個物品；兩邊物品都必須有可計算手續費的系統價格（背包夠才能上架，會立刻託管給出物品）。

## 列表

```
/交易所 列表 [頁碼]
```

瀏覽所有玩家的活躍掛單（每頁 5 筆），右側按鈕可直接**接受**（同 guild 內成交）。接受時會校驗你的物品是否足夠並收 1% 手續費。

## 我的

```
/交易所 我的
```

查看自己目前的掛單（**僅自己可見**），右側附下架按鈕。

## 取消

```
/交易所 取消 編號
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `編號` | 掛單編號（從 `/交易所 我的` 看） | 是 |

下架後託管的物品會退回你的袋子。
