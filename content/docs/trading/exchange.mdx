# 交易所（已整合進市集）

<Callout type="warn">
  **交易所已搬家！** 物物交換已整合進 [`/市集`](/docs/trading/marketplace)，改用可分批成交的訂單簿。舊的 `/交易所 上架` 已停用；此頁只說明**舊掛單**怎麼處理。

## 現在怎麼物物交換？

改用 `/市集`：

- **上架**：`/市集 物物交換 付出物品 付出數量 想要物品 想要數量`
- **瀏覽**：`/市集 逛攤` → 交易類型篩選「物物交換」

好處是可以**分批成交**（多位玩家分批跟你換、依比例撥付），並套用了與其他掛單一致的**反洗幣價值檢查**。詳見 [市集 — 物物交換](/docs/trading/marketplace#物物交換)。

## 舊掛單怎麼辦？

改版前掛在 `/交易所` 的掛單仍然有效，會照原本的 7 天效期運作，並由排程在到期時**自動退回**託管物品（改版前這段退款排程缺漏，現已補上）。你也可以隨時手動處理：

### 我的

```
/交易所 我的
```

查看自己在舊交易所還沒結束的掛單（**僅自己可見**），右側附下架按鈕。

### 取消

```
/交易所 取消 編號
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `編號` | 掛單編號（從 `/交易所 我的` 看） | 是 |

下架後託管的物品會退回你的袋子。
