# 伺服器功能

## 自助身份組

機器人會在指定頻道貼出**身份組選單**，你可以自己點選領取或取消身份組（例如「開台通知」或各種遊戲身份組），不需要麻煩管理員。

- 在身份組訊息上點選對應選項即可領取。
- 再次點選可取消該身份組。

## 動態語音頻道

不必預先開一堆語音頻道——加入指定的「建立語音頻道」入口後，機器人會**自動幫你開一個專屬的臨時語音頻道**：

- 你成為該頻道的擁有者，可以管理頻道（改名、設定人數上限等）。
- 當頻道內所有人都離開後，頻道會**自動刪除**，保持頻道列表整潔。

<Callout type="info">
  新建立的頻道預設名稱會提示你「記得改名喔！」，可自行重新命名。
</Callout>

## 遊戲房

想跟朋友圍一桌玩遊戲、又不想在公共頻道洗版？用 `/開房` 開一間**專屬的遊戲房討論串**，房內可以使用所有遊戲指令（管理員指令除外）。

```
/開房 [類型]
```

| 參數 | 說明 | 選項 |
| --- | --- | --- |
| 類型 | 房間類型，不填預設公開 | 公開：所有人都看得到、可加入／私人：只有被房主拉進來的人看得到 |

- 房間會以 `遊戲房#001 - 你的名字` 流水號命名，開在你下指令的頻道底下。
- **私人房**只有被拉進來的人看得到，在房內 `@提及` 即可邀請朋友。
- 每人同時只能開一間，想換房請先關閉現有的。

### 關房方式

三種方式任選：

1. **釘選訊息按鈕**：開房時機器人會在房內釘選一則歡迎訊息（含「🚪 關閉遊戲房」按鈕），隨時從討論串右上角的釘選圖示找到它，不怕被聊天訊息洗掉。
2. **`/關房` 指令**：在房內或伺服器任何地方都能用，會自動找到你開的房。
3. **自動封存**：房間 24 小時沒有動靜會自動封存關閉。

<Callout type="info">
  只有房主與管理員能關房。房間關閉後討論串會鎖定並封存，不會刪除，聊天紀錄都還在。
</Callout>

## 邀請獎勵

邀請新朋友加入伺服器可以賺逼幣。機器人會追蹤每個邀請連結帶進來的成員，並發放獎勵給邀請人與被邀請人。

詳細的獎勵金額、每日上限與追回機制，請見[福利、稅務與邀請](/docs/economy/welfare-tax)。
