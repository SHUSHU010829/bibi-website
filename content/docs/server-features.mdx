# 伺服器功能

## 自助身份組

機器人會在指定頻道貼出**身份組選單**，你可以自己點選領取或取消身份組（例如「開台通知」或各種遊戲身份組），不需要麻煩管理員。

- 在身份組訊息上點選對應選項即可領取。
- 再次點選可取消該身份組。

## 動態語音頻道

不必預先開一堆語音頻道——加入指定的「建立語音頻道」入口後，機器人會**自動幫你開一個專屬的臨時語音頻道**：

- 你成為該頻道的擁有者，可以管理頻道（改名、設定人數上限等）。
- 當頻道內所有人都離開後，頻道會**自動刪除**，保持頻道列表整潔。

<Callout type="info">
  新建立的頻道預設名稱會提示你「記得改名喔！」，可自行重新命名。
</Callout>

## 邀請獎勵

邀請新朋友加入伺服器可以賺逼幣。機器人會追蹤每個邀請連結帶進來的成員，並發放獎勵給邀請人與被邀請人。

詳細的獎勵金額、每日上限與追回機制，請見[福利、稅務與邀請](/docs/economy/welfare-tax)。
