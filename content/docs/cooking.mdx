# 烹飪

把 [釣魚](/docs/fishing) 釣到的魚和 [農場](/docs/farming) 收成的蔬菜煮成 buff 食物。每道料理對應一種加成，效果會在挖礦、打工、地下城、釣魚、收成時生效。勾選**煤炭烤製**會額外消耗背包裡的**煤炭**，換來**更強、更持久**的強化版效果，並讓食物**保鮮時間 ×1.5**。

<Callout type="warn">
  烹飪**不會立即生效**——成品會收進 🥡 **食物倉庫**，等你用 `/食物` 主動食用才會發動 buff。新鮮度會隨時間衰減，越早吃效果越完整（詳見下方「食物倉庫與新鮮度」）。
</Callout>

## 指令

```
/烹飪 [食物] [煤炭烤製]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `食物` | 要製作的料理（下拉選項）；**不填則私下列出所有食譜一覽** | 否 | — |
| `煤炭烤製` | 是否消耗煤炭升級效果（需持有足夠煤炭） | 否 | 否 |

<Callout type="info">
  **按次數**生效的食譜（打工、釣魚、挖礦、收成類）會直到對應次數用完才消失，跨冷卻也能慢慢用；**按時間**生效的多是 **地下城 ATK／DEF／HP 上限** 與 **全屬性** 類，到期就消失。**全屬性**會同時加成挖礦幸運、地下城 ATK、打工收入，效果最全面但時效最短。
</Callout>

<Callout type="info">
  **魚排便當**改成**按打工次數**計算（而非固定時間），煮一份後接下來幾次打工都會吃到加成。**海鮮拼盤**會把烹飪與釣魚串起來：吃完接下來幾竿的**成功率**與**稀有魚機率**都會提升。
</Callout>

<Callout type="info">
  **次數型 buff 可以累計**：同類型的**按次數**料理（打工、釣魚、挖礦、收成類）再吃一份會**把次數加上去**，不會重置——例如挖礦幸運已剩 3 次，再吃一份章魚飯就變 **6 次**，不用等舊的用完。不同食譜或不同新鮮度各自的加成值都會保留，**逐次消耗時先用最舊那份的效果值**，用完才換下一份。
</Callout>

<Callout type="warn">
  **時間型 buff 仍會覆蓋重置**：**按時間**生效的料理（地下城 ATK／DEF／HP 上限、全屬性…）再吃一份同類型會**覆蓋並重置**倒數（按下食用前會跳確認框）。**烹飪**本身只是把食物放進倉庫，不會影響當前 buff。不同類型（例如打工 +20% 與地下城 ATK +20）可同時持有，**唯一例外**是「**全屬性 +X%**」類料理（戰前晚餐、玉米巧達濃湯、熔岩鍋、豐收盛宴、黑玫瑰精華…）——它與任一單屬性 buff（地下城 ATK／DEF／HP、打工收入、挖礦幸運、釣魚加成、收成加成）**互斥**，吃下去會蓋掉所有單屬性 buff（會跳確認框），反過來吃單屬性也會蓋掉全屬性。食物加成獨立於挖礦幸運上限之外計算。
</Callout>

## 食物倉庫與新鮮度

煮好的食物**不會自動吃掉**，而是收進 🥡 **食物倉庫**，等你下指令食用才會發動 buff。打開倉庫：

```
/食物
```

倉庫會依「平均新鮮度由低到高」排序，**最該優先吃的擺在最上面**。食用一律從**最舊那份**開始（避免食材浪費）：

- **時間型料理**（或某種只剩 1 份時）→ 卡片附「✨ 食用 1 份」按鈕，按下直接吃掉最舊那份。
- **次數型料理且有多份時** → 卡片改成「🍽️ 選份數食用」按鈕，按下會問你**這次要吃幾份**（下拉選 1～N 份），一次吃 N 份就**累計 N ×每份次數**。例如倉庫有 4 份章魚飯，選吃 2 份就一口氣 +6 次挖礦幸運。

要覆蓋現有 buff 時（時間型同類型、或全屬性↔單屬性互斥）會跳確認框，避免不小心蓋掉長效加成；次數型累計不會覆蓋，直接加上去。

### 衰減曲線

每份食物從**烹飪當下**開始計時，新鮮度從 100% 線性衰減到 0%：

| 階段 | 普通食物 | 🔥 煤炭烤製（×1.5） | 新鮮度 |
| --- | --- | --- | --- |
| 全新 → 開始衰減 | 0 ～ **12 小時** | 0 ～ **18 小時** | **100%** |
| 衰減期（線性） | 12 小時 ～ **7 天** | 18 小時 ～ **10.5 天** | 100% → 0% |
| 完全腐壞 | 超過 7 天 | 超過 10.5 天 | **0%（自動變堆肥）** |

倉庫卡片會用顏色標籤直觀提示新鮮程度：

| 標籤 | 區間 |
| --- | --- |
| 🟢 新鮮 | 80% ～ 100% |
| 🟡 普通 | 50% ～ 79% |
| 🟠 偏舊 | 20% ～ 49% |
| 🔴 快壞了 | 1% ～ 19% |

### 食用時怎麼算

食用瞬間結算新鮮度，套用方式視 buff 形式而定：

- **數值類**（打工 +X%、ATK +X、挖礦幸運 +X%、全屬性 +X%、釣魚成功率/稀有度 +X%、收成 +X%）→ **效果數值 × 新鮮度**。例如鯊魚生魚片基礎 +18 ATK，新鮮度 60% 就只給 +11 ATK。
- **持續時間** 與 **使用次數**（次數型 buff 的「接下來 N 次」、時間型 buff 的 1.5h / 2h 等）**不衰減**，仍照食譜原值給滿。

換句話說：放越久，**效果強度**越打折，但**時長／次數**不縮水。

### 過期：自動變廚餘堆肥

超過完全腐壞時間的食物會在你下次打開 `/食物`（或執行 `/烹飪`）時被自動清理，**每一份腐壞食物換 1 份 🍂 廚餘堆肥**進背包——食材不會白白消失，只是強度沒了。倉庫頂端會以小字提示「剛清掉 X 份腐壞食物 → +X 廚餘堆肥」。

## 副產品：廚餘堆肥

**每次烹飪都會自動產出 1 份 🍂 廚餘堆肥**放進背包，可拿到 [農場](/docs/farming) 當肥料使用——煮飯本身就在為農場存材料。**🌹 黑玫瑰精華**還會額外副產 🌟 **月光露水**（農場最高階肥料）。蔬菜配方需要先在 `/農場` 收成蔬菜才能煮。

## 跨系統串接

烹飪是把 [釣魚](/docs/fishing)、[農場](/docs/farming)、[挖礦](/docs/mining)、[地下城](/docs/dungeon) 串起來的中樞：

- **魚**（釣魚）＋ **蔬菜**（農場）→ `/烹飪` → 進 🥡 食物倉庫
- 🥡 食物倉庫 → `/食物` 食用 → 觸發挖礦／打工／地下城／釣魚／收成的 buff
- **煤炭**（挖礦）＝ 煤炭烤製升級效果＋保鮮 ×1.5
- **廚餘堆肥**（烹飪副產 ＋ 過期食物轉化）→ 回流農場當肥料

詳細的資源循環圖見 [農場與種田 — 跨系統循環](/docs/farming#跨系統循環)。
