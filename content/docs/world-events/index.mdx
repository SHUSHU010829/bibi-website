# 世界事件 🌍

世界事件是**全伺服器**的共同目標。事件不是排程跑出來的，而是玩家在挖礦 / 釣魚 / 收成 / 通關地下城時**有機率觸發**——某個玩家「中獎」就開了一個事件，全服都看得到、都能一起捐物資。

達成目標後，全服啟動一個 **24 或 48 小時的 buff**，所有人都能享受。沒達標的話事件就靜靜過期，buff 不會啟動。

## 怎麼觸發

每次以下行為都會 roll 一個獨立機率，命中就**自動開啟對應事件**：

| 觸發行為 | 對應觸發物資 |
| --- | --- |
| 挖到指定礦石 | 黃金 / 鑽石 / 鐵礦 |
| 釣到指定魚 | 章魚 / 鯊魚 |
| 收成指定作物 | 草莓 / 黑玫瑰 |
| 通關一次地下城 | 任意 |

<Callout type="info">
  事件**自動開啟**，玩家不需要做任何事；命中後伺服器公告頻道會發一則 Container 告知大家「事件開始」。
</Callout>

## 並發與冷卻

- **同時最多 1 個募集中事件**：避免一次開太多搶物資。如果已經有事件在募集中，新觸發的事件會被擋下。
- **同類事件 168 小時（7 天）冷卻**：避免短時間反覆觸發同一個。
- **buff 中的事件不占募集名額**：buff 期間可以再開新事件。

## 指令

```
/世界事件
```

列出目前**進行中**的所有世界事件，包含募集中與 buff 中。

- **個人面板**（ephemeral，只有你看得到）。
- 每個事件下方有「🎁 捐獻物資」按鈕，按了會跳出**物資選單** → **數量按鈕**。
- 系統會**優先扣個人背包**，不足時才扣公會倉庫（你必須在公會中）。
