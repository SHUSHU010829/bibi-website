# 事件清單

目前內建 11 個事件。觸發機率、需求量、獎勵 buff 都寫在 `world_events.json`，未來會持續擴充。

- **觸發**：挖到黃金 5% 機率
- **徵集**：鐵礦 ×80、煤炭 ×100
- **獎勵**：全服挖礦冷卻 -15%（24h）

> 礦坑遭遇坍塌，需要全服礦工合力修復。
