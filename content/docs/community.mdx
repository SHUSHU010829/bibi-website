# 客服與投票

## 票務客服

當你需要協助時，可在票務面板點擊按鈕開立一張**專屬的票務頻道**，只有你與管理團隊看得到，方便私下溝通。

- 在票務面板按下「創建票務」按鈕，機器人會幫你開一個專屬頻道。
- 在頻道內描述你的問題，管理團隊會盡快回覆。
- 處理完成後，點擊「關閉票務」按鈕即可關閉（頻道會在數秒後刪除）。

<Callout type="info">
  每人同時只能有一張開啟中的票務。請先關閉現有票務再開新的。
</Callout>

## 建議申訴系統

想對伺服器提出建議、申訴檢舉，或申請開新頻道／身份組？在建議申訴面板的下拉選單選擇類型，就會開立一個對應的討論串。

| 類型 | 用途 |
| --- | --- |
| 📺 頻道增開 | 申請開設新頻道 |
| 🏷️ 身份組增開 | 申請新增遊戲身份組 |
| ⚖️ 申訴檢舉 | 申訴或檢舉事件（私密溝通） |
| ✨ 功能許願 | 許願希望機器人增加的功能 |
| 💬 其他建議 | 其他任何建議 |

討論完成後，同樣點「關閉票務」按鈕通知管理員處理。

## 投票系統

伺服器的重大決策（新增／封存遊戲頻道、舉辦活動、修改規則等）會透過投票表決。投票發起後，大家用訊息上的按鈕表態即可，時間到自動結算（預設開放 48 小時）。

不同投票範本有不同的選項與通過門檻：
