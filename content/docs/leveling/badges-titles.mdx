# 徽章與稱號

隨著你在伺服器活躍，會解鎖各式各樣的**徽章**，並可把它們設為等級卡上的**稱號**與**展示徽章**。

## 徽章圖鑑

```
/level badges
```

查看所有徽章的解鎖進度。徽章依照你的活躍度、里程碑、特殊成就等條件自動解鎖。

## 設定稱號

```
/level title 設定 徽章
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `徽章` | 從已解鎖的徽章，或目前等級 tier 中選一個當作稱號 | 是 |

設定後，稱號會顯示在你的等級卡上。

## 展示徽章

### 設定

```
/level displaybadges 設定
```

依序挑選最多 5 個已解鎖的徽章，顯示在等級卡下方。

### 重置

```
/level displaybadges 重置
```

回到預設（依解鎖順序顯示前 5 個）。

## 等級卡主題

```
/level cardtheme 主題
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `主題` | 選擇等級卡的顏色主題 | 是 |

<Callout type="info">
  等級卡（`/level profile`）會綜合顯示你的等級、經驗進度、稱號與展示徽章；卡片主題可隨時切換。
</Callout>

## 等級排行榜

```
/level rank
```

查看伺服器的等級排行榜，看看誰最活躍。另可使用更通用的 [`/leaderboard`](/docs/economy) 查看各類排行（經驗、等級、逼幣、訊息、語音、反應等）。
