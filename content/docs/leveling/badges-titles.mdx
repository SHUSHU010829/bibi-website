# 徽章與稱號

隨著你在伺服器活躍，會解鎖各式各樣的**徽章**，並可把它們設為等級卡上的**稱號**與**展示徽章**。

## 徽章與成就圖鑑

```
/檔案 分頁:成就
```

把等級徽章（活躍度、訊息、語音、連勝、社交、特殊）與遊戲稱號（挖礦、賭場、股市、樂透、拍賣）的解鎖進度集中在同一張圖鑑，含進度條與 ⚡ 接近解鎖提示。

## 設定稱號

```
/稱號 設定 徽章:<選項>
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `徽章` | 從已解鎖的徽章、目前等級 tier，或已解鎖的遊戲稱號中挑一個當稱號 | 是 |

設定後，稱號會顯示在你的等級卡上。

<Callout type="info">
  這個稱號槽與**遊戲稱號**共用——挖礦、賭場、股市、樂透、拍賣解鎖的稱號也能掛在這裡。詳見 [遊戲稱號與成就](/docs/leveling/game-titles)。
</Callout>

## 展示徽章

### 設定

```
/稱號 展示徽章 徽章1:<id> 徽章2:<id> …
```

依序挑選最多 5 個已解鎖的徽章，顯示在等級卡下方。

### 重置

```
/稱號 重置展示
```

回到預設（依解鎖順序顯示前 5 個）。

## 等級卡顏色

等級卡顏色已改由[商店](/docs/shop)購買，5 種顏色（粉紅、海藍、金箔、薄荷、墨黑）各 **500 金幣**永久解鎖。

1. 用 `/商店` 在「等級卡顏色」分類選喜歡的顏色購買
2. 開 `/背包`，到「🌈 等級卡顏色」選單把它裝備上
3. 下次 `/檔案` 就會看到新顏色

<Callout type="info">
  舊版用過 `/level cardtheme` 設過顏色的玩家不用再買一次，bot 啟動時會自動把對應顏色補進背包並標記為已裝備。
</Callout>

## 等級排行榜

```
/排行榜 類別:等級
```

查看伺服器的等級排行榜，看看誰最活躍。`/排行榜` 也包含訊息、語音、頻道熱度、挖礦、賭場贏家／輸家等其他類別，用下拉選單切換。
