# 每日簽到

每天簽到可以領取 XP 與逼幣，連續簽到還有額外加成。

## `/每日簽到`

```
/每日簽到 [押倍]
```

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `押倍` | 把今天的金幣翻倍，但隔天若沒簽到，連勝（streak）直接歸零，且不能用補簽卡 | 否 | False |

簽到後會產生一張**簽到卡片**，顯示今天的日期與連勝天數。

## 連勝獎勵

- **基礎獎勵**：100 XP、60 逼幣。
- **連勝加成**：每連續一天，額外增加 XP / 逼幣（XP 連勝加成最多累計 30 天，逼幣最多累計 10 天）。
- **連勝倍率**：
  - 連續 7 天以上：XP ×1.5、逼幣 ×1.5
  - 連續 30 天以上：XP ×2.0、逼幣 ×2.0

每日簽到的重置時間為**台北時間午夜（00:00）**。

<Callout type="warn">
  使用 `押倍` 雖然金幣翻倍，但只要隔天忘記簽到，連勝就會直接歸零，且這天無法用補簽卡補救。請斟酌使用。
</Callout>

## 補簽卡

連勝萬一斷掉，可用**補簽卡**保護。

```
/補簽卡
```

查看你目前的補簽卡庫存與規則。

- **取得方式**：每連續簽到 30 天解鎖 1 張。
- **庫存上限**：最多持有 3 張。
- 當你某天忘記簽到時，補簽卡可保護連勝不歸零（使用 `押倍` 的那天除外）。
