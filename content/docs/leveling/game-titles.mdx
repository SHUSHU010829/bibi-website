# 遊戲稱號與成就

在挖礦、賭場、股市、樂透、拍賣中達成特定條件，就能解鎖各遊戲的**稱號**。這些稱號與等級卡的稱號**共用同一個展示槽**——挑一個掛上去，就會顯示在你的錢包卡與升等公告上。

<Callout type="info">
  成就條件全部由既有資料自動推導，達標時會在你進行該活動（領到逼幣）時自動解鎖，不需要手動領取。
</Callout>

## 查看與設定

```
/稱號 清單
/稱號 設定 稱號
/成就
```

| 指令 | 說明 |
| --- | --- |
| `/稱號 清單` | 依遊戲分類列出所有稱號與你的解鎖進度（⭐ 展示中 ・ ✅ 已解鎖 ・ 🔒 未解鎖） |
| `/稱號 設定 稱號` | 把錢包卡上展示的稱號切換成任一**已解鎖**的稱號；選「清除」則改回顯示等級稱號 |
| `/成就` | 用進度條檢視各遊戲稱號的達成度 |

<Callout type="info">
  展示稱號也可以在 [`/level title`](/docs/leveling/badges-titles) 設定，和等級徽章稱號二選一。
</Callout>

## 稱號一覽

### ⛏️ 挖礦

| 稱號 | 解鎖條件 |
| --- | --- |
| 🪵 煤炭採集者 | 累積挖礦 50 次，且持有 ≥ 500 逼幣 |
| 🔨 鐵鍛師 | 累積合成 10 件，且歷史鐵礦 ≥ 100 顆 |
| 💎 寶石獵人 | 歷史黃金 ≥ 20 顆，且歷史鑽石 ≥ 1 顆 |
| 👑 礦坑之王 | 當週挖礦數量第一（每週更替） |
| ✨ 傳說礦工 | 累積挖礦 1000 次、歷史鑽石 ≥ 5 顆，且週冠 ≥ 3 次 |

### 🎰 賭場

| 稱號 | 解鎖條件 |
| --- | --- |
| 🎰 賭場常客 | 累積下注 100 局 |
| 💰 一夜致富 | 單局派彩 ≥ 10,000 逼幣 |
| 🃏 Jackpot 得主 | 開出一次拉霸 Jackpot |

### 📈 股市

| 稱號 | 解鎖條件 |
| --- | --- |
| 📈 股海散戶 | 完成 10 筆股票交易 |
| 💹 股神 | 累積賣股 ＋ 股利收入 ≥ 50,000 逼幣 |

### 🎟️ 樂透

| 稱號 | 解鎖條件 |
| --- | --- |
| 🎟️ 樂透愛好者 | 累積購買 50 張樂透 |
| 🏆 頭獎得主 | 中過一次樂透頭獎 |

### 🏷️ 拍賣

| 稱號 | 解鎖條件 |
| --- | --- |
| 🏷️ 拍賣商人 | 拍賣成交 20 件 |

## 礦坑之王（每週爭奪）

**礦坑之王 👑** 是唯一每週更替的稱號。每週一 **00:01** 結算上週挖礦數量榜，把冠軍頒發礦坑之王並在頻道公告，同時卸下上一任。用 [`/挖礦排行`](/docs/mining) 查看本週戰況。累積稱王 3 次是解鎖**傳說礦工**的條件之一。
