# 遊戲稱號與成就

在挖礦、賭場、股市、樂透、拍賣、BOSS 共鬥中達成特定條件，就能解鎖各遊戲的**稱號**。這些稱號與等級卡的稱號**共用同一個展示槽**——挑一個掛上去，就會顯示在你的錢包卡與升等公告上。

<Callout type="info">
  成就條件全部由既有資料自動推導，達標時會在你進行該活動（領到逼幣）時自動解鎖，不需要手動領取。
</Callout>

## 查看與設定

```
/檔案
/稱號 設定 徽章:<選項>
```

| 指令 | 說明 |
| --- | --- |
| `/檔案` | 到「🏆 成就」分頁，同時顯示等級徽章與遊戲稱號的完整圖鑑與解鎖進度（✅ 已解鎖 ・ 🔒 未解鎖） |
| `/稱號 設定 徽章` | 把等級卡上展示的稱號切換成任一**已解鎖**的徽章、遊戲稱號或目前等級 tier |

<Callout type="info">
  等級徽章與遊戲稱號共用同一個展示槽，最後設定的會生效。詳見 [徽章與稱號](/docs/leveling/badges-titles)。
</Callout>

## 礦坑之王（每週爭奪）

**礦坑之王 👑** 是唯一每週更替的稱號。每週一 **00:01** 結算上週挖礦數量榜，把冠軍頒發礦坑之王並在頻道公告，同時卸下上一任。用 [`/排行榜 類別:挖礦`](/docs/mining) 查看本週戰況。累積稱王 3 次是解鎖**傳說礦工**的條件之一。
