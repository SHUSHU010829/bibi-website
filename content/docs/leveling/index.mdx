# 經驗與等級

在伺服器互動就能累積**經驗值（XP）**並升等。等級越高，等級卡越好看，還能解鎖徽章與稱號。

<Callout type="info">
  簽到、等級卡等指令屬於一般指令，在 **市民服務處 │ 🏢** 頻道使用，詳見[指令頻道規範](/docs/getting-started#指令頻道規範)。
</Callout>

## XP 來源

| 來源 | 獲得量 | 限制／條件 |
| --- | --- | --- |
| 聊天發言 | 每則 15–25 XP | 冷卻 30 秒；訊息至少 4 個字 |
| 語音 | 每分鐘 10 XP | 頻道內需有 **2 人以上**；靜音／拒聽與閒置（AFK）頻道不算 |
| 收到表情反應 | 每個反應 2 XP | 每人每日上限 50 XP |
| 每日簽到 | 基礎 100 XP | 詳見 [每日簽到](/docs/leveling/checkin) |

## 經驗倍率

以下倍率會讓你獲得的 XP 變多。**多個倍率同時存在時，採「取最高」而非相加**。

| 倍率來源 | 倍率 | 適用範圍 |
| --- | --- | --- |
| 連續簽到 7 天以上 | ×1.5 | 簽到 XP |
| 連續簽到 30 天以上 | ×2.0 | 簽到 XP |
| Twitch 訂閱 Tier 1 / 2 / 3 | ×1.5 / ×2.0 / ×3.0 | 發言、語音、簽到 |
| 伺服器加成（Boost） | ×2.0 | 發言、語音、簽到 |
| XP 加成藥水 | ×1.5 或 ×2.0 | 主動互動取得的 XP（詳見[商店](/docs/shop)） |

<Callout type="info">
  「取最高」表示：例如你同時有伺服器加成（×2.0）與 Twitch Tier 1（×1.5），系統會套用 ×2.0，而不是 ×3.0。
</Callout>

## 升等公告

達到特定里程碑等級（5、10、20、30、50、75、100）時，機器人會發送升等卡片公告，並發放額外的逼幣獎勵（詳見[逼幣經濟](/docs/economy)）。

## 個人檔案、稱號與排行榜

| 指令 | 說明 |
| --- | --- |
| `/檔案` | 個人檔案聚合入口：等級卡、礦工檔案、錢包、賭場紀錄、持股、成就 6 個分頁。零參數、永遠公開、永遠看自己；用按鈕在分頁間切換，只有原呼叫者能切換 |
| `/稱號 設定 徽章` | 設定等級卡稱號（徽章 / 等級 tier / 已解鎖的遊戲稱號） |
| `/稱號 展示徽章 / 重置展示` | 自選等級卡下方要展示的 5 個徽章 |
| `/排行榜 [類別] [時間]` | 統一排行榜：等級、訊息、語音、頻道熱度、挖礦、賭場贏家／輸家（用下拉切類別、按鈕切時間） |

<Callout type="info">
  `/檔案` 可在「市民服務處 │ 🏢」、「財富牢改城 │ 💼」、「逼逼金證卷 │ 🪙」三個頻道使用。背包仍維持獨立的 `/背包` 指令。
</Callout>

<Callout type="info">
  等級卡顏色已改由[商店](/docs/shop)購買並到 `/背包` 裝備。原本用過 `/level cardtheme` 設過顏色的玩家，bot 啟動時會自動補一筆背包紀錄並標記為已裝備，不會掉色。
</Callout>

徽章、稱號的詳細玩法請見 [徽章與稱號](/docs/leveling/badges-titles)。
