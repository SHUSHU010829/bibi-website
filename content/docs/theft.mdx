# 盜賊系統

潛入其他玩家的**錢包**偷金幣，得手有賞、失手被通緝。通緝犯錢包透明、任人追捕，可花保釋金自首洗白，或低調潛伏熬過時效。整個迴路是：**偷竊 → 失風追逃 → 通緝 → 追捕 / 自首 / 潛伏**。

<Callout type="info">
  只偷得到**錢包**（`/金幣` 顯示的餘額）的錢；存進**銀行**的資產（定存、黃金金庫）都絕對安全。怕被偷就用 `/銀行` 把主要資產鎖進去。
</Callout>

## 🥷 偷竊

```
/偷竊 對象
```

得手是**潛行**的——只有你自己看得到結果，對方不會收到通知（但可花錢 `/報案` 查兇手）。

| 項目 | 規則 |
| --- | --- |
| 基礎成功率 | 55%（夾在 15% ~ 85%） |
| 惡名加成 | 每點惡名 +1% 成功率，最多 +15% |
| 🕶️ 夜行衣 | 本次成功率 +15% |
| 🔒 對方保險箱 | 你的成功率 −20% |

**偷取金額**採累進級距，只抽對方錢包各區段的一部分（越有錢的區段抽越少）：

| 錢包區間 | 抽成 |
| --- | --- |
| 0 – 5,000 | 12% |
| 5,000 – 30,000 | 8% |
| 30,000 以上 | 4% |

- **單次上限**＝`min(3,000 + 惡名 × 300, 12,000)`：惡名越高一次偷越多，封頂 12,000。
- **黑市抽成 20%**：偷到的金額先被黑市抽走 20%（蒸發），實際入袋是剩下的 80%。

### 防護閘（防霸凌）

| 保護 | 規則 |
| --- | --- |
| 🛡️ 新手保護 | 目標入服未滿 **7 天**不可偷 |
| 🪹 低餘額免疫 | 目標錢包 **< 500** 不可偷 |
| 💸 本錢門檻 | 你自己錢包 **< 300** 不能偷（要付得起失風的賞金） |
| 🥷 每日上限 | 每人每天最多偷 **5 次** |
| ⏳ 同目標冷卻 | 對同一人 **8 小時** |
| 🚧 目標被偷上限 | 同一人每天最多被偷 **3 次** |
| 🐕 看門狗 | **70% 機率**幫目標擋下（擋下才消耗）；30% 會被小偷用食物引開 / 打盹 / 追野貓而漏掉，這隻狗留著下次繼續守 |

| 道具 | 價格 | 效果 |

## 🏃 失風追逃

偷竊失手**不會當場被全鎮通緝**，而是先進入一段只有你自己看得到的「逃亡關卡」。逐關選路線甩開追捕，**累積前進 4 格就清白脫身**：

| 路線 | 前進 | 起始被逮率 |
| --- | --- | --- |
| 🌆 混入人群 | 1 格 | 12%（最穩） |
| 🌫️ 鑽暗巷 | 2 格 | 20% |
| 🏙️ 飛簷走壁 | 3 格 | 30%（最快也最險） |

- **被逮率隨你已逃的距離升高**：每前進 1 格 +10%，最高 85%——越接近終點越燙手。
- ⏰ **放著不逃滿 10 分鐘 = 視同沒逃掉**，直接送上通緝榜。

<Callout type="info">
  🕊️ **躲到 4 格 → 清白脫身**：神不知鬼不覺，**沒上通緝榜、賞金一毛沒扣**。

  🚔 **半路被逮 → 才正式通緝**：此刻才凍結你錢包裡的賞金、公開通緝令，任人追捕。
</Callout>

## 🚨 通緝與賞金

失風被逮就上**通緝榜**，錢包裡的賞金被凍結託管：

```
賞金 = 1,000 + 惡名 × 100 + 歷史累積贓款 × 8%
       （夾在 300 ~ min(20,000, 你當前錢包)）
```

賞金當下即從你錢包**實扣凍結**，所以上限不會超過你付得起的額度。惡名越高、偷越多 → 賞金越肥 → 越多人想抓，形成「爬得越高摔得越重」的平衡。

- **通緝時效 24 小時**：期間沒被抓到 → 自動洗白、**託管賞金全額退回**、惡名 −1。潛伏熬過時效是「不損失、只是提心吊膽一天」的合法出路。

## 🎯 追捕

任何玩家都能追捕通緝犯，瓜分賞金：

```
/追捕 對象
```

| 項目 | 規則 |
| --- | --- |
| 成功率 | `50% + (獵人攻擊力 − 通緝犯攻擊力) × 0.3%`，夾在 20% ~ 80% |
| 抓到 | 託管賞金全給獵人，通緝犯再賠**錢包 10% 贖罪金**（一半給獵人、一半進治安基金） |
| 追捕失敗 | 該獵人本輪**不能再追同人**；通緝犯進躲藏冷卻（**4 分 − 惡名 × 30 秒，下限 1 分**——惡名越高躲越短、越快能繼續行動） |
| 自動脫罪 | 通緝犯累積逃脫 **5 次** → 賞金退回、洗白 |
| 每日上限 | 每人每天最多追捕 **5 次** |

## 🙇 自首

通緝犯主動認罪，立即解除通緝、惡名 −1。系統會**先全額退回被凍結的賞金，再沒收保釋金**——保釋金由三段疊加，不再是「花小錢秒洗白」。

**自首時機有兩道門檻**（防止上榜就秒洗白、逼你先面對獵人）：

| 門檻 | 規則 |
| --- | --- |
| 🕵️ 至少被追捕一次 | **沒被任何獵人追捕過就不能自首**——先撐過獵人這關才有洗白的資格 |
| ⏱️ 上榜鎖定期 | 剛上榜一段時間內不能自首：**基礎 10 分、每點惡名 −30 秒、下限 4 分**（惡名越高越早能自首） |

過了門檻才付得起保釋金洗白，保釋金代價如下：

| 代價 | 規則 |
| --- | --- |
| 📉 基礎沒收 | 託管賞金的 **60%** |
| 📈 累犯遞增 | **24 小時內**每多自首一次 **+20%**（60% → 80% → 100%），最重**全額沒收** |
| 💰 贓款抽成 | 額外吐回**歷史累積贓款的 5%**，偷越多自首越貴 |
| 🚔 假釋觀察期 | 自首後 **6 小時內不能偷竊** |

<Callout type="warn">
  保釋金總額會夾在你付得起的範圍內，不會把你扣成負債；但沒收額超過凍結賞金時，錢包會再倒扣差額。自首依然能拿回被凍結的賞金——它從此是「認錯的代價」，不是「花小錢消災」。
</Callout>

## 🕵️ 報案

被偷了不會收到通知，但你可以**花錢委託偵探**調閱近 **24 小時**內偷過你的紀錄：

```
/報案 [偵探]
```

偵探分四級，越貴查到率越高、越少出包，也越常有線人爆料：

| 偵探 | 委託費 | 查到率 | 出包率 | 線人爆料 |
| --- | --- | --- | --- | --- |
| 🕵️ 菜鳥偵探 | 300 | 50% | 28% | 4% |
| 🕵️‍♂️ 資深偵探（預設） | 800 | 72% | 10% | 11% |
| 🎩 神探 | 2,000 | 90% | 3% | 24% |
| 👑 王牌偵探 | 5,000 | 100% | 5% 捲款跑路＋5% 變身阿拉蕾 | 45% |

- **查到率**是「每筆竊案查出兇手的機率」；線索越舊（接近 24 小時邊緣）越難查。
- 不指定偵探時預設委託**資深偵探**。近期沒人偷你時不收委託費。

### 🎲 調查途中的意外

委託後有機率遇上各種事件——**越菜的偵探越容易出包**；就算是王牌，也仍有極小機率翻車：

**好事件**（幫到你）：

| 事件 | 效果 |
| --- | --- |
| 🐦 線人爆料 | **保證破案**：直接鎖定近期偷你最多的主嫌 |
| 💎 黑吃黑反殺 | 破案之外，偵探**順手從主嫌錢包搶一筆賠給你**（主嫌真的會掉錢） |
| 🎣 釣魚執法 | 偵探沒揪出兇手，但幫你設下陷阱——**下一個偷你的人當場失風被通緝** |

**壞事件**（會二選一，結果公開廣播到治安頻道）：

| 事件 | 效果 | 選項 A | 選項 B |
| --- | --- | --- | --- |
| 🕵️‍♂️💨 捲款跑路 | 偵探拿了委託費人間蒸發 | 🏃 追上去逮他 | 🤷 自認倒楣 |
| 🤝💰 被兇手收買 | 偵探被買通回報「查無此人」 | 🔨 拆穿雙面偵探 | 😮‍💨 算了認栽 |
| 🕵️‍♂️🔪 壞人偵探 | 委託費之外還黑吃黑（最高 **2,500**） | 🗡️ 當場翻臉討錢 | 🤷 自認倒楣 |
| 🤖⚡ 變身阿拉蕾 | **王牌專屬** 5%：變身阿拉蕾捅你、捲走 **5,000** | 🍡 拿零食哄她 | 😱 拔腿就跑 |
| 🕳️ 監守自盜 | 查到最後兇手竟是偵探自己 | 📢 報警抓他（50% 討回+獎金） | 🤝 收編當內應（**下次報案免費**） |
| 🎰 賭徒偵探 | 偵探慫恿你拿委託費賭一把 | 🎲 跟他賭（50% 委託費雙倍） | 🚶 不賭，退你錢 |
| 🐛 抓錯人 | 偵探亂指一通、咬定某路人是兇手 | 🤥 將錯就錯辦他 | 🙏 道歉放人 |
| 🍺 偵探喝掛 | 醉倒沒查成，酒醒退你**一半**委託費 | —（無選擇） | — |

<Callout type="info">
  「反擊型」壞事件（捲款跑路 / 被收買 / 壞人偵探 / 阿拉蕾）的選項 A 都是 **50% 賭一把**：成功討回全部損失（委託費 + 被捲走的錢），失敗被反咬一口、再從你錢包多搶一筆。其餘事件各有專屬玩法。
</Callout>

查出兇手後，對每名兇手可**三選一**（同一案件同一兇手只能處理一次）：

| 選項 | 規則 |
| --- | --- |
| 🗡️ 強制決鬥 | 對方不用同意，攻擊力高者勝，**贏了討回被偷金額的一半**（每個兇手只能打一次） |
| 🎯 懸賞通緝 | 付「**被偷金額的 200%**」（最低 **2,000**）把他掛上全鎮通緝榜，讓所有人追捕 |
| 🕊️ 放過他 | 一筆勾銷，之後報案不再列出他 |

<Callout type="info">
  🎯 **懸賞通緝**：被你懸賞的兇手會立刻上通緝榜、**不能再偷竊，也不能花保釋金自首**，只能被追捕或逃亡。

  - 他**被抓** → 你拿回他的**贖罪金分成**，幫你討回一點損失。
  - 他**逃掉 / 撐到時效** → 你出的懸賞金**全數進治安基金**（不退你、也不退他）。
</Callout>

## 💰 治安基金

被逮贖罪金的一半 + 自首保釋金 + **沒人領到的懸賞金**（被懸賞者逃掉 / 到期）會累積成**治安基金**，**每週日 21:00** 全額發給該週賞金收入最高的獵人。犯罪者繳的錢與流掉的懸賞最終回饋給最勤勞的賞金獵人。

## 🥷 惡名（notoriety）

小偷的聲望值，影響成功率、單次偷取上限與頭上賞金：

| 事件 | 惡名變化 |
| --- | --- |
| 得手 | +1 |
| 失手 | +2 |
| 被捕 | −2 |
| 自首 | −1 |
| 時效洗白 | −1 |
| 每天沒偷竊 | −1（自然衰退） |

惡名越高：成功率略升、一次能偷更多——但**賞金也更肥、更多人想抓**，形成自我節制的天花板。
