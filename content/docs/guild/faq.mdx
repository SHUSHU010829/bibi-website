# 常見問題

<Accordions>
  <Accordion title="公會 buff 跟其他來源的 buff 怎麼疊？">
    公會 buff 會跟個人 buff（食物、裝備、宴會等）一起**累加**，在 `/加成` 可以看到所有來源的明細與剩餘時間。
  </Accordion>

  <Accordion title="退會 / 被踢後個人累計捐款會保留嗎？">
    不會。離開公會就清空，重新加入需從 0 開始累積。
  </Accordion>

  <Accordion title="公會升等需要做什麼？">
    不用手動操作。只要總捐款達到門檻就會自動升等並通知，新 buff 立刻對所有成員生效。
  </Accordion>

  <Accordion title="副會長可以指派其他副會長嗎？">
    不行，**只有會長**能在「⚙️ 公會管理」按鈕區指派或撤銷副會長。
  </Accordion>

  <Accordion title="轉讓會長給副會長後，副會長身分會怎樣？">
    直接升任會長；原會長變回一般成員，原本的副會長頭銜消失。
  </Accordion>

  <Accordion title="找不到 `/公會 解散`、`/公會 踢人` 這些指令？">
    已合併到 `/公會 資訊` 底部的「⚙️ 公會管理」按鈕區。低頻、會長專屬的操作改成按鈕，避免 slash command 列表爆量。
  </Accordion>

  <Accordion title="存進倉庫的東西可以拿回來嗎？">
    不能。存入即放棄所有權，這是設計上的硬規則，避免有人「存進去刷貢獻 → 自己領回」白嫖。
  </Accordion>

  <Accordion title="倉庫滿了怎麼辦？">
    升公會等級會自動擴張容量（Lv.2 → Lv.10 三倍），或加蓋倉庫擴建建築，或請會員多用掉一些。
  </Accordion>

  <Accordion title="為什麼我剛存進去就不能領？">
    新存物品有 **1 小時保護期**，全員都不能領，避免協同掃倉。即使保護期過了，「同存入者 24 小時內」也不能領回自己存的那項。
  </Accordion>

  <Accordion title="手續費怎麼算？為什麼有時候 20 有時候 80？">
    公式 `max(20, 市價 × 10%)`。低單價物資（石頭、小雜魚）一律 20 起跳，高單價物資（鑽石、熔岩魚）會按 10% 算，越貴越貴。
  </Accordion>

  <Accordion title="手續費跑去哪了？">
    直接進**公會金庫**，沒有消失。會員之間付的手續費還是留在公會內部循環。
  </Accordion>

  <Accordion title="熔爐 / 建築升級費誰出？">
    一律從**公會金庫**扣，不是會長自己的錢。會長 / 副會長是「代表公會執行」，所以拿公會的錢理所當然——金庫流水也會有對應紀錄，可在 `/公會 倉庫 → 📜 紀錄` 查到。
  </Accordion>

  <Accordion title="熔爐製造建材 / 鋼錠是個人的還是公會的？">
    公會的。原料從公會倉庫扣，產出物入公會倉庫，全程不會碰到個人背包。
  </Accordion>

  <Accordion title="建材 / 鋼錠可以拿出來賣嗎？">
    不行。它們是公會專屬中間物資，只用於建築升級。要賣的話請改賣原料（石頭、煤炭、鐵礦）。
  </Accordion>

  <Accordion title="倉庫擴建是擴個人背包還是公會倉庫？">
    **公會倉庫**。倉庫擴建建築會增加公會倉庫**每個物品**的容量上限，但個人背包不受影響。
  </Accordion>

  <Accordion title="宴會的「副本傷害」會套用到世界 BOSS 嗎？">
    **不會**。宴會的「副本傷害」buff **只對地下城戰鬥（`/地下城`）生效**，世界 BOSS（`/魔王 攻擊`）不吃。想加強 BOSS 戰，請走公會等級 Lv.4 / Lv.5（攻擊力 +5% / +10%）與訓練場 Lv.3（BOSS 傷害 +5%）。
  </Accordion>

  <Accordion title="宴會冷卻是個人還是公會？召集後如果有新成員加入會吃到 buff 嗎？">
    冷卻是**公會層級共用**——同一公會 24 小時內只能開一場。宴會 buff 跟著公會狀態走，召集後加入的新成員，在宴會結束前進來的話**也會吃到**剩餘時間的 buff。
  </Accordion>
</Accordions>
