# 成員與管理

## 副會長

公會可由會長指派**副會長**，協助處理日常營運。副會長的權限**幾乎等同會長**，但**不能解散公會、不能轉讓會長、也不能再指派／撤銷其他副會長**。

入口：`/公會 資訊` →「⚙️ 公會管理」→「🛡️ 指派副會長」/「🪶 撤銷副會長」，按下後從成員下拉選單挑人。

| 操作 | 會長 | 副會長 | 一般成員 |
| --- | --- | --- | --- |
| 邀請玩家加入 | ✅ | ✅ | ❌ |
| 處理申請（核准 / 拒絕） | ✅ | ✅ | ❌ |
| 踢出成員 | ✅ | ✅（不能踢會長 / 其他副會長） | ❌ |
| 編輯公會簡介 / 會規 | ✅ | ✅ | ❌ |
| 領取週任務獎勵 | ✅ | ✅ | ❌ |
| 指派 / 撤銷副會長 | ✅ | ❌ | ❌ |
| 轉讓會長 | ✅ | ❌ | ❌ |
| 解散公會 | ✅ | ❌ | ❌ |
| 捐款 / 看資訊 / 查任務 | ✅ | ✅ | ✅ |

- 副會長人數沒有硬性上限，會長可視公會規模指派。
- 在 `/公會 資訊` 的成員列表中，副會長會標 🛡️，會長標 👑，排序為 **會長 → 副會長 → 一般成員**。
- 會長身分被「⚙️ 公會管理 → 👑 轉讓會長」轉出去後，原會長自動降為一般成員；若新會長原本是副會長，會自動升任。

<Callout type="warn">
  副會長被踢出公會、退會或被會長從「⚙️ 公會管理 → 🪶 撤銷副會長」撤銷後，身分立刻失效。指派時請挑值得信任的長期成員。
</Callout>

## 加入公會：邀請 vs 申請

加入公會有兩條路：**會長主動邀請**，或**玩家主動申請**。兩者都要對方同意才會生效。

<Tabs items={['🪪 會長邀請', '📨 玩家申請']}>
  <Tab value="🪪 會長邀請">
    ```
    /公會 邀請 [使用者]
    ```

    會長執行後，被邀請的玩家會收到一張帶有「接受 / 拒絕」按鈕的卡片。邀請有效期 **168 小時（7 天）**，超時自動失效。

    - 不能邀請機器人或自己。
    - 若對方已在其他公會、或公會人數已滿，會擋下並提示原因。
    - 同一人同時只能有一張 pending 邀請。
  </Tab>
  <Tab value="📨 玩家申請">
    ```
    /公會 申請 [名稱] [理由?]
    ```

    玩家主動申請加入某公會。可附上**最多 100 字**的理由給會長參考。

    - 申請有效期 **168 小時（7 天）**。
    - 若被該公會拒絕，需等 **24 小時冷卻**才能再次申請同一個公會。
    - 會長 / 副會長在 `/公會 資訊` 看到「📨 待審申請 N」紅色按鈕（pending > 0 才會顯示），按下後進入列表逐一核准 / 拒絕。
  </Tab>
</Tabs>

## 管理：踢人、轉讓、解散

這些**低頻會長操作**集中在 `/公會 資訊` 底部的「⚙️ 公會管理」按鈕區，會長 / 副會長按權限看到不同按鈕，避免 slash command 列表過長。

| 按鈕 | 說明 | 對象 |
| --- | --- | --- |
| 👢 踢人 | 從成員下拉選單挑一位踢出公會。不能踢自己；副會長另外不能踢會長 / 其他副會長。 | 會長 / 副會長 |
| 🛡️ 指派副會長 | 從成員下拉選單挑一位指派為副會長。 | 僅會長 |
| 🪶 撤銷副會長 | 從成員下拉選單挑一位副會長撤銷其身分（仍是公會成員）。 | 僅會長 |
| 👑 轉讓會長 | 選擇新會長後**多一層確認**才執行。轉讓完原會長變回一般成員，不可逆。 | 僅會長 |
| 💥 解散公會 | 紅色按鈕。按下後跳出確認 Container：列出分配人數、每人分得多少、被沒收的鎖定金。確認後金庫**平分**給有資格的成員，公會永久消失。 | 僅會長 |
| `/公會 退會` | 一般成員、副會長可直接退會；會長想退會請先用「👑 轉讓會長」或「💥 解散公會」。 | 一般成員、副會長 |

<Callout type="warn">
  解散後公會名稱可被新公會使用，原成員的個人捐款累積也會清空。會長還會進入**重建冷卻**，不能立刻再建新公會。確認過才按「確定解散」。
</Callout>

<Callout type="info">
  按鈕互動全程**只有按按鈕的人看得到**（ephemeral），不會洗版頻道。對選到非公會成員、或對方已是 / 不是副會長等狀況，會在同一則訊息回覆錯誤說明。
</Callout>
