# 公會擴張：熔爐 / 精煉站 / 公會建築 🏗️

公會升到中後期，會長 / 副會長能用**金庫資金 + 公會倉庫的礦石**，把公會本部從「單純收禮金的場」改造成可主動經營的基地。整套系統可分三塊：

- **熔爐 / 精煉站**：把礦石加工成**建材 / 鋼錠**（公會專屬中間物資）。
- **公會建築**：用建材 / 鋼錠 + 金幣蓋出**礦坑 / 訓練場 / 倉庫擴建**，給全公會 buff。
- **建材 / 鋼錠**：只存於公會倉庫，**不能存入 / 領取 / 寄售**，只能透過熔爐製造、建築消耗。

| 解鎖門檻 | 系統 | 提供 |
| --- | --- | --- |
| 公會 Lv.2 | 公會建築 | 礦坑 / 訓練場 / 倉庫擴建（用既有礦石蓋 Lv.1） |
| 公會 Lv.4 | 熔爐 | 解鎖建材配方;再升一級解鎖鋼錠配方 |
| 公會 Lv.5 | 精煉站 | 3 階折扣，最高 -30% 素材消耗 |

## 熔爐：把礦石煉成建材 / 鋼錠

`/公會 熔爐` 打開後顯示目前熔爐 / 精煉站等級，下方用 Select Menu 選配方再按「製造 ×N」。

| 產出 | 基礎配方（每 1 個） | 解鎖條件 |
| --- | --- | --- |
| 🧱 **建材** | 石頭 ×30 + 煤炭 ×1 | 熔爐 Lv.1 |
| ⛓️ **鋼錠** | 鐵礦 ×20 + 煤炭 ×10 | 熔爐 Lv.2 |

熔爐建造費用（由會長 / 副會長從公會金庫支出）：

| 階段 | 費用 | 解鎖 |
| --- | --- | --- |
| 建造熔爐（Lv.0 → Lv.1） | 10,000 逼幣 | 建材配方 |
| 升級熔爐（Lv.1 → Lv.2） | 30,000 逼幣 | 鋼錠配方 |

<Callout type="info">
  原料**只從公會倉庫扣**，不會動到個人背包。所以蓋熔爐前，先用 `/公會 存入` 把成員的石頭 / 煤炭 / 鐵礦搬進倉庫。產出的建材 / 鋼錠也直接寫進公會倉庫。
</Callout>

## 精煉站：降低熔爐素材消耗

精煉站本身不能直接製造，但**會自動套用折扣**到熔爐配方上。

| 精煉站等級 | 升級費用 | 素材消耗折扣 | 建材實扣 | 鋼錠實扣 |
| --- | --- | --- | --- | --- |
| Lv.1 | 50,000 | -10% | 27 石 + 1 煤 | 18 鐵 + 9 煤 |
| Lv.2 | 100,000 | -20% | 24 石 + 1 煤 | 16 鐵 + 8 煤 |
| Lv.3 | 150,000 | -30% | 21 石 + 1 煤 | 14 鐵 + 7 煤 |

<Callout type="info">
  折扣同比應用到所有素材；不會免費（向上取整避免 1 個煤被打折成 0）。
</Callout>

## 公會建築：礦坑 / 訓練場 / 倉庫擴建

`/公會 建築` 顯示三棟建築的目前等級、下一級需求與升級後效果。升級操作**只有會長 / 副會長**能執行;普通成員可以查看進度但不能升級。

### ⛏️ 礦坑（給挖礦加成）

| 等級 | 升級需求 | 累計 buff |
| --- | --- | --- |
| Lv.1 | 建材 ×1 | 挖礦冷卻 -5% |
| Lv.2 | 建材 ×5 | 挖礦冷卻 -10% |
| Lv.3 | 建材 ×15、鋼錠 ×5 | 挖礦冷卻 -10%、挖礦幸運 +5% |

### 🏋️ 訓練場（給地下城 / BOSS 加成）

| 等級 | 升級需求 | 累計 buff |
| --- | --- | --- |
| Lv.1 | 鋼錠 ×1 | 地下城傷害 +5% |
| Lv.2 | 鋼錠 ×8、建材 ×5 | 地下城傷害 +10%、暴擊率 +3% |
| Lv.3 | 鋼錠 ×15、建材 ×10 | 地下城傷害 +10%、暴擊率 +5%、BOSS 傷害 +5% |

### 📦 倉庫擴建（提升公會倉庫容量）

每升一級，**公會倉庫的每個物品容量都會再 +N**（累積加成，不只當前等級）。

| 等級 | 升級需求 | 容量加成（每物品） |
| --- | --- | --- |
| Lv.1 | 建材 ×1、5,000 逼幣 | +50 |
| Lv.2 | 建材 ×2、10,000 逼幣 | +150（50+100） |
| Lv.3 | 建材 ×4、15,000 逼幣 | +350（50+100+200） |
| Lv.4 | 建材 ×8、鋼錠 ×2、30,000 逼幣 | +750（+400） |
| Lv.5 | 建材 ×15、鋼錠 ×5、50,000 逼幣 | +1,550（+800） |

<Callout type="info">
  倉庫擴建的容量加成**與公會等級的容量乘數同時生效**。例如公會 Lv.5（×1.9）+ 倉庫擴建 Lv.5（+1,550）下，鐵礦容量會是 `floor(300 × 1.9) + 1550 = 2,120`。
</Callout>

## 建材 / 鋼錠的特殊規則

- **存在地點**：只在公會倉庫，不會出現在個人背包 / 魚袋。
- **取得方式**：只能透過熔爐製造。
- **消耗方式**：只用於公會建築升級。
- **不可寄售 / 存入 / 領取**：在 `/公會 寄售`、`/公會 存入`、`/公會 領取` 都不會出現在物品選單。

## 升級操作流程

<Steps>
  <Step>
    **準備建材 / 鋼錠**

    先 `/公會 存入` 把成員手中的石頭 / 煤炭 / 鐵礦搬進公會倉庫 → `/公會 熔爐` 製造建材或鋼錠。
  </Step>
  <Step>
    **累積金庫餘額**

    若升級費用要扣金幣，請成員 `/公會 捐款` 把逼幣推進金庫。
  </Step>
  <Step>
    **執行升級**

    會長 / 副會長進 `/公會 熔爐` 或 `/公會 建築`，符合條件的升級按鈕才會亮起；按下即扣資源完成升級。
  </Step>
</Steps>

<Callout type="warn">
  熔爐 / 精煉站 / 建築的升級費用一律從**公會金庫**扣，不是會長自己的錢。會長 / 副會長代表公會執行操作即可。
</Callout>
