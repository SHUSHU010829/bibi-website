# 公會宴會 🍱

當公會把**農膳坊升到 Lv.5**，會長 / 副會長就能召集一場**公會宴會**——花費公會倉庫的食材，**全公會成員**在限定時間內共享一組強力 buff。每場宴會只能擇一菜單，且公會層級共用 **24 小時冷卻**。

<Callout type="info">
  入口在 `/公會 建築` 畫面底部「🍽️ 公會宴會」區塊，**只有會長 / 副會長**看得到「召集宴會」按鈕；按下後彈出選單挑菜色。
</Callout>

## 召集流程

<Steps>
  <Step>
    **農膳坊先升到 Lv.5**

    宴會解鎖門檻＝農膳坊建築 Lv.5。`/公會 建築` 升級即可。
  </Step>
  <Step>
    **倉庫備齊食材**

    宴會材料**從公會倉庫扣**，不會動到個人背包。先用 `/公會 存入` 把成員手上的食材搬進來。
  </Step>
  <Step>
    **會長 / 副會長召集**

    `/公會 建築` →「🍽️ 召集宴會」→ Select 選菜單 → 確認 Container 顯示扣料明細 → 按「開席」即扣料、buff 立刻對全公會生效。
  </Step>
  <Step>
    **席間享受 buff**

    宴會持續期間，公會所有成員的對應 buff 自動疊加（不限線上）。可在 `/加成` 看到目前生效的宴會 buff 與剩餘時間。
  </Step>
</Steps>

## 重點規則

| 規則 | 說明 |
| --- | --- |
| 解鎖門檻 | 公會農膳坊 Lv.5 |
| 召集權限 | 會長 / 副會長 |
| 冷卻時間 | 24 小時（公會共用，不是個人） |
| 同時數量 | 一場宴會結束前不能再開新場 |
| 材料來源 | 公會倉庫（建材 / 鋼錠例外，仍需精煉產出） |
| Buff 適用 | 公會所有成員、不限線上、不限時當下是否在公會 |

<Callout type="info">
  宴會 buff 與個人 buff（食物、裝備、公會等級、公會建築）**全部累加**，在 `/加成` 可以看到所有來源的明細與剩餘時間。
</Callout>
