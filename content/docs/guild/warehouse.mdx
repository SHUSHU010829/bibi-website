# 公會倉庫 📦

公會倉庫是會員間**互助分享原料**的池子：手上太多的礦／作物／魚可以存進來給公會用，缺料的成員付一點手續費就能領出對應數量。**Lv.2 解鎖**。

<Callout type="info">
  倉庫**不是**個人物品的暫存箱。**存入即放棄所有權，不可收回**——等同捐贈。整套機制設計是讓資源在公會內部流轉，不是給個人換倉。
</Callout>

## 怎麼用

```
/公會 倉庫                       ← 查看存貨 / 你今日已領紀錄
/公會 存入 [物品] [數量]         ← 把背包 / 魚袋的資源放進倉庫
/公會 領取 [物品] [數量]         ← 從倉庫領出（也可在倉庫畫面直接按按鈕）
```

倉庫畫面把礦石 / 作物 / 魚類分塊顯示，每項旁邊都有「領 1」「領 5」「領上限」快捷按鈕，數字就是實際扣的手續費——直接點即可，不需重打指令。

## 支援物品與單次上限

倉庫第一版只收**原料型成品**（已進入背包 / 魚袋），田裡未收割、料理成品、種子、合成材料、Buff 道具**不可存**。

| 分類 | 物品 | 單次取出上限 |
| --- | --- | --- |
| ⛏️ 礦石 | 石頭、煤炭 | 15 |
| | 鐵礦 | 10 |
| | 黃金 | 5 |
| | 鑽石 | 1 |
| 🌾 作物 | 紅蘿蔔 | 15 |
| | 玉米 | 10 |
| | 草莓 | 5 |
| | 黑玫瑰 | 1 |
| 🎣 魚類 | 小雜魚、鯽魚 | 15 |
| | 鯊魚 | 10 |
| | 章魚 | 5 |
| | 熔岩魚 | 1 |

<Callout type="info">
  **建材 🧱 / 鋼錠 ⛓️** 是公會專屬加工品，只能由[熔爐](./expansion#熔爐把礦石煉成建材--鋼錠)製造、由[公會建築](./expansion#公會建築礦坑--訓練場--倉庫擴建)消耗，**不能** `/公會 存入` / `/公會 領取` / `/公會 寄售`。
</Callout>

倉庫**容量上限**隨公會等級放大（Lv.2 = 1.0 倍 → Lv.10 = 3.0 倍），並可由[倉庫擴建建築](./expansion#-倉庫擴建提升公會倉庫容量)再追加：

| 等級 | 容量乘數 | 例：鐵礦容量 |
| --- | --- | --- |
| Lv.2 | 1.0 | 300 |
| Lv.3 | 1.3 | 390 |
| Lv.4 | 1.6 | 480 |
| Lv.5 | 1.9 | 570 |
| Lv.10 | 3.0 | 900 |

## 領取的規則（避免資源被吸乾）

| 規則 | 預設 | 會長可調 |
| --- | --- | --- |
| 公會等級 ≥ 2 | 系統解鎖門檻 | ❌ |
| 已加入公會 ≥ N 小時 | 24 小時 | ✅ 6h ~ 168h |
| 個人累積貢獻 ≥ V | 200 | ✅ 0 ~ 5000 |
| 每日領取次數 | 2 次 | ✅ 1 ~ 3 |
| 同一物品一天只能取一次 | 固定 | ❌ |
| 同存入者 24 小時內不能領回自己存的物品 | 固定 | ❌ |
| 新存物品保護期（全員都不能領） | 1 小時 | ❌ |

> 「累積貢獻」= 金庫捐款 + 倉庫存入市價 × 10%（換算貢獻值）+ BOSS 戰貢獻。

## 手續費（按市價比例，不是固定值）

```
fee = max(20, ⌈ 單價 × 數量 × 費率 ⌉)
```

預設費率 **10%**，會長可調 5% ~ 20%。**手續費直接進公會金庫，不消失**——付出去的錢還是留在公會內部循環。

| 範例 | 計算 | 手續費 |
| --- | --- | --- |
| 取 15 顆石頭（單價 8） | max(20, ⌈ 8×15×0.1 ⌉) | 20 |
| 取 5 顆鐵礦（單價 60） | max(20, ⌈ 60×5×0.1 ⌉) | 30 |
| 取 1 顆鑽石（單價 800） | max(20, ⌈ 800×1×0.1 ⌉) | 80 |
| 取 1 條熔岩魚（單價 600） | max(20, ⌈ 600×1×0.1 ⌉) | 60 |

<Callout type="warn">
  原本提案的「固定 20 元手續費」對鑽石、黑玫瑰、熔岩魚這種高單價物資等於白送，會被當作免費取貨。改成市價比例後，越貴的物品手續費越高，貢獻門檻才扛得住。
</Callout>

## 倉庫設定（會長 / 副會長）

`/公會 倉庫` 畫面右下「⚙️ 倉庫設定」按鈕，打開 Modal 後可調這四項：

| 設定 | 預設 | 範圍 |
| --- | --- | --- |
| 每日次數（每位成員） | 2 | 1 ~ 3 |
| 手續費率 | 10% | 5% ~ 20% |
| 入會時間門檻 | 24 小時 | 6h ~ 168h |
| 最低貢獻門檻 | 200 | 0 ~ 5000 |

- 留**空白**送出 = 恢復系統預設。
- 數字超出範圍會被擋下，不會破壞平衡。

## 倉庫紀錄（會長 / 副會長）

`/公會 倉庫` 畫面右下「📜 紀錄」按鈕，列出近 30 天的存 / 取流水，每筆顯示時間、操作者、物品、數量、手續費。頂部固定一行**會長監督**：

> 近 30 天會長淨領取價值：N 幣（含手續費回流 M 幣）

讓所有會員一眼確認會長有沒有自肥。

## 大額存入廣播

單次存入**市價 ≥ 10,000 逼幣**（例如黑玫瑰 ×8、熔岩魚 ×17）時，會自動在公會頻道發布「🎁 大額捐贈！」公告，讓會員看到誰在挺公會。

## 公會解散時的倉庫怎麼處理

解散公會時，倉庫所有物品依當下單價**整批折現入金庫**，再依現行解散規則平分給有資格的成員（在會時間達門檻者）。沒有人會「私吞倉庫」這種事，因為倉庫從來就不是個人資產。
