# 公會 🏰

公會是一群玩家**組隊長期經營**的小組織。會長負責管理人員與金庫，成員一起**捐款累積等級**換取整個公會通用的 buff，並合力完成**每週任務**領取金庫獎勵。

## 指令總覽

| 子指令 | 用途 | 對象 |
| --- | --- | --- |
| `/公會 建立 [名稱]` | 建立新公會（花費 5,000 逼幣） | 未加入公會的玩家 |
| `/公會 資訊 [名稱]` | 查看公會詳情（不填＝查自己的公會） | 所有人 |
| `/公會 邀請 [使用者]` | 邀請玩家加入公會 | 會長 / 副會長 |
| `/公會 申請 [名稱] [理由]` | 主動申請加入指定公會 | 未加入公會的玩家 |
| `/公會 退會` | 退出目前所屬公會 | 一般成員、副會長 |
| `/公會 捐款 [金額]` | 捐逼幣進公會金庫 | 公會成員 |
| `/公會 任務` | 查看本週公會任務進度（達標後按鈕領取） | 公會成員 |
| `/公會 倉庫` | 查看公會倉庫存貨與今日領取額度 | 公會成員 |
| `/公會 存入 [物品] [數量]` | 把背包 / 魚袋資源捐入倉庫 | 公會成員 |
| `/公會 領取 [物品] [數量]` | 從倉庫領取資源（付手續費） | 公會成員 |
| `/公會 寄售 [物品] [數量] [售價]` | 把倉庫物資上架到市集 | 會長 / 副會長 |
| `/公會 熔爐` | 用礦石製造建材 / 鋼錠（Lv.4 解鎖） | 公會成員 |
| `/公會 建築` | 礦坑 / 訓練場 / 倉庫擴建（Lv.2 解鎖） | 公會成員（升級限會長 / 副會長） |
| `/公會 排行` | 查看伺服器公會排行榜 | 所有人 |

<Callout type="info">
  以下功能改由**按鈕**操作，沒有獨立 slash command：
  - **編輯簡介**：`/公會 資訊` → 會長 / 副會長看到「📝 編輯簡介」按鈕
  - **待審申請**：`/公會 資訊` → 有 pending 申請時會長 / 副會長看到「📨 待審申請 N」紅色按鈕
  - **領週任務獎勵**：`/公會 任務` → 達標時會長看到「🎁 一鍵領取」按鈕
  - **倉庫設定 / 紀錄**：`/公會 倉庫` → 會長 / 副會長看到對應按鈕
  - **公會管理**（踢人 / 轉讓會長 / 指派 / 撤銷副會長 / 解散）：`/公會 資訊` 底部的「⚙️ 公會管理」按鈕區，會長 / 副會長按權限顯示不同按鈕
</Callout>

## 建立公會

```
/公會 建立 [名稱]
```

繳交 **5,000 逼幣**即可建立一個新公會，並自動成為**會長**。

| 規則 | 說明 |
| --- | --- |
| 名稱長度 | 1–12 字 |
| 禁字 | `@`、`#`、`<`、`>`、`` ` `` |
| 名稱唯一 | 同伺服器不允許重複名稱 |
| 加入限制 | 每位玩家在同一伺服器**只能屬於一個公會** |

<Callout type="info">
  想另立門戶？先 `/公會 退會`，或在 `/公會 資訊` →「⚙️ 公會管理」→「💥 解散公會」結束目前公會，才能再 `/公會 建立`。
</Callout>

## 公會簡介 / 會規

每個公會都可以放一段**對外的自介文字**——可以寫公會風格、入會條件、活動時間、捐款建議等等，讓申請者進來前就先有概念。

編輯入口在 `/公會 資訊` 底部的「📝 編輯簡介 / 會規」按鈕，**會長與副會長**都能按；按下後彈出 Modal 輸入。

- 內容最多 **500 字**，支援多行。
- 留空送出＝清除目前的簡介。
- 在 `/公會 資訊` 會以「📜 公會簡介 / 會規」區塊顯示，**所有人**都看得到。

<Callout type="info">
  寫簡介時可以列幾條重點：例如「歡迎挖礦狂魔」「每週至少打一次 BOSS」「捐款隨意但越多越好」——對申請者來說比公會名稱更有參考價值。
</Callout>
