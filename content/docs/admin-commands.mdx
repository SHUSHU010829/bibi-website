# 管理員指令

以下指令需要**管理權限**，僅供伺服器管理員使用。一般成員無法執行。本頁只列出用途概覽，不展開詳細參數。

## 經濟與等級

| 指令 | 用途 |
| --- | --- |
| `/give-coins` | 發放或扣除指定成員的逼幣 |
| `/circulation` | 查看全伺服器的逼幣總流通量 |
| `/level-admin give-xp` | 為某個身份組的所有成員統一加 XP |
| `/level-admin roles` | 管理「等級 → 身份組」對應表（set / remove / list / apply） |

## 賭場與股票

| 指令 | 用途 |
| --- | --- |
| `/casino-stats` | 查看賭場各遊戲的投報率（RTP）統計 |
| `/lotteryadmin` | 樂透管理（強制開獎、建立期數、執行訂閱、發送提醒等） |
| `/stock-event` | 觸發或管理股市事件（崩盤、利多等） |

## 內容管理

| 指令 | 用途 |
| --- | --- |
| `/food-admin` | 食物資料管理（add 新增 / batch 批次 / import 匯入菜單 / delete 刪除） |
| `/recommendation-admin` | 推薦管理（刪除、重新分析分類） |

## 面板與系統設定

| 指令 | 用途 |
| --- | --- |
| `/ticket setup` | 在當前頻道部署票務面板 |
| `/ticket suggestion-setup` | 部署建議申訴面板 |
| `/ticket proposal` | 發起／結束遊戲頻道提案投票（start / end / cancel） |
| `/ticket vote create` | 依範本發起一場投票（遊戲新增／封存、活動、規則修改、一般提案） |
| `/setup-roles` | 設定自助身份組選單（send / add / remove） |

<Callout type="info">
  投票範本的選項與通過門檻說明請見[客服與投票](/docs/community)。
</Callout>
