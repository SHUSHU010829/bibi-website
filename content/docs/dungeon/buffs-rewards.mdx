# 食物 Buff、公會與稱號

## 食物 buff（戰前必吃）

`/烹飪` 有 5 道專為地下城設計的料理，買菜備料先看 [烹飪](/docs/cooking)：

| 料理 | 效果 | 適用情境 |
| --- | --- | --- |
| 🔪 鯊魚生魚片 | 地下城 ATK **+18**／2 hr | 衝 5F 與 Mini-BOSS 必備 |
| 🍜 鯊魚麵 | 地下城 ATK **+20**／3 hr | 長時間連刷 |
| 🐙 章魚鎧甲鍋 | 地下城 DEF **+10**／2 hr | 怕被反殺、撐麻痺主題（冰窟）必備 |
| 🍖 燉肉鍋 | 地下城 **HP 上限 +30**／2 hr | 戰前一鍋，等於白送 30 HP |
| 🍱 戰前晚餐 | **全屬性 +10%**／1 hr | 衝刺場（Mini-BOSS）一次堆滿 |

<Callout type="info">
  **食物 buff 衝突規則**：同 type 互蓋（兩款 ATK 料理只能擇一），**全屬性 +10%** 與任一單屬性 buff 互斥。所以這五道**最多同時吃三種**：鯊魚生魚片／鯊魚麵擇一（ATK）＋ 章魚鎧甲鍋（DEF）＋ 燉肉鍋（HP 上限）；或單獨吃戰前晚餐（會蓋掉其他單屬性 buff）。煤炭烤製版本效果更強、保鮮 ×1.5，囤貨黨直接無腦勾起來。
</Callout>

## 公會加成

加入 [公會](/docs/guild) 後，公會建築與等級會疊到地下城三圍上（詳細數值見公會頁）：

- **HP 上限**、**ATK**、**DEF**、**體力上限**、**體力恢復速度**、**戰利品金幣 / 礦石倍率**
- 公會 Lv.4 與 Lv.5 還會解鎖**主題王戰加成**（攻擊力 / 出手次數）

## 稱號

地下城相關稱號（與其他遊戲共用 `/稱號 設定` 與 `/檔案` 等級卡展示）：

| 稱號 | 解鎖條件 | 性質 |
| --- | --- | --- |
| 🏆 地下城征服者 | 本月地下城通關次數 Top 1 | 每月輪替 |
| 🐉 屠龍者 | 累積擊敗 Mini-BOSS **10 次** | 永久 |
| ❄️ 寒冰英雄 | 擊敗冰晶女王 **5 次** | 永久（主題稱號） |
| 🏛️ 遺跡學者 | 通關廢墟主題 **50 次** | 永久（主題稱號） |

完整稱號清單與獎勵見 [遊戲稱號與成就](/docs/leveling/game-titles)。

## 跨系統串接

- **挖礦** → 提供武器／盾合成的礦石（鐵礦、煤炭、鑽石）與藏寶圖碎片
- **釣魚 + 農場 + 烹飪** → 提供地下城專用 buff 食物
- **公會** → HP 上限、ATK、DEF、體力、戰利品倍率全部疊加
- **裝備修復** → 武器／盾耐久回復都走 `/裝備` 統一管理
- **背包「地下城」分類** → 集中管理生命藥水、體力藥水
