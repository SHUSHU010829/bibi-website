# 斷劍王與亡靈制

傳說之劍（🔥，`/合成` 打造的最強武器，ATK 180 / DEF 40）雖然威猛，耐久耗盡一樣會**斷**。系統會記錄每個人在地下城把傳說之劍砍斷的次數，做成**每週斷劍榜**——但這不是榮耀榜：**當週斷最多的人**會被封為 **☠️ 斷劍王**，並背上「**亡靈制**」詛咒（debuff）整整一週。

<Callout type="warn">
  亡靈制是**懲罰**，不是加成。斷劍化成的怨靈會反噬持劍者，讓你在地下城變弱。愛惜你的傳說之劍，別讓自己戴上這頂亡靈王冠。
</Callout>

## 斷劍紀錄與公告

- 只要你在地下城把 🔥 傳說之劍砍斷（**不論勝敗**都算），就記一次斷劍紀錄。
- 每斷一次會發一則**小公告**到公告頻道，顯示是誰斷的、**本週已斷幾次**、**生涯累計幾次**。
- 戰鬥結算面板也會標示「💥 傳說之劍應聲斷裂！」，頻道戰報同步帶上一行。

<Callout type="info">
  只有**傳說之劍**會進斷劍榜與觸發公告；鐵劍、鋼劍、黃金劍、鑽石劍斷掉不列入。
</Callout>

## 斷劍榜

```
/排行榜 類別:斷劍王
```

| 區間 | 統計內容 |
| --- | --- |
| **本週** | 當週（週一 00:00 ~ 下週一）把傳說之劍砍斷的次數 |
| **全部時間** | 生涯累計斷劍次數（不重置） |

每週榜以**週一為首**的 Mon–Sun 週界計算，跟 [挖礦週榜](/docs/mining) 同一個週界。

## 每週結算

**每週一 00:05** 結算上週斷劍榜：

<Steps>
  <Step>統計上週每個人把傳說之劍砍斷的次數</Step>
  <Step>取**斷最多**的人封為 ☠️ 斷劍王，烙上「亡靈制」詛咒（效期到下週一結算）</Step>
  <Step>同伺服器上一任持有者自動卸任（同時只會有一位背詛咒）</Step>
  <Step>在公告頻道公布斷劍王與本週斷劍榜，並私訊當事人</Step>
</Steps>

<Callout type="info">
  **上週沒人斷劍就不結算。** 若整週沒有任何人把傳說之劍砍斷，就不會產生斷劍王、也不會有人被施加亡靈制——這一週平安無事。
</Callout>

## 亡靈制（詛咒內容）

被封為斷劍王期間，在**地下城**中承受以下 debuff：

| 效果 | 說明 |
| --- | --- |
| ⚔️ 地下城傷害 **−15%** | 亡靈纏身，出手變鈍（ATK 最低保底 1，不會歸零） |
| 🗡️ 武器每場**多扣 1 耐久** | 兵刃更快崩壞——諷刺地讓你更容易再斷一把劍 |
| 👻 亡靈軍團作祟 | 地下城**勝利**時有 **15%** 機率被怨靈奪走 **200 逼幣**、吸走 **12% HP**（HP 最低保底 1） |

<Callout type="info">
  亡靈制與食物、公會、世界事件等加成獨立計算，直接疊在最終數值上（傷害是相減）。中了詛咒還是能靠 [食物 buff](/docs/dungeon/buffs-rewards) 與公會加成把三圍補回來。
</Callout>

### 效期與解除

- **效期一週**：從被封斷劍王的那個週一，到下週一結算為止。
- **自動解除**：詛咒只存到期時間（compute-on-read），時間一到自動失效，不需要任何操作。
- **連任續命**：下一週你又是斷最多的人，詛咒會直接續期到再下週。
- **換人卸任**：有別人超過你成為新斷劍王，你的亡靈制會在結算時立刻解除。

在 `/加成` → **時效狀態** 可以看到目前是否被亡靈制纏身與剩餘時間。

## 怎麼避免中詛咒？

- **別硬撐低耐久的傳說之劍**：耐久偏低時先到 [`/裝備`](/docs/mining/crafting) 用礦石修復，或改用鑽石劍打低樓層，把傳說之劍留給關鍵戰。
- **少刷幾場**：斷劍榜比的是「次數」，狂刷到斷好幾把反而把自己送上王座。
- 真的中了也別慌：詛咒一週就過，期間靠料理與公會加成硬扛即可。
