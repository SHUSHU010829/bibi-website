# 地下城

帶上武器、盾、生命藥水深入地下城打怪，撿礦石、逼幣、藥水、種子甚至傳說素材碎片 ⚔️。Phase H+ 把單回合勝率制改成**多回合 HP 戰鬥**：你跟怪物各自有 HP、ATK、DEF，每回合輪流出招，撐到對方倒下才算贏；HP 也跨戰鬥持續存在，被打殘了要等回血、吃藥水或喝食物 buff，不像舊版打完就重置。

```
/地下城
```

<Callout type="warn">
  **赤手空拳能進，但勝率極低。** 至少先用 [`/合成`](/docs/mining/crafting) 打一把**鐵劍**（鐵礦 ×15）再來；Lv.5 解鎖 [盾](/docs/mining/crafting) 後也順手做一面**鐵盾**，能格擋怪物攻擊、大幅減少 HP 耗損。
</Callout>

## 戰鬥流程一覽

進入地下城後會抽選**主題**＋**樓層**裡的一隻怪物，然後進入回合制戰鬥：

<Steps>
  <Step>**玩家攻擊**：扣怪物 HP，可能暴擊（×1.5 傷害）</Step>
  <Step>**怪物反擊**：扣玩家 HP；有盾可機率**格擋**（完全免傷）</Step>
  <Step>**狀態效果結算**：中毒／燃燒每回合扣 HP，暈眩跳過下回合</Step>
  <Step>**自動藥水**（若開啟且 HP 低於 30%）：自動喝藥水回血</Step>
  <Step>重複 1–4，直到怪物倒下、玩家倒下、或滿 **20 回合**判平手</Step>
</Steps>

戰鬥結束後：

- **勝利**：拿戰利品（依樓層倍率），怪物 HP 重置給下一場
- **死亡**：HP 歸 0，**25% 機率**掉落部分背包戰利品（隨機一格折扣），體力與武器耐久照常扣
- **撤退**（戰鬥中按「逃跑」）：**退 50% 體力**，武器耐久照扣，無戰利品
- **20 回合平手**：視同撤退處理

<Callout type="info">
  HP 歸 0 不會掉等級、不會清背包礦石，只會 25% 機率掉落該場「**將要**獲得的戰利品」其中一格。回房間（離開地下城面板）後 HP 不會立刻回滿，要等自然回復或喝藥水。
</Callout>

## 體力系統

地下城跟挖礦／釣魚共用同一個**體力池**：

- 上限 **12 點**（公會 Lv.5 + 公會建築可拉到上限）
- 每 **1 小時** +1 點，自然回復不分上下線
- 喝**體力藥水**（600 逼幣／瓶，到 `/商店 地下城道具` 買）：+5 體力，每日最多 3 瓶
- 各樓層體力消耗：1F–2F **1 點**、3F–4F **2 點**、5F & Mini-BOSS **3 點**

<Callout type="info">
  體力 0 時不能進地下城，但 HP 還會繼續回。
</Callout>
