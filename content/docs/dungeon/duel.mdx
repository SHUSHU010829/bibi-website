# 1v1 決鬥

向其他玩家發起 **1v1 逼幣決鬥**，勝者通吃彩池 ⚔️。決鬥是純 PvP 玩法，戰鬥力由雙方的**武器**決定，賭注由雙方各押。

```
/決鬥 對象 賭注
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `對象` | 要挑戰的玩家（不能是 bot 或自己） | 是 |
| `賭注` | 雙方各押的逼幣數，**100 ~ 5,000** | 是 |

## 流程

<Steps>
  <Step>發起者下指令後，對方會收到**接受 / 拒絕 / 取消**邀請</Step>
  <Step>對方按下「**接受決鬥**」即扣雙方賭注並開打</Step>
  <Step>勝者拿走 **2 倍賭注**（扣 5% 系統抽成）</Step>
  <Step>對方按「拒絕／取消」，或 **2 分鐘**內未回應自動取消，賭注全額退回</Step>
</Steps>

## 戰鬥力

勝負由雙方**武器戰鬥力 + 隨機數**決定，武器越好越有優勢——跟 [地下城](/docs/dungeon) 同一套基礎攻擊力。決鬥**不消耗體力、不扣武器耐久、不扣 HP**，是純粹的 PvP 賭金玩法。

| 武器 | 戰鬥分 |
| --- | --- |
| 👊 赤手空拳 | 20 |
| 🗡️ 鐵劍 | 45 |
| ⚔️ 鋼劍 | 70 |
| 🌟 黃金劍 | 100 |
| 💎 鑽石劍 | 140 |
| 🔥 傳說之劍 | 200 |

每方戰鬥分再加 **0–30** 的隨機數，數值高者獲勝。

## 限制

- **每日最多 3 場**（成功完成的場次計算，未接受的不算）
- 同一對玩家**冷卻 6 小時**才能再戰
- 同時只能有一場進行中的決鬥邀請，等它結束才能再發起新的
- 賭注下限 **100 逼幣**、上限 **5,000 逼幣**
- 系統抽成 **5%**（從勝者的彩池扣）

<Callout type="warn">
  決鬥純運氣＋武器決勝。如果想穩贏，先去 [合成](/docs/mining/crafting) 把武器升到鋼劍以上再來；連赤手對打鋼劍的對手幾乎是穩輸。
</Callout>

<Callout type="info">
  決鬥**不消耗任何戰鬥資源**（HP、體力、耐久、藥水），跟 [地下城](/docs/dungeon) 戰鬥不會互相影響。被人挑釁了直接接，無痛打一場。
</Callout>
