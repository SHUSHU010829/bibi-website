# 歡迎使用逼逼機器人

逼逼機器人（bibi-bot）是一隻多功能的 Discord 機器人，集**等級成長、逼幣經濟、賭場遊戲、樂透、股票市場、商店、社群活動與各種實用工具**於一身，讓伺服器更熱鬧、更好玩。

## 快速開始

先加入 Discord 伺服器，即可開始體驗所有功能。

[加入 Discord](https://discord.gg/shushu010829)

所有功能都以**斜線指令**操作：在訊息框輸入 `/`，就會跳出指令選單與參數提示。第一次使用建議先看 [開始使用](/docs/getting-started)。

## 功能總覽

### 成長與經濟
- [等級系統](/docs/leveling) — 靠聊天、語音、簽到累積經驗值（XP）升等，解鎖徽章與稱號。
- [逼幣經濟](/docs/economy) — 錢包、逼幣來源、轉帳、定存、每日／每週任務、福利與財產稅。
- [商店與背包](/docs/shop) — 用逼幣購買顏色身份組、加成藥水、錢包卡面與自訂稱號。

### 遊戲與投資
- [賭場遊戲](/docs/casino) — 拉霸、骰寶、二十一點、HI-LO、輪盤、尋寶、火箭、射龍門、賽馬、德州撲克。
- [樂透](/docs/lottery) — 大樂透 6/49 與小樂透 3/20，支援包牌與訂閱。
- [股票市場](/docs/stocks) — 買賣股票、領配息、看走勢圖。
- [挖礦與打工](/docs/mining) — 挖礦、打工賺逼幣，合成鎬子升級裝備。
- [釣魚與烹飪](/docs/fishing) — 釣魚賺逼幣、解鎖三大釣點，把魚煮成強力 buff 食物。

### 社群與互動
- [互動活動](/docs/activities) — 自辦活動、有獎預測與問答。
- [客服與投票](/docs/community) — 票務客服、建議申訴、社群投票表決。
- [伺服器功能](/docs/server-features) — 自助身份組、動態語音頻道、邀請獎勵。

### 工具與自動推播
- [實用工具](/docs/utilities) — 吃什麼、地點推薦、天氣、加密貨幣、即時匯率、抽籤與更多趣味指令。
- [自動推播](/docs/automated) — 每日早安卡、Steam 特價、免費遊戲、Twitch 開台通知與週報。

### 管理員
- [管理員指令](/docs/admin-commands) — 需要管理權限的指令一覽。
