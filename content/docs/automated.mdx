# 自動推播

逼逼機器人會定時主動推播各種內容，不需要你下任何指令。以下時間皆為台北時間。

| 功能 | 推播時間 | 內容 |
| --- | --- | --- |
| 早安卡 | 每天 08:00 | 每日運勢卡片，含農曆／生肖資訊與一句詩詞 |
| Steam 特價 | 每 2 小時（活躍時段） | 篩選史低價的 Steam 遊戲特價情報 |
| 免費遊戲 | 每 6 小時（活躍時段） | Epic、Steam 等平台的限免遊戲情報 |
| Twitch 開台通知 | 每分鐘檢查 | 追蹤的實況主開台時即時通知並標註身份組 |
| RSS 推播 | 每 2 小時 | 訂閱來源（Threads 等）的最新貼文 |
| 每週報告 | 每週日 21:00 | 伺服器一週的活躍統計報告 |

<Callout type="info">
  想接收 Twitch 開台通知，記得到[自助身份組](/docs/server-features)領取對應的通知身份組。
</Callout>

## 早安卡

每天早上機器人會在主頻道發送一張早安卡，結合當天的農曆、生肖與一句詩詞，為大家開啟新的一天。

## 遊戲情報

機器人會自動爬取 Steam 史低特價與各平台限時免費遊戲，並推播到對應頻道，讓你不錯過撿便宜的機會。特價推播只在白天活躍時段進行，避免半夜洗版。

## 開台通知

追蹤的實況主一開台，機器人會立即在通知頻道發出開台訊息並標註通知身份組，讓粉絲第一時間進去看直播。

## 每週報告

每週日晚上，機器人會整理伺服器一週的活躍數據（訊息、語音、互動等）做成報告，回顧大家這週的熱度。

## 推播頻道

機器人會把不同類型的公告推播到對應的主題頻道，方便你各看各的：

| 頻道 | 推播內容 |
| --- | --- |
| 特價喜加一 │ 💰 | Steam 特價、限時免費遊戲情報 |
| 華爾街日報 │ 📰 | [股市](/docs/stocks)突發事件與市場報告 |
| 唐人街彩報 │ 🗞️ | 拉霸大獎、賽馬與[樂透](/docs/lottery)開獎公告 |
| 逼逼大灑幣 │ 🧧 | [互動活動](/docs/activities)（自辦活動、預測、問答）發布 |

<Callout type="info">
  早安卡與每週報告會發在伺服器主頻道。實際頻道仍以伺服器內公告為準。
</Callout>
