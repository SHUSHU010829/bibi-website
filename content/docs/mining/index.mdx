# 挖礦與打工

除了賭場與股票，你還可以靠**挖礦**和**打工**穩定賺取逼幣。挖到的礦石可以賣錢，也能拿去**合成裝備**：用礦石打造更好的**鎬子**挖得更快、挖到更稀有的礦，或鍛造**武器**去闖地下城打怪。想闖**地下城**、跟人**決鬥**，都得先有一把像樣的劍。礦石還能**贈送**給朋友，或拿到 [玩家交易](/docs/trading) 跟其他玩家一口價買賣、徵求、競標或以物易物。

本頁是挖礦系統的核心玩法（挖礦、賣出、打工、檔案）；延伸玩法請見：

- [合成與裝備](/docs/mining/crafting) — 鎬子、武器、釣竿合成與鎬子修復
- [地下城與決鬥](/docs/mining/dungeon) — 體力、戰鬥力、戰利品、突發事件、PVP 決鬥
- [BOSS 共鬥](/docs/mining/boss) — 每週六 21:00 全伺服器一起打 BOSS、階段轉換、Combo、尾刀獎勵
- [玩家交易](/docs/trading) — 市集（一口價、徵求、競標、贈送）與交易所（物物交換）

<Callout type="info">
  挖礦、賣出、打工、合成、裝備、地下城、決鬥、贈送等指令都在 **財富牢改城 │ 💼** 頻道使用；`/市集` 與 `/交易所` 改在 **暗巷易物所** 頻道使用（`/背包`、`/市集 逛攤`、`/市集 我的攤位`、`/交易所 我的` 是只有自己看得到的私人指令，哪裡都能開）。
</Callout>

## 挖礦

```
/挖礦
```

挖一次礦，每次有 **2 小時**冷卻時間，挖到的礦石會放進背包。稀有度越高的礦越值錢，挖到**鑽石**時還會全服公告慶祝 ✨。

<Callout type="info">
  想少等冷卻？冷卻中打開 `/背包`，對「CD 縮短券」按 **使用** 就能立即少等 30 分鐘（剩餘不足 30 分會直接歸零）。
</Callout>

### 礦石一覽

| 礦石 | 稀有度 | 賣價 | 基礎掉落機率 | 初始挖礦數量 |
| --- | --- | --- | --- | --- |
| 石頭 | 普通 | 8 逼幣 | 55% | 1–3 |
| 煤炭 | 普通 | 20 逼幣 | 25% | 1–2 |
| 鐵礦 | 稀有 | 60 逼幣 | 12% | 1 |
| 黃金 | 稀有 | 200 逼幣 | 6% | 1 |
| 鑽石 | 傳說 | 800 逼幣 | 1% | 1 |

<Callout type="info">
  上表「基礎掉落機率」是 **luck 0**（沒有任何加成）時各礦石的權重比例。「幸運值（luck）」越高，挖到稀有礦（鐵礦／黃金／鑽石）的機率越高、石頭與煤炭相對變少。鎬子、幸運藥水與贊助／Twitch 加成都會提升 luck，但全部加總最多 **+25%**。「初始挖礦數量」是每次挖到該礦時的基礎顆數（鑽石鎬會再 **+1**），實際進帳會再乘上數量。
</Callout>

### 賭石（找鑑定師）

挖到**石頭**時，`/挖礦` 的結果訊息會多出一顆 **🔍 找鑑定師賭石** 按鈕。付費請鑑定師把這次挖到的石頭**逐顆開出**，有機率變成更值錢的礦——當然，也可能全部碎掉！

- **只有剛挖到時能賭**：按鈕只認你**最新一次**挖到的石頭，挖到別的礦或再挖一次就失效，逾時（預設 10 分鐘）也會失效。每次挖到的石頭**只能賭一次**。
- **費用**：每顆石頭 **150 逼幣**（挖到幾顆就一次幫你全開）。
- **開出機率**（高風險高報酬）：

| 結果 | 機率 |
| --- | --- |
| 💥 碎掉（什麼都沒有） | 60% |
| 煤炭 ×1–2 | 18% |
| 鐵礦 ×1 | 10% |
| 黃金 ×1 | 7% |
| 💎 鑽石 ×1 | 2% |

<Callout type="warn">
  賭石是純運氣的金幣 sink，長期期望值是**虧的**，圖的就是那一發開出鑽石的爽感（開出鑽石一樣會全服公告）。背包快滿或金幣不夠時無法賭，先 `/賣出` 清一清。
</Callout>

## 背包

```
/背包
```

你的個人背包（只有自己看得到），把挖礦與商店的東西整合在同一張卡片：

- **礦石**：各礦石數量、背包容量（基礎 **100 格**，裝滿就不能再挖，要先賣出）、全部賣出可得逼幣。
- **挖礦狀態**：目前的鎬子與耐久、下次可挖礦時間。
- **挖礦道具**：幸運藥水（挖礦時自動生效）、CD 縮短券（冷卻中可按 **使用**）、**傳說素材碎片**（地下城掉落，**合成傳說之劍的材料** —— 想確認數量就來這裡看）。
- **商店道具與 buff**：你買的顏色身份組／卡面／自訂稱號、生效中的加成與到期時間。可裝備的道具（顏色身份組、卡面、等級卡顏色、自訂稱號）都整合在**同一個下拉選單**裡，選到自訂稱號會跳出視窗讓你設定文字，其餘則直接裝備。

<Callout type="info">
  **CD 縮短券**：挖礦冷卻中時在 `/背包` 對它按 **使用**，立即少等 30 分鐘（剩餘不足 30 分直接歸零）。最多持有 **30 張**，可到 `/商店` 購買。
</Callout>

## 賣出

```
/賣出 [物品] [數量]
```

把背包裡的礦石、魚或蔬菜賣給系統換逼幣，依**當日行情**計價（蔬菜為固定價）。這是統一賣給系統的指令——`物品` 下拉選單同時涵蓋各種礦石、魚種與蔬菜。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `物品` | 要賣的礦石、魚種或蔬菜（下拉選項） | 否 | 不填則賣出背包全部 |
| `數量` | 要賣的數量 | 否 | 不填則賣出該物品全部 |

<Callout type="info">
  礦石與魚的收購價依**當日行情**波動 **±30%**（由日期決定、全服一致、同一天重算不變），可用 `/行情` 查看；蔬菜為固定價（紅蘿蔔 30／玉米 90／草莓 250／黑玫瑰 900）。`/賣出` 在 **財富牢改城 │ 💼** 或 **神農百草園 │ 🥕** 都能用。釣魚見 [釣魚](/docs/fishing)、烹飪見 [烹飪](/docs/cooking)、種田見 [農場與種田](/docs/farming)。
</Callout>

## 打工

```
/打工
```

打工賺取穩定收入，冷卻 **4 小時**，每日最多 **6 次**，適合在挖礦冷卻時順手賺一點。

打工設有**等級制**：依**累計打工次數**自動升級，等級越高每次的報酬區間越好（升級無須操作，達標自動生效）。

| 等級 | 稱號 | 累計打工次數 | 每次報酬 |
| --- | --- | --- | --- |
| Lv.1 | 打工新手 | 0 次 | 80–120 逼幣 |
| Lv.2 | 兼職人員 | 60 次 | 95–135 逼幣 |
| Lv.3 | 正式員工 | 220 次 | 110–155 逼幣 |
| Lv.4 | 資深員工 | 480 次 | 125–175 逼幣 |
| Lv.5 | 王牌員工 | 900 次 | 140–185 逼幣 |

## 稱號與成就

挖礦的成就稱號（煤炭採集者、鐵鍛師、寶石獵人、傳說礦工，以及每週爭奪的礦坑之王）已整合進**跨遊戲稱號系統**，和賭場、股市、樂透、拍賣的稱號共用同一套 `/稱號 設定` 與 `/檔案` 的「🏆 成就」分頁與等級卡展示槽。完整稱號清單與解鎖條件請見 [遊戲稱號與成就](/docs/leveling/game-titles)。

## 礦工檔案

```
/檔案
```

到「⛏️ 礦工檔案」分頁，查看自己的完整檔案：展示稱號、已解鎖稱號數、週冠次數、目前鎬子與耐久、錢包餘額、背包用量，以及**生涯統計**（累積挖礦次數、合成件數、地下城次數）與**歷史採集量**。

## 挖礦排行

```
/排行榜 類別:挖礦
```

本週挖礦**數量**排行榜（前 10 名）。每週一 **00:01** 結算，把上週第一名頒發 **礦坑之王 👑** 稱號並在頻道公告，同時卸下上一任。快搶下第一個礦坑之王！
