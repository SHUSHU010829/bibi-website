# 挖礦與打工

除了賭場與股票，你還可以靠**挖礦**和**打工**穩定賺取逼幣。挖到的礦石可以賣錢，也能拿去**合成更好的鎬子**，讓你挖得更快、挖到更稀有的礦。鎬子越好，闖**地下城**、跟人**決鬥**也越有優勢。礦石還能**贈送**給朋友或上**拍賣行**競標。

<Callout type="info">
  挖礦、賣礦、打工、合成、裝備、地下城、決鬥、贈送、拍賣等指令都在 **財富牢改城 │ 💼** 頻道使用（`/背包` 是只有自己看得到的私人指令，哪裡都能開）。
</Callout>

## 挖礦

```
/挖礦
```

挖一次礦，每次有 **2 小時**冷卻時間，挖到的礦石會放進背包。稀有度越高的礦越值錢，挖到**鑽石**時還會全服公告慶祝 ✨。

<Callout type="info">
  想少等冷卻？冷卻中打開 `/背包`，對「CD 縮短券」按 **使用** 就能立即少等 30 分鐘（剩餘不足 30 分會直接歸零）。
</Callout>

### 礦石一覽

| 礦石 | 稀有度 | 賣價 |
| --- | --- | --- |
| 石頭 | 普通 | 8 逼幣 |
| 煤炭 | 普通 | 20 逼幣 |
| 鐵礦 | 稀有 | 60 逼幣 |
| 黃金 | 稀有 | 200 逼幣 |
| 鑽石 | 傳說 | 800 逼幣 |

<Callout type="info">
  「幸運值（luck）」越高，挖到稀有礦的機率越高。鎬子、幸運藥水與贊助／Twitch 加成都會提升 luck，但全部加總最多 +25%。
</Callout>

## 背包

```
/背包
```

你的個人背包（只有自己看得到），把挖礦與商店的東西整合在同一張卡片：

- **礦石**：各礦石數量、背包容量（基礎 **100 格**，裝滿就不能再挖，要先賣礦）、全部賣出可得逼幣。
- **挖礦狀態**：目前的鎬子與耐久、下次可挖礦時間。
- **挖礦道具**：幸運藥水（挖礦時自動生效）、CD 縮短券（冷卻中可按 **使用**）、**傳說素材碎片**（地下城掉落、未來合成用 —— 想確認數量就來這裡看）。
- **商店道具與 buff**：你買的顏色身份組／卡面／自訂稱號、生效中的加成與到期時間，並能直接用選單裝備。

<Callout type="info">
  **CD 縮短券**：挖礦冷卻中時在 `/背包` 對它按 **使用**，立即少等 30 分鐘（剩餘不足 30 分直接歸零）。最多持有 **30 張**，可到 `/商店` 購買。
</Callout>

## 賣礦

```
/賣礦 [礦石] [數量]
```

把背包裡的礦石賣給系統換逼幣。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `礦石` | 要賣的礦石種類（下拉選項） | 否 | 不填則賣出背包全部 |
| `數量` | 要賣的數量 | 否 | 不填則賣出該礦石全部 |

## 打工

```
/打工
```

打工賺取穩定收入，每次可得 **80–120 逼幣**。冷卻 **4 小時**，每日最多 **6 次**，適合在挖礦冷卻時順手賺一點。

## 合成

```
/合成 裝備 [確認]
```

用礦石合成更好的鎬子。鎬子能縮短挖礦冷卻、提升 luck，鑽石鎬還能一次多挖一個礦。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `裝備` | 要合成的鎬子（下拉選項） | 是 | — |
| `確認` | 目前的鎬子還有耐久時，設為 `是` 才會替換掉它 | 否 | 否 |

### 鎬子與配方

| 鎬子 | 合成材料 | 挖礦冷卻 | luck | 耐久 |
| --- | --- | --- | --- | --- |
| 木鎬 | 初始裝備 | — | — | 永久 |
| 鐵鎬 | 鐵礦 ×15 | −30 分 | +5% | 50 次 |
| 黃金鎬 | 黃金 ×5、鐵礦 ×20 | −45 分 | +8% | 50 次 |
| 鑽石鎬 | 鑽石 ×5、黃金 ×10 | −60 分（並一次多挖 1 個礦） | +12% | 50 次 |

<Callout type="warn">
  除了木鎬，其他鎬子都有耐久（50 次）。耐久耗盡會自動換回木鎬，需要再合成一把。
</Callout>

## 裝備

```
/裝備
```

查看目前鎬子的屬性，以及各個合成配方所需材料的進度（已足夠 ✅／還缺 ❌）。

## 地下城

```
/地下城
```

消耗 **1 點體力**深入地下城戰鬥，勝利可獲得礦石、逼幣或傳說素材碎片 ⚔️。

- **體力**：上限 **10 點**，每 **1 小時**自動回復 1 點，耗盡時要休息一下再來。
- **勝率**：由你的攻擊力與怪物血量（50–200）決定，介於 **20%–90%**。攻擊力越高勝率越高。
- **攻擊力**：基礎 30，依鎬子加成。合成更好的鎬子能明顯提升勝率。

| 鎬子 | 攻擊力 |
| --- | --- |
| 木鎬 | 30 |
| 鐵鎬 | 45 |
| 黃金鎬 | 55 |
| 鑽石鎬 | 65 |

### 戰利品

勝利後隨機掉落以下其中一種：

| 戰利品 | 內容 | 機率 |
| --- | --- | --- |
| 礦石 | 鐵礦 ×3 | 40% |
| 逼幣 | 150–300 逼幣 | 35% |
| 傳說素材碎片 | ×1（保留供未來合成用） | 15% |
| 槓龜 | 什麼都沒掉 | 10% |

<Callout type="info">
  背包裝滿時掉落的礦石會自動折算成逼幣，不會浪費。
</Callout>

## 決鬥

```
/決鬥 對象 賭注
```

向其他玩家發起 1v1 逼幣決鬥，勝者通吃彩池 ⚔️。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `對象` | 要挑戰的玩家（不能是 bot 或自己） | 是 |
| `賭注` | 雙方各押的逼幣數，**10 ~ 5000** | 是 |

發起後對方會收到邀請，按下「接受決鬥」即扣雙方賭注並開打，勝者拿走 **2 倍賭注**。勝率由雙方鎬子攻擊力決定，裝備越好越有優勢。對方可按「拒絕／取消」，或 **2 分鐘**內未回應自動取消，賭注全額退回。

<Callout type="warn">
  同時只能有一場進行中的決鬥邀請，等它結束才能再發起新的。
</Callout>

## 贈送

```
/贈送 對象 礦石 數量
```

把背包裡的礦石免手續費送給其他玩家 🎁。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `對象` | 收禮的玩家（不能是 bot 或自己） | 是 |
| `礦石` | 要送的礦石種類（下拉選項） | 是 |
| `數量` | 要送的數量 | 是 |

每日最多贈送 **3 次**，且對方背包要有足夠空間才送得出去。

## 拍賣

拍賣行讓你把礦石掛牌競標，由出價最高者得標 🏷️。每件拍賣 **24 小時**後結標，成交收取 **5%** 手續費；無人出價則自動退回礦石。

### 查看清單

```
/拍賣 清單
```

查看目前拍賣中的物品、最高出價與截止時間。出價時要用到清單裡的**拍賣編號**。

### 掛牌

```
/拍賣 掛牌 礦石 數量 起標價
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `礦石` | 要拍賣的礦石（下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `起標價` | 起標**總價**（不是每顆），至少為系統收購價的 **80%** | 是 |

每人同時最多掛 **5 件**拍賣品。

### 出價

```
/拍賣 出價 編號 金額
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `編號` | 拍賣編號（從 `/拍賣 清單` 查看） | 是 |
| `金額` | 出價**總額**，需比目前最高價至少高 **5%** | 是 |

出價時會先凍結你的逼幣；若被別人超越會**自動退款**，不必擔心卡住。不能對自己的拍賣品出價。

## 稱號與成就

挖礦的成就稱號（煤炭採集者、鐵鍛師、寶石獵人、傳說礦工，以及每週爭奪的礦坑之王）已整合進**跨遊戲稱號系統**，和賭場、股市、樂透、拍賣的稱號共用同一套 `/稱號 設定` 與 `/檔案` 的「🏆 成就」分頁與等級卡展示槽。完整稱號清單與解鎖條件請見 [遊戲稱號與成就](/docs/leveling/game-titles)。

## 礦工檔案

```
/檔案
```

到「⛏️ 礦工檔案」分頁，查看自己的完整檔案：展示稱號、已解鎖稱號數、週冠次數、目前鎬子與耐久、錢包餘額、背包用量，以及**生涯統計**（累積挖礦次數、合成件數、地下城次數）與**歷史採集量**。

## 挖礦排行

```
/排行榜 類別:挖礦
```

本週挖礦**數量**排行榜（前 10 名）。每週一 **00:01** 結算，把上週第一名頒發 **礦坑之王 👑** 稱號並在頻道公告，同時卸下上一任。快搶下第一個礦坑之王！
