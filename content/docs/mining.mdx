# 挖礦與打工

除了賭場與股票，你還可以靠**挖礦**和**打工**穩定賺取逼幣。挖到的礦石可以賣錢，也能拿去**合成更好的鎬子**，讓你挖得更快、挖到更稀有的礦。

<Callout type="info">
  挖礦、礦袋、賣礦、打工、合成、裝備等指令都在 **庶民勞動局 │ 💼** 頻道使用。
</Callout>

## 挖礦

```
/挖礦 [加速]
```

挖一次礦，每次有 **2 小時**冷卻時間，挖到的礦石會放進礦袋。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `加速` | 消耗 1 張「CD 縮短券」，本次挖礦後的冷卻時間少 30 分鐘 | 否 | 否 |

稀有度越高的礦越值錢，挖到**鑽石**時還會全服公告慶祝 ✨。

### 礦石一覽

| 礦石 | 稀有度 | 賣價 |
| --- | --- | --- |
| 石頭 | 普通 | 8 逼幣 |
| 煤炭 | 普通 | 20 逼幣 |
| 鐵礦 | 稀有 | 60 逼幣 |
| 黃金 | 稀有 | 200 逼幣 |
| 鑽石 | 傳說 | 800 逼幣 |

<Callout type="info">
  「幸運值（luck）」越高，挖到稀有礦的機率越高。鎬子、幸運藥水與贊助／Twitch 加成都會提升 luck，但全部加總最多 +25%。
</Callout>

## 礦袋

```
/礦袋
```

查看你的礦石背包、容量、目前的鎬子與耐久、道具存量，以及下次可挖礦的時間。背包基礎容量為 **100 格**，裝滿就不能再挖，要先賣掉一些礦石。

## 賣礦

```
/賣礦 [礦石] [數量]
```

把背包裡的礦石賣給系統換逼幣。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `礦石` | 要賣的礦石種類（下拉選項） | 否 | 不填則賣出背包全部 |
| `數量` | 要賣的數量 | 否 | 不填則賣出該礦石全部 |

## 打工

```
/打工
```

打工賺取穩定收入，每次可得 **80–120 逼幣**。冷卻 **4 小時**，每日最多 **6 次**，適合在挖礦冷卻時順手賺一點。

## 合成

```
/合成 裝備 [確認]
```

用礦石合成更好的鎬子。鎬子能縮短挖礦冷卻、提升 luck，鑽石鎬還能一次多挖一個礦。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `裝備` | 要合成的鎬子（下拉選項） | 是 | — |
| `確認` | 目前的鎬子還有耐久時，設為 `是` 才會替換掉它 | 否 | 否 |

### 鎬子與配方

| 鎬子 | 合成材料 | 挖礦冷卻 | luck | 耐久 |
| --- | --- | --- | --- | --- |
| 木鎬 | 初始裝備 | — | — | 永久 |
| 鐵鎬 | 鐵礦 ×15 | −30 分 | +5% | 50 次 |
| 黃金鎬 | 黃金 ×5、鐵礦 ×20 | −45 分 | +8% | 50 次 |
| 鑽石鎬 | 鑽石 ×5、黃金 ×10 | −60 分（並一次多挖 1 個礦） | +12% | 50 次 |

<Callout type="warn">
  除了木鎬，其他鎬子都有耐久（50 次）。耐久耗盡會自動換回木鎬，需要再合成一把。
</Callout>

## 裝備

```
/裝備
```

查看目前鎬子的屬性，以及各個合成配方所需材料的進度（已足夠 ✅／還缺 ❌）。
