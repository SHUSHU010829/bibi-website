# 挖礦與打工

除了賭場與股票，你還可以靠**挖礦**和**打工**穩定賺取逼幣。挖到的礦石可以賣錢，也能拿去**合成裝備**：用礦石打造更好的**鎬子**挖得更快、挖到更稀有的礦，或鍛造**武器**去闖地下城打怪。想闖**地下城**、跟人**決鬥**，都得先有一把像樣的劍。礦石還能**贈送**給朋友、上**拍賣行**競標，或到**市集**和其他玩家買賣、換礦、徵求。

<Callout type="info">
  挖礦、賣出、打工、合成、裝備、地下城、決鬥、贈送、拍賣等指令都在 **財富牢改城 │ 💼** 頻道使用；`/市集` 改在 **暗巷易物所** 頻道使用（`/背包` 與 `/市集 逛攤` 是只有自己看得到的私人指令，哪裡都能開）。
</Callout>

## 挖礦

```
/挖礦
```

挖一次礦，每次有 **2 小時**冷卻時間，挖到的礦石會放進背包。稀有度越高的礦越值錢，挖到**鑽石**時還會全服公告慶祝 ✨。

<Callout type="info">
  想少等冷卻？冷卻中打開 `/背包`，對「CD 縮短券」按 **使用** 就能立即少等 30 分鐘（剩餘不足 30 分會直接歸零）。
</Callout>

### 礦石一覽

| 礦石 | 稀有度 | 賣價 | 基礎掉落機率 |
| --- | --- | --- | --- |
| 石頭 | 普通 | 8 逼幣 | 55% |
| 煤炭 | 普通 | 20 逼幣 | 25% |
| 鐵礦 | 稀有 | 60 逼幣 | 12% |
| 黃金 | 稀有 | 200 逼幣 | 6% |
| 鑽石 | 傳說 | 800 逼幣 | 1% |

<Callout type="info">
  上表「基礎掉落機率」是 **luck 0**（沒有任何加成）時各礦石的權重比例。「幸運值（luck）」越高，挖到稀有礦（鐵礦／黃金／鑽石）的機率越高、石頭與煤炭相對變少。鎬子、幸運藥水與贊助／Twitch 加成都會提升 luck，但全部加總最多 **+25%**。石頭與黃金一次可能挖到多個（石頭 1–3、煤炭 1–2），實際進帳會再乘上數量。
</Callout>

### 賭石（找鑑定師）

挖到**石頭**時，`/挖礦` 的結果訊息會多出一顆 **🔍 找鑑定師賭石** 按鈕。付費請鑑定師把這次挖到的石頭**逐顆開出**，有機率變成更值錢的礦——當然，也可能全部碎掉！

- **只有剛挖到時能賭**：按鈕只認你**最新一次**挖到的石頭，挖到別的礦或再挖一次就失效，逾時（預設 10 分鐘）也會失效。每次挖到的石頭**只能賭一次**。
- **費用**：每顆石頭 **150 逼幣**（挖到幾顆就一次幫你全開）。
- **開出機率**（高風險高報酬）：

| 結果 | 機率 |
| --- | --- |
| 💥 碎掉（什麼都沒有） | 60% |
| 煤炭 ×1–2 | 18% |
| 鐵礦 ×1 | 10% |
| 黃金 ×1 | 7% |
| 💎 鑽石 ×1 | 2% |

<Callout type="warn">
  賭石是純運氣的金幣 sink，長期期望值是**虧的**，圖的就是那一發開出鑽石的爽感（開出鑽石一樣會全服公告）。背包快滿或金幣不夠時無法賭，先 `/賣出` 清一清。
</Callout>

## 背包

```
/背包
```

你的個人背包（只有自己看得到），把挖礦與商店的東西整合在同一張卡片：

- **礦石**：各礦石數量、背包容量（基礎 **100 格**，裝滿就不能再挖，要先賣出）、全部賣出可得逼幣。
- **挖礦狀態**：目前的鎬子與耐久、下次可挖礦時間。
- **挖礦道具**：幸運藥水（挖礦時自動生效）、CD 縮短券（冷卻中可按 **使用**）、**傳說素材碎片**（地下城掉落，**合成傳說之劍的材料** —— 想確認數量就來這裡看）。
- **商店道具與 buff**：你買的顏色身份組／卡面／自訂稱號、生效中的加成與到期時間。可裝備的道具（顏色身份組、卡面、等級卡顏色、自訂稱號）都整合在**同一個下拉選單**裡，選到自訂稱號會跳出視窗讓你設定文字，其餘則直接裝備。

<Callout type="info">
  **CD 縮短券**：挖礦冷卻中時在 `/背包` 對它按 **使用**，立即少等 30 分鐘（剩餘不足 30 分直接歸零）。最多持有 **30 張**，可到 `/商店` 購買。
</Callout>

## 賣出

```
/賣出 [物品] [數量]
```

把背包裡的礦石或魚賣給系統換逼幣，依**當日行情**計價。這是統一賣給系統的指令——`物品` 下拉選單同時涵蓋各種礦石與魚種。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `物品` | 要賣的礦石或魚種（下拉選項） | 否 | 不填則賣出背包全部 |
| `數量` | 要賣的數量 | 否 | 不填則賣出該物品全部 |

<Callout type="info">
  礦石與魚的收購價都會依**當日行情**波動 **±30%**（由日期決定、全服一致、同一天重算不變），可用 `/行情` 查看當日礦石收購價。釣魚與烹飪的完整玩法見 [釣魚與烹飪](/docs/fishing)。
</Callout>

## 打工

```
/打工
```

打工賺取穩定收入，冷卻 **4 小時**，每日最多 **6 次**，適合在挖礦冷卻時順手賺一點。

打工設有**等級制**：依**累計打工次數**自動升級，等級越高每次的報酬區間越好（升級無須操作，達標自動生效）。

| 等級 | 稱號 | 累計打工次數 | 每次報酬 |
| --- | --- | --- | --- |
| Lv.1 | 打工新手 | 0 次 | 80–120 逼幣 |
| Lv.2 | 兼職人員 | 60 次 | 95–135 逼幣 |
| Lv.3 | 正式員工 | 220 次 | 110–155 逼幣 |
| Lv.4 | 資深員工 | 480 次 | 125–175 逼幣 |
| Lv.5 | 王牌員工 | 900 次 | 140–185 逼幣 |

## 合成

```
/合成 裝備 [確認]
```

用礦石合成裝備，分成兩種：**鎬子**（採集用，縮短挖礦冷卻、提升 luck，鑽石鎬還能一次多挖一個礦）與**武器**（戰鬥用，決定地下城與決鬥的戰鬥力）。

| 參數 | 說明 | 必填 | 預設 |
| --- | --- | --- | --- |
| `裝備` | 要合成的鎬子或武器（下拉選項） | 是 | — |
| `確認` | 目前同類裝備還有耐久時，設為 `是` 才會替換掉它 | 否 | 否 |

### 鎬子與配方（採集）

| 鎬子 | 合成材料 | 挖礦冷卻 | luck | 耐久 |
| --- | --- | --- | --- | --- |
| 木鎬 | 初始裝備 | — | — | 永久 |
| 鐵鎬 | 鐵礦 ×10 | −30 分 | +5% | 50 次 |
| 黃金鎬 | 黃金 ×5、鐵礦 ×15、煤炭 ×8 | −45 分 | +8% | 50 次 |
| 鑽石鎬 | 鑽石 ×7、黃金 ×14、煤炭 ×10 | −60 分（並一次多挖 1 個礦） | +12% | 50 次 |

### 武器與配方（戰鬥）

武器決定你在**地下城**打怪與**決鬥**的戰鬥力，**鎬子不再影響戰鬥**。赤手空拳也能打，但**勝率極低**（上限只有 10%），強烈建議先打一把劍。高階武器有**暴擊率**，暴擊時直接獲勝、金幣戰利品再 ×1.5。

| 武器 | 合成材料 | 戰鬥力 | 暴擊 | 耐久 |
| --- | --- | --- | --- | --- |
| 赤手空拳 | 初始狀態（可打怪但勝率極低） | 20 | — | 永久 |
| 鐵劍 | 鐵礦 ×15 | 45 | — | 60 次 |
| 鋼劍 | 鐵礦 ×30、煤炭 ×20 | 70 | 3% | 60 次 |
| 黃金劍 | 黃金 ×8、鐵礦 ×24、煤炭 ×10 | 100 | 6% | 60 次 |
| 鑽石劍 | 鑽石 ×8、黃金 ×18、煤炭 ×12 | 140 | 10% | 50 次 |
| 傳說之劍 | 傳說素材碎片 ×18、鑽石 ×12、煤炭 ×15 | 200 | 20% | 80 次 |

<Callout type="warn">
  鎬子（非木鎬）與武器（非赤手）都有耐久：鎬子每次**挖礦**扣 1、武器每場**地下城**扣 1。耐久耗盡會自動退回木鎬／赤手，需要再合成一把。**煤炭**是中後期裝備的關鍵材料（黃金鎬以上的鎬子、鋼劍以上的武器都要用到），別急著把它賣光。傳說之劍要用地下城掉落的**傳說素材碎片 ×18**，慢慢累積吧！
</Callout>

### 鎬子修復

鎬子快沒耐久時，不一定要重新合成一把，可以**修復**現有的鎬子（木鎬無耐久、不需修復）：

- **材料修復**：到 `/背包` 對目前的鎬子按 **修復**，固定消耗一些**石頭＋煤炭**把耐久補滿，**不影響最大耐久**——是長期最划算的方式。
- **劣質磨刀石**：到 `/商店` 以 **250 逼幣**購買，再到 `/背包` 使用，可立即把耐久補滿，但鎬子的**最大耐久 −10**（最大耐久不足 20 時無法使用）。應急方便，但用多了鎬子會越來越快壞，建議優先用材料修復。最多持有 **20 個**。

<Callout type="info">
  修復只回**耐久**、不改變鎬子等級。想換更好的鎬子還是得用 `/合成`。
</Callout>

```
/裝備
```

查看目前**鎬子**與**武器**的屬性、戰鬥力，以及鎬子・武器各個合成配方所需材料的進度（已足夠 ✅／還缺 ❌）。

## 地下城

```
/地下城
```

消耗 **1 點體力**深入地下城戰鬥，勝利可獲得礦石、逼幣、道具或傳說素材碎片 ⚔️。

<Callout type="warn">
  **赤手空拳也能打，但勝率極低（上限 10%）！** 強烈建議先用 `/合成` 打造一把武器（最低階的鐵劍只要鐵礦 ×15）再來闖，戰鬥力差很多。
</Callout>

- **體力**：上限 **10 點**，每 **1 小時**自動回復 1 點，耗盡時要休息一下再來。不想等回復？可到 `/商店` 買**體力藥水**，購買後立即補 **5 點**（不超過上限、體力已滿時不可購買，每日最多買 **2 瓶**）。
- **勝率**：由你的**戰鬥力**與怪物血量（60–280）決定。**持有武器**時介於 **20%–90%**；**赤手空拳**則上限只有 **10%**（約 7–10%，非常難贏）。戰鬥力越高勝率越高。
- **戰鬥力**：基礎 20，依**武器**加成（鎬子不影響戰鬥）。合成更好的武器能明顯提升勝率。
- **暴擊**：高階武器有暴擊率，暴擊時直接獲勝、金幣戰利品再 ×1.5。
- **武器耐久**：每打一場消耗 1 點，耗盡會退回赤手空拳，要再合成。

| 武器 | 戰鬥力 |
| --- | --- |
| 赤手空拳 | 20（勝率極低，上限 10%） |
| 鐵劍 | 45 |
| 鋼劍 | 70 |
| 黃金劍 | 100 |
| 鑽石劍 | 140 |
| 傳說之劍 | 200 |

### 戰利品

| 戰利品 | 內容 | 機率（約） |
| --- | --- | --- |
| 鐵礦 | ×3 | 25% |
| 煤炭 | ×4 | 13% |
| 黃金 | ×1 | 8% |
| 逼幣（小） | 150–320 逼幣 | 20% |
| 逼幣（大） | 450–850 逼幣 | 6% |
| 幸運藥水 | ×1（挖礦時自動生效） | 6% |
| CD 縮短券 | ×1（冷卻中在 `/背包` 使用） | 6% |
| 傳說素材碎片 | ×1（合成傳說之劍的材料） | 3% |
| 槓龜 | 什麼都沒掉 | 13% |

<Callout type="info">
  撿到的東西都會進背包：礦石若超過背包容量會自動折算成逼幣、CD 縮短券超過持有上限（30 張）也會折算成逼幣，不會浪費。
</Callout>

### 突發事件

`/挖礦`（約 12%）與 `/地下城`（約 15%）行動後，有機率觸發**突發事件**，有好有壞：

- **挖礦時**：隱藏礦脈（本次採集翻倍）、廢棄礦車（撿到逼幣）、時間裂隙（立即解除挖礦冷卻）、流浪商人（送你幸運藥水）、遠古化石（研究員給你一筆謝禮金幣）、迴音坑道（送你 CD 縮短券 ×1）、礦坑崩塌（掉落部分礦石）、**怪物突襲**（用你的武器自動應戰，贏了拿逼幣、輸了被搶走部分礦石——所以礦工也該帶把劍！）。
- **地下城時**：藏寶箱（額外逼幣）、遺落補給（額外黃金）、神秘藥水（回體力）、磨刀石（武器耐久全滿）、流浪鐵匠（送你 CD 縮短券 ×1）、晶洞密室（額外鑽石）、遺忘的鍛爐（撿到劣質磨刀石）、陷阱（損失體力）、**詛咒祭壇**（用體力賭一筆金幣，有賺有賠）、**精英怪**（高風險高報酬，有機會掉傳說素材碎片）。

<Callout type="info">
  突發事件撿到的礦石、逼幣、道具同樣會進你的 `/背包`／錢包。
</Callout>

## 決鬥

```
/決鬥 對象 賭注
```

向其他玩家發起 1v1 逼幣決鬥，勝者通吃彩池 ⚔️。

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `對象` | 要挑戰的玩家（不能是 bot 或自己） | 是 |
| `賭注` | 雙方各押的逼幣數，**10 ~ 5000** | 是 |

發起後對方會收到邀請，按下「接受決鬥」即扣雙方賭注並開打，勝者拿走 **2 倍賭注**。勝負由雙方**武器戰鬥力**決定，武器越好越有優勢。對方可按「拒絕／取消」，或 **2 分鐘**內未回應自動取消，賭注全額退回。

<Callout type="warn">
  同時只能有一場進行中的決鬥邀請，等它結束才能再發起新的。
</Callout>

## 拍賣

拍賣行讓你把礦石掛牌競標，由出價最高者得標 🏷️。每件拍賣 **24 小時**後結標，成交收取 **5%** 手續費；無人出價則自動退回礦石。

### 查看清單

```
/拍賣 清單
```

查看目前拍賣中的物品、最高出價與截止時間。出價時要用到清單裡的**拍賣編號**。

### 掛牌

```
/拍賣 掛牌 礦石 數量 起標價
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `礦石` | 要拍賣的礦石（下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `起標價` | 起標**總價**（不是每顆），至少為系統收購價的 **80%** | 是 |

每人同時最多掛 **5 件**拍賣品。

### 出價

```
/拍賣 出價 編號 金額
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `編號` | 拍賣編號（從 `/拍賣 清單` 查看） | 是 |
| `金額` | 出價**總額**，需比目前最高價至少高 **5%** | 是 |

出價時會先凍結你的逼幣；若被別人超越會**自動退款**，不必擔心卡住。不能對自己的拍賣品出價。

## 市集

`/市集` 是玩家之間的交易市集 🏪，比 `/拍賣` 更彈性：除了競標，還能用**一口價賣礦**、**一口價賣魚**、**以物易物換礦**，或貼出**徵求單**收購別人的礦。所有掛單都會**託管**你交出的礦石、魚或逼幣，**24 小時**後到期；無人成交或自行下架時，託管的東西會**自動退回**。每人同時最多掛 **8 件**（所有類型合計）。`/市集` 在 **暗巷易物所** 頻道使用。

<Callout type="info">
  `/市集 賣礦` 是**掛牌賣給其他玩家**（收逼幣），和直接賣給系統的 `/賣出` 不一樣；想賺多一點就掛市集，想立刻清背包就用 `/賣出`。
</Callout>

### 逛攤

```
/市集 逛攤
```

瀏覽市集所有掛單，**僅自己可見**，可用**交易類型**與**物品類別**（礦石／魚類）下拉篩選想看的掛單。每筆掛單右側都有對應的按鈕可直接成交——賣礦／賣魚按 **購買**、換礦按 **接受交換**、徵求按 **賣給他**、競標按 **出價**。成交前都會跳出**確認面板**，避免誤觸。

### 我的攤位

```
/市集 我的攤位
```

查看自己目前的掛單，每筆右側附 **下架** 按鈕。下架後託管的礦石／逼幣會退回你的帳戶。

### 賣礦（一口價）

```
/市集 賣礦 礦石 數量 總價
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `礦石` | 要賣的礦石（下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `總價` | 一口價**總額**，至少為系統收購價的 **80%** | 是 |

買家一鍵買走，成交收取 **5%** 手續費。

### 賣魚（一口價）

```
/市集 賣魚 魚 數量 總價
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `魚` | 要賣的魚種（下拉選項） | 是 |
| `數量` | 數量 | 是 |
| `總價` | 一口價**總額** | 是 |

和賣礦一樣掛牌賣給其他玩家，成交收取 **5%** 手續費，無人購買會自動退回魚袋。釣魚與烹飪詳見 [釣魚與烹飪](/docs/fishing)。

### 換礦（以物易物）

```
/市集 換礦 給的礦石 給的數量 想要礦石 想要數量
```

用某種礦換另一種礦，**免手續費**。給出與想要的礦石不能相同。

### 徵求（收購單）

```
/市集 徵求 想要礦石 想要數量 付款方式 [金幣總額] [付的礦石] [付的數量]
```

貼出「我要收某礦」的收購單，付款方式可選**金幣**或**礦石**：

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `想要礦石` | 你想收購的礦石 | 是 |
| `想要數量` | 想收購的數量 | 是 |
| `付款方式` | `金幣` 或 `礦石` | 是 |
| `金幣總額` | 付款方式為金幣時填 | 否 |
| `付的礦石` / `付的數量` | 付款方式為礦石時填 | 否 |

掛單時會先**託管**你要付的金幣或礦石。別人賣給你後，若以金幣支付，賣方會被收取 **5%** 手續費。

### 競標

```
/市集 競標 礦石 數量 起標價 [一口價]
```

| 參數 | 說明 | 必填 |
| --- | --- | --- |
| `礦石` | 要競標的礦石 | 是 |
| `數量` | 數量 | 是 |
| `起標價` | 起標**總價**，至少為系統收購價的 **80%** | 是 |
| `一口價` | 選填，出價達到此價立即成交（須 ≥ 起標價） | 否 |

出價需比目前最高價至少高 **5%**，被超越時**自動退款**；成交收取 **5%** 手續費。

## 稱號與成就

挖礦的成就稱號（煤炭採集者、鐵鍛師、寶石獵人、傳說礦工，以及每週爭奪的礦坑之王）已整合進**跨遊戲稱號系統**，和賭場、股市、樂透、拍賣的稱號共用同一套 `/稱號 設定` 與 `/檔案` 的「🏆 成就」分頁與等級卡展示槽。完整稱號清單與解鎖條件請見 [遊戲稱號與成就](/docs/leveling/game-titles)。

```
/檔案
```
